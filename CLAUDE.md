Пример: `törichtes` — Adelung «ehedem unsinnig, toll» + контекст цикла (безумие) → «безумное», не «безрассудное».

**ТАБЛИЦА-ДОКАЗАТЕЛЬСТВО** (форсинг против ложного «проверено»): перед «проверено» — таблица **всех** источников (порядок (1) Adelung+Grimm; (2) DWDS, Pfeifer, Duden, Duden Synonyme, Wiktionary, etymonline, Leipzig; (3) БНРС, Multitran, PONS, Langenscheidt DE→RU; (4) БРНС, Multitran RU→DE — обратная сверка КАЖДОГО RU-кандидата). Пустых ячеек нет; нет цитаты → НЕ проверен. **Без полной таблицы «проверено» ЗАПРЕЩЕНО** — писать «ЧАСТИЧНАЯ, пропущены: X». word-check.py печатает чеклист + блок обязательных WebFetch.

## Translation Rules

**Правила — обобщения опыта, не аксиомы; при конфликте с грамматикой приоритет грамматике.** Подстрочник = пословное соответствие, не литер. перевод.

**Точность > поэтика**: RU грамотный, существующий (проверять сочетаемость), но не обязан звучать красиво. Точное > привычного (`блестящих` не `светлых`; `в споре` не `наперебой`); при равной точности — частотный; устоявшийся RU — не аргумент держаться менее точного; не добавлять отсутствующих оттенков (уменьшит. суффиксы, ложные архаизмы, аспект). **Поэтически — только если DE поэт. в 1820-х** (Adelung gehoben/dichterisch, Grimm poetisch). **Многословный/дефисный — если точнее** (правило 45: `Verlangen → страстное стремление`, `Täuschung → обман-иллюзия`). **DE→RU, английский не использовать** (кроме исключений).

**Порядок работы** для знаменательного слова (вкл. частицы/наречия: `immerfort`=непрестанно; `je`=хоть когда-нибудь):
1. **DWDS** (значения/регистр/этимология); для XIX в. также **Grimm** (`dwds.de/wb/dwb/<W>`) + **Adelung Wien 1811** (BSB); регистр дрейфует.
2. **Приставка+корень** (Er-sterben ≠ sterben).
3. **Близкие слова развести** (Boden/Erde, Blüte/Blumen).
4. **Контекст > этимологии/словаря**: при двух значениях (`starrt`=застыл+уставился) — оба; модальные предположения (`mag`=должно быть) — отдельный RU; поэт. компаунды расширены (Felsengrund DWDS=Felstal, но П.9→«скальные ущелья»; Wüstenei — Grimm включает felsige Berglandschaften).

**Частицы doch/denn/nun/wohl по контексту** (конвенции цикла):
- **denn**: Ja/Nein- или soll/sind-V1-вопрос → «**Неужели**» (П.03,04,21); w-вопрос → «**же**» (П.13,20); discourse «then» после Nun → «**тогда**» (П.21).
- **doch**: эмфат. подтверждение → «**ведь**» (П.03); с `ja` → «**ведь же**» (П.20); в w-вопросе → «**же**» (П.06); в Ja/Nein-вопросе контраст → «**И всё же**» (П.21).
- **Nun**: temporal → «**Теперь**» (П.01,05,10,11,23); discourse → «**Ну да**»/«**Что ж**» (П.13,15,21). Запятая после `Nun` у Müller'а = discourse.
- **wohl** (модальная): «**, наверное,**» вводным (П.11,13,15,21). **Искл. `wohl`+числительное** = градация-усиление → «**целых**» (П.23 «hatt' ich auch wohl drei»; Adelung «wohl zehn Mahl»), не «наверное».
- **so** в `so...dass` — степень (правило 25). **schon** в `Tret' ich auch schon` — усилитель уступки.

**RU пунктуация**: запятые при обращениях/вводных/деепричастиях; **не калькировать DE автоматически** (`Nun, ...`DE → «Теперь не далеко» RU без запятой); при замене слова — перепроверить согласование («своё отражение»→«свой образ» м.р.). **Неочевидный выбор — сигнализировать** (идиомы, архаизмы, многозначность, регистр).

**Источник DE** — шубертовская редакция (как поют), по убыванию авторитета (файлы/рендеринг — § Schubert-партитуры):
1. **Автограф Schubert'а 1827** — первоисточник (рукопись с зачёркиваниями; описки есть: П.8 L5 `jeden` vs грам. `jedem`).
2. **NSA Bärenreiter** (ред. W. Dürr) — Urtext, исправляет описки автографа (нотные конвенции — «Что НЕ доверять в NSA», § партитуры).
3. **Rastl «Original Spelling»** (schubertsong.uk) — реконструкция Schubert sung-text в Müller-орфографии; **лучший для апострофов/пунктуации**.
4. **Müller WH1824** (Wikisource `Die_Winterreise_(Müller)`) — оригинал; понимание «что Schubert изменил» + проверка пунктуации Schubert'а.
5. **Naxos PDF** (Deutsche Schubert-Lied-Edition 1999) — **12+ опечаток** в 20 песнях (`mit zum` П.16, `Habe ich ja` П.20, `im mattem` П.18); не основной.

Когда сверяться по типу расхождения — правило 61. **Орфография** по реформе 1996: ß после кратких→ss (Muß→Muss), после долгих остаётся (Blumenstrauß); Th→T (Thore→Tore, Thränen→Tränen — наша конвенция модернизации).

**Карта правил**: сегментация 1·4·5·9·10·17·18·23·27·30 | порядок слов 2·3 | грамматика DE→RU 11·12·13·14·15·16·19·20·24·25·26·34·36·40·41·50·55 | лексика и проверка 6·21·28·29·32·35·37·42·43·45·46·48·56·62 | Müller↔Schubert 22·61 | аннотации 8·47·59 | RU-типографика 44·54·58 | процесс 7·63.

1. **Каждое DE-слово — в подстрочнике.** Не пропускать служебные (da, schon, doch); знаменательные (модал./част./нареч.) обязаны иметь RU (`mögen` 3 = «должно быть»). Слово с самостоятельным RU — отдельный сегмент даже в рамке (`hinaus → вон`, `nach → вслед`). RU-предлог поглощает приставку, если передаёт направление (`In die Felsengründe lockte... hin → «В глуби заманил»`). **Прозрачность**: буквальный разбор > словарный (`Gibt... sich hin → отдаёт... себя`, не «предаётся»).

2. **По умолчанию сохранять DE порядок слов**; перестройка — только когда RU **нечитаем**. Сохраняются: **поэт. инверсия V-letzt** (`Von der Straße her ein Posthorn klingt` → «С улицы почтовый рожок доносится»); **поэт. инверсия Gen впереди** (`meiner Hoffnung Grab` → «На моей надежды могиле»); **постпозитивная аппозиция** (`Der Winter kalt und wild` — прил. без склонения, предикат-эквивалент: «Зима холодная и буйная»).

3. **Менять порядок только когда RU нечитаем.** «На моей прекрасной любимой доме» — нельзя; «Тут я подумал уже в своём ослеплении» — ок.

4. **Одно RU = одно DE по возможности.** Группировать только когда иначе невозможно. Отделяемые приставки с ясным RU — отдельный сегмент (`dahin → вдаль`). Рамка с многоточием (`pfiff'...aus`) — когда RU не разделить. **Многоточие — при разрыве словами в DE**; рядом стоящие (Folge nach, Komm her) — без. **Порядок в рамке = текстовый порядок DE**.

5. **`de` пустая** — для DE без RU-эквивалента (внутри рамки).

6. **Неочевидный выбор — сигнализировать, не делать молча**: идиомы, архаизмы, многозначность, регистр, интерпретативные отступления от буквального — списком спорных мест с альтернативами. Конвенция: **Liebchen = любимая** (не «милая»; Lieb- от Liebe).

7. **Не писать содержательные изменения без приглашения. Не пушить без явного «пушим»/«делаем».** Одобрение перед записью в файл (не только перед commit; файл сразу виден на dev-сервере). Порядок: предложить → «ок» → писать → показать → «пушим» → commit/push. **Аннотации, `variant_de`/`variant_ru`, ретроактивные правки — каждый случай отдельно.** Одобрение конкретного шага не распространяется на сопутствующие. **Условное одобрение** («если всё чётко, пушим») — сначала результат, потом действие.

8. **В аннотациях переводить немецкие фрагменты**: «(es eher bemerken — это раньше заметить)», не просто «(es eher bemerken)».

9. **Артикли (ein, der, die, das) ≠ «некий/этот»** по умолчанию. Артикль **слева** от сущ. — объединять со СЛЕДУЮЩИМ сегментом: `{"ru":"верной","de":"ein treues"}`; **НЕ с предыдущим** (`Welch ein törichtes` → `Welch`+`ein törichtes`+`Verlangen`). **Исключения** (артикль несёт смысл): (а) числительное «один» при контрасте (`Einen Weiser` П.20 = «Один указатель»); (б) демонстрат. `das` без сущ. (`Das nenn ich einen Morgen` П.18 → «Это называю...»); (в) анафор. `der` = конкретный известный (П.08 «der Tag» = «тот день»).

10. **Предлоги — отдельные сегменты.** `{"ru":"с","de":"mit"}` + `{"ru":"сердцами","de":"den Herzen"}`.

11. **Связка sein → «—» отдельным сегментом, и исключения.** (а) **Основное**: bin/bist/ist/sind/seid — тире, не «есть»/«является»; для любого именного сказуемого (сущ., прил., местоим., adjektives Partizip — статив; модификатор `wie`/`manche Stunde`). **Рамка `Adj.+sein-форма` запрещена.** Тире — RU-стандарт даже при DE Verbletzt (`wie müd' ich bin` → «усталый / — / я,»). Прошедшее war / Konj.II wär' — RU-глагол. (б) **Modal+sein-Inf** — пара RU-глаголов (`Könnt... sein → Можете... быть`, не тире). (в) **Perfekt от глаголов изменения**: «Die Blumen sind erstorben» = цветы угасли (связка не применяется); признак — причастие от глагола изменения (ersterben, einziehen, aufstehen); **модификатор `wie` блокирует Perfekt-чтение**: `Mein Herz ist wie erstorben` — адъективное причастие (статив), тире («есть словно угасшее»); так же `wie tot`, `wie betrunken`. (г) **Funktionsverbgefüge** (`offen lassen`, `liegen lassen`, `zu Ende sein mit`) — идиоматически одним RU-глаголом, не покомпонентно; рамка без многоточия (DE рядом) или с (при разрыве).

12. **Идиомы**: буквальный перевод предпочтительнее, если грамматичен (`im Streit → в споре`); по смыслу — когда буквально невозможен (`Was fragen sie nach → Что за дело им до`). **Ложные соответствия**: `sich zur Ruhe legen` = просто лечь, RU «лечь на покой» = умереть. **Сдвиг подлежащего**: `ich hatte X → у меня была X`; `was hat es? → что с ним?`. **Сдвиг управления**: DE транзитив → RU предложное (`klagen+Akk`: `was es mir klagt → на что жалуется`, не «что»). **Однозначные идиомы**: `bei sich denken → подумать про себя` (молча). **Все случаи — с lang-аннотацией.**

13. **Косвенная речь.** В RU нет согласования времён: после «подумал...» одновременное действие → настоящее («подумал: он освистывает»).

14. **Возвратное «свой» — только в пределах одной RU-клаузы.** DE-посессив, согласованный с субъектом своей клаузы → «свой». При разбиении на две клаузы (AcI через «как»/«:», подчинит., новое предл.) — «свой» через границу не работает: антецедент извне через «его/её/их». Пример: `Mein Herz sieht ... sein eignes Bild` → «Моё сердце видит: на небе нарисован **его** собственный образ» (правило 40).

15. **Род адаптируется.** Sie (Wetterfahne ж.р.) → он (флюгер м.р.). При значимом расхождении — lang-аннотация.

16. **Verbklammer и tun-Periphrase.** Aux+Partizip II = рамка-многоточие (`bin... eingezogen → пришёл`). Aux+Inf (Modal-Klammer, Futur): `hätt'... sollen`, `wird... gewinnen`. RU подлежащее перед смысловым глаголом, как в DE; содержательный verb в конце DE → в конце RU. **НЕ Verbklammer**: стандартный Nebensatz S-Inf-V-fin (`was ich zittern kann` — придаточное, не рамка). **Tun-Periphrase** в Volkslied: `tun` — вспомог. (не «делать»), несёт habituell-эмфатику; RU-аналог — частица **«знай»** (Ожегов: «постоянно, не переставая»); сегментация: tun → «знай» отдельным сегментом, смысловой Inf — в конце RU (`Tun sich... erlaben` → «Знай, себя... услаждают»). **Субстантивированный инфинитив** (das Klagen) → RU инфинитив, не отглаг. сущ. (`Klagen ist für Toren → Жаловаться — для безумцев`, не «Жалобы»; связь с `klagt`).

17. **Предлог, поглощённый падежом, + mit-конструкции.** `gehüllt in Schnee → окутан снегом` — `{"ru":"снегом","de":"in Schnee"}` (творит. поглощает `in`). **Творительный образа действия** — слово действия/манеры, не часть тела: DE `mit+Adj+N` → RU «X шагом/поступью/походкой» (ходьба), «X голосом/взглядом» (речь/взгляд); НЕ «X ногой/стопой» (`mit trägem Fuß` → «вялым шагом»). **`mit` без явного Dat — наречие сопричастности**, не предлог: «тоже»/«вместе»/«заодно» (`Fall' ich selber mit zu Boden` → «падаю я сам тоже на землю»). Различие: `mit ihm` — предлог; `mit` без объекта — наречие.

18. **Никаких пустых сегментов (`ru=""`).** DE без RU — сливается с соседним в `de` (рамка). **Пустой `de`** (при непустом `ru`) — для RU-добавок для грамматики: «что» для dass, «Если/Когда» для V1-условий, «как» для AcI (правило 40), «Тот» как восстановленный антецедент `der`, RU-предлог при DE Dat./Akk. без предлога (`mir → ко мне/от меня/у меня`). Перед добавлением — не решается ли пунктуацией. **`'s`=es** (формальный субъект) сливается с глаголом (`'s führt → Ведёт`). **Различение `'s`**: перед глаголом = `es` (поглощается); перед сущ. ср.р. или после предлога = `das` (артикль, правило 9: `An's Haus`). **Отрицание — Модель A**: Keiner/nichts/nimmer → отрицательное слово + сегмент `«не»←пустой de` + глагол отдельно (П.02/20/22/24); «не» в глагол не вшивать. **Kein нейтрален** («nicht ein» — обязательное отрицание именной группы; усиленное было бы gar kein, у Мюллера таких нет): RU-дефолт — голое «не» при глаголе, сегмент `«не»←kein` (П.07/10/13/22-Ohren); отрицательное слово («никакого») — только при аргументе из оригинала (П.04 — вынос kein-группы перед подлежащим) или по необходимости референции (`kein Gott → «ни один бог»` П.22); субстантивное Keiner → «никто» (П.20/24).

19. **Модальные глаголы в контексте.** (а) Sollen во 2 л. (du sollst) — воля говорящего: `Sollst meinen Tritt nicht hören → Моих шагов не услышишь`. (б) **Совещательный вопрос 1-го лица** `soll/will ich + Inf?` — НЕ «должен ли я»/«хочу»: RU дательно-инфинитивное «**Мне + инф.?**» (П.01 «Зачем мне дольше оставаться», П.04 «Мне не взять», П.17 «Что мне мешкать», П.24 «Мне с тобой идти?»); для модальных/идиоматичных мест обратную сверку RU→DE (21) прогонять СРАЗУ. (в) **Ja/Nein-вопросы — интонацией, без «ли»** («ли» навязывает фокус на предыдущем слове); в цикле «ли» — только П.07 `не... ли` (36). (г) **Soll-аннотации** — канон: «кто-то или что-то хочет/требует (внешняя воля, обстоятельства, судьба)»; П.01 вводит, П.04/24 ссылаются.

20. **Эллипсис глагола.** DE допускает опущение, RU — нет: `Sacht, sacht die Türe zu! → Прикрою тихо-тихо дверь`. Обязательна lang-аннотация. Если опущенный verb **неопределим** (намеренный эллипсис-возглас) — RU **тире-маркер**, не домысливание: `Lustig in die Welt hinein → Весело — в мир` (не «Иду весело»).

21. **Не угадывать — проверять; Adelung-градации значимы.** (а) Приоритет — смыслы 1820-х (Adelung+Grimm); Adelung ≠ DWDS modern → у Müller'а Adelung. **Близкие RU-синонимы развести по корням** (трепетать DE zittern ≠ трепаться DE flattern). **RU→DE обратная сверка — sanity check, не аргумент**. При «проверить точность» — Шаг 0 word-check.py + вывод в чат. (б) **Грамматические утверждения**: DWDS, Duden, E-VALBU (grammis.ids-mannheim.de/verbvalenz); первая ассоциация может быть ложной. (в) **Adelung-градации значимы**: «erste Grad», «mit Widerwillen», «weniger als…» меняют RU: `trübe` (erste Grad der Unlust) → «сумрачный», не «мрачный»; `träge` (mit Widerwillen) → «вялый»; `säumen` → «мешкать»; `matt` → «слабый»; `elend` → компаунд (45). Для значимого слова — Adelung+Grimm, не только DWDS modern.

22. **Schubert vs Müller — по иерархии «Источник DE»; три источника Müller'а.** Приоритет: автограф+NSA > Rastl > Müller WH1824; LiederNet/Hampson — НЕТ. При первой итерации — список различий с цитатами **до** записи. **`variant_de`/`variant_ru` — только для sung-вариантов** (поют ОБА: П.4 erstorben/erfroren, П.20 Wegen/Straßen, П.23 L7 Ach/Ja — «Ja» поёт напр. M. Wilmering). Правки M vs S, которые не поют, — НЕ повод для variant (П.21 L12 «Und» из WH1824 не поют — variant убран, разница в meaning-аннотации; П.23 L4 könnten→wollten не поют — только аннотация). **Три источника Müller'а**: (1) **Urania 1823** — 12 стихов; (2) **Deutsche Blätter 1823** — 10 «продолжений» (Krähe, Mut, Wegweiser, Leiermann); (3) **Waldhornist II 1824** — все 24 с правками. Schubert: Часть I (1-12) по Urania, Часть II (13-24) по Waldhornist; Deutsche Blätter в Вену 1827 не доходил. **Алгоритм DB/WH/S**: DB=WH=S, отличие у S → правка Schubert'а; DB≠WH=S → правка Müller'а; DB=S≠WH невозможно. Применим только к 10 «продолжениям»; для Urania/Waldhornist'а без DB — расхождение = правка Schubert'а. LiederNet/schubertsong.uk «Müller» — часто Deutsche Blätter.

23. **Элизии (апостроф)** — последовательно в `lines_de`, `title_de`, `de`: zieh', dacht', hätt', hab', Wär', Ruh', pfiff', Gefror'ne, An's, sah', Ging', dritt'. Перед многоточием рамки апостроф сохраняется (`pfiff'... aus`). **Apokope конца -e/-en** → апостр.; **синкопа -e- середины слова** (sehn/stehn/angesehn) — БЕЗ апостр. (Müller WH1824).

24. **Перевод через отрицание при расхождении коннотаций.** `lau` (акцент «недостаточно горячий») → «негорячи», не «теплы» (RU «тёплый» позитивно).

25. **`So` в `so...dass` — всегда степень**: «так»/«настолько», никогда «едва/еле»; направление — следствием в dass-придаточном.

26. **Konjunktiv II ≠ настоящее.** Hätte/wäre/wollte — прошедшее + «бы»: wolltet → «хотели бы». После «словно/как будто» — прошедшее без «бы». **`Als`+Konj.II (ирреал. сравнение) — единообразно «Словно»** (П.03, 05, 23).

27. **Шубертовские музыкальные повторы.** (а) **Повтор через ` / `** в `lines_de`, `de`/`ru`: `"Und / Ihr dringt..."`, `{"ru":"А / Вы","de":"Und / Ihr"}` (первый — основной, второй — повтор). (б) **Прямые повторы внутри строки**: Schubert повторяет слова в вокальной партии; `lines_de` отражает sung-text; проверять по NSA (основной), Naxos подтверждает (`Wein', wein'` П.16 L12, `so elend, so elend` П.12 L12). **Повтор как ресурс**: одно DE дважды — два слота для разных RU пластов (`so elend, so elend` → «так жалок, так несчастен»).

28. **Одно DE через цикл — один RU; близкие в одной песне — разные.** Конвенции: `rauschen → шуметь`, `Geselle → дружище`, `bald → скоро` (нейтр., не «вскоре» литер.), `still → «тих-»` (П.07/08/10; «неподвижн-» — за `unbeweglich`/`unverrückt` П.07/20, поэтому `stillstehen` П.24 — «без движения»), `starr → «застыв-»` (П.04/07/24), `stier → «застылые»` (П.23), `fest → «неотрывно»` (П.23). Близкие (Boden/Erde, Flur/Wiese/Feld; Schmerz/Leid/Leiden) — разные RU. Каузативные пары: `dringen aus → вырываться`, `durchdringen → прорвать`, `drängen → рваться`. **Модификаторы блокируют** (so/noch/je/gar). **Substantiv vs attributiv**: `manches Blatt` (attr.) → «не один лист» vs `manches` (subst.) → «всякое». **Морф. пары в одной песне** (Wegweiser/Weiser → Путевой указатель/указатель) — различать. **Авторские лексич. цепочки** (Tot-X П.21; weisen-: Wegweiser/Weiser П.20 → weisest ab П.21; treu+Wanderstab: П.02+15+21; безум-: törichtes П.20 → Toren П.22) — сохранять корень в RU; теряется связь — отметить lang-аннотацией. Контекстуальное отступление — с сигнализацией (6). **RU шире обоих DE** — дифференциация искусственна: одно RU + lang-аннотация (П.22 sprechen/sagen → оба «говорит»). **Капитализация по контексту**: `Gott` → «Бог» (монотеист., П.1) vs «бог» (политеистич./общефилос., П.22 с «Götter» — строчная). **Конвенция `erst`**: в `nun erst`, `jetzt erst`, `in... erst` — временной контраст, не первичность: «лишь»/«только» (= не раньше, только сейчас), **не «впервые»**; относится к обстоятельству, не объекту.

29. **Регистр двунаправленно, переворот за 200 лет.** (а) Возвышенное DE → возвышенное RU; нейтральное → нейтральное; не «украшать» поэтизмами обычное у Müller'а. Проверять DWDS (gehoben/dichterisch/veraltet) + Adelung + Grimm: DWDS `veraltet` отражает совр. язык, в 1820-х могло быть стандартным (`ward`). Ловушки без основания: «око», «знать», «грезить», «вещать». Пример: `Angesicht` (DWDS gehoben) → «лик» vs `Gesicht` → «лицо». (б) **Переворот за 200 лет**: регистр некоторых слов изменился — `hell` (Adelung edler) / `licht` (Adelung «im gemeinen Leben»); совр. DWDS наоборот. **Доверять Adelung 1811 для регистра 1820-х.**

30. **Предлог/направит. наречие глагольной конструкции — в рамку с глаголом.** `suchen nach`, `fragen nach` → `{"ru":"ищу","de":"such'... nach"}`. То же для `her`, `hin`, `herauf`, `hinaus` (`Von der Straße her ein Posthorn klingt` → `her... klingt` с глаголом, `Von` — отдельный предлог). **Но отделяемая приставка с ясным RU-предлогом — отдельный сегмент под предлог** (`sieht... an → «на»` П.24, `seh's... an → «по»` П.19); связанная в причастии (`angesehn` П.23) — в рамке, RU-предлог через пустой `de`.

32. **Глагольные приставки несут смысл.** Zer- = разрушение (zerbrechen); Er- = завершение (ersterben — угаснуть); Ver- = изменение/утрата. Не переводить только корень: zerschmelzen ≠ schmelzen.

34. **Futurisches Präsens; Futurperfekt через Präsens Perfekt.** (а) Wenn+Präsens → RU **будущее**: `Wenn die Gräser sprossen wollen` = когда захотят прорасти. **V1+явный Nom без wenn/als** — почти всегда бессоюзное условие или Folgesatz после императива; в RU «Если/Когда/И» с пустым `de` (18): `Spielt der Wind mit meinem Blatte` → «Когда играет ветер...». Исключение: V1 как ответ на вопрос по рифмовке. (б) **Futurperfekt через Präsens Perfekt + наречие будущего**: Aux+Partizip II + `morgen`/`bis morgen`/`übermorgen` = vollendete Zukunft; RU **будущее**, не Präteritum (признак: прошедшее при наречии будущего нелогично): `Und morgen früh ist alles zerflossen` → «И завтра утром всё **растечётся**».

35. **Спектр значений: абстрактное/предметное; классические/библейские аллюзии.** (а) Контекст определяет: `Angedenken` = воспоминание+сувенир (с `mitnehmen` → предметное); `Fluss` — общее, контекст (прозрачная, замерзает, Bach) → «речка». **Предлоги многозначны**: `Durch` = «через» (поперёк) или «по» (вдоль) — `Durch des Bergstroms trockne Rinnen` = вдоль русел. (б) **Аллюзии — сохранять узнаваемость**: `Jeder Strom wird's Meer gewinnen` = Экклезиаст → «Каждая река обретёт море».

36. **Cross-distribution: смыслы распределены по-разному.** Если DE-конструкция и RU делят смысл не параллельно — не писать ложных пар: `Ob's... wohl` ↔ «Уж не... ли» (`ob`=«не...ли», `wohl`=«уж»); вся DE-конструкция в рамку с многоточием, лишняя RU-частица — пустой `de`, с lang-аннотацией.

37. **Связность ландшафтной сцены.** Проверять согласованность слов с общей картиной: «скальные лощины» (пологие) ≠ «горная река»+«вниз» — один неточен.

40. **AcI (Acc+Inf/Partizip) в RU невозможен.** Передавать: (а) **«как»-клаузой** (`Fühlst deinen Wurm... sich regen → Чувствуешь, как твой червь встрепенётся`); или (б) **двоеточием** (`Einen Weiser seh' ich stehen → Один указатель вижу я: стоит`) — стандарт RU после глаголов восприятия. **Предпочесть (б)** при меньшем числе пустых de. Обе формы создают новую клаузу → «свой» через границу не работает (14).

41. **Возвратные RU + `Mit` — сохранять «с».** `Mit+Dat+sich-verb`: `Mit` = comitative/manner, не чистый инструменталь; RU возвратные (встрепенуться, обернуться) не принимают чистый творительный — «С горячим жалом встрепенётся».

42. **Не приписывать оригиналу переклички из перевода.** Если RU создал параллель (`unwirtbarem → бесприютном` и `Obdach → приют`), проверить DE: разные корни? Если да — параллели у Müller'а нет.

43. **Перед переводом значимого слова — grep по всем песням** (`grep '"de": "<слово>"' src/data/songs/*.json`): следовать конвенции (28) либо отступить с сигнализацией. Также — другие морф. формы корня в этой же песне (Wegweiser↔Weiser): различать.

44. **Избегать RU конструкций с поверхностной омографией**: «так, как» vs «так как»; «что бы» vs «чтобы»; «то же» vs «тоже»; «также» vs «так же». Для `so wie` — «Такими, какими…», «Как…», «Подобно тому, как…». Двусмысленные на беглом восприятии — переформулировать.

45. **Компаунды через дефис для непереводимых корневых семей/многослойных значений.** RU без однословного эквивалента — 2-3+ компонентов через дефис. Когда: (а) многозначность важна; (б) ни один синоним не покрывает всё; (в) контекст требует всей палитры. Все компоненты склоняются согласованно. Любая часть речи: сущ. (`Traum → сон-мечта`, `Täuschung → обман-иллюзия`), глаг. (`erlaben → освежать-услаждать`), прил. (`elend → жалок-несчастен-обездолен`).

46. **«Дословно»/«буквально» vs «прямой перевод».** «Дословно X — Y» требует корневого + грамматически параллельного соответствия (Apfel → яблоко). Корни/морфология не совпадают, но Y — однословный эквивалент → «прямой перевод» (Träumer → «сновидец»). При частичном покрытии — оговорка.

47. **В lang-аннотациях цитировать фактическую форму строки, не лемму** (лемма отдельно: `Глагол nachdenken — ...`, не атрибутируется Müller'у). **Перед оформлением стандартной lang-аннотации** (отрицание, sollen, AcI, эллипсис) — посмотреть как в других песнях того же типа и единообразить `segment_range` (обычно узкий, на DE-слово) и формулировку. Внутри одной аннотации все RU-формы одного DE-слова — единообразны.

48. **Структурные утверждения о цикле проверять чтением песен.** «X — последнее появление Y», «впервые…» — текст может опровергнуть; проверка: grep + чтение; различать формы (`lassen` и `verlassen`); перекрёстные ссылки — программно (Python). Утверждения об историч. изданиях — по нескольким независимым источникам.

50. **Dass-Ausrufesatz: `Ach, dass... so [adj]!` — Klage** (сетование, не удивление; к 1820-м архаично). Перевод: «Ах, что [Subj] так [Adj]!» + усилительная «же» (dass→что, so→так; «же» — пустой `de`): `Ach, dass die Luft so ruhig!` → «Ах, что же воздух так спокоен!» (П.12).

54. **Русская типографика** (аннотации, заголовки, `ru`): «...» ёлочки (не `"..."`); — длинное тире (не `-`; дефис только в компаундах); **ё** обязательна (ещё/всё/её/идёт/найдёт/чётко/своё/нём); одинарный пробел; многоточие `...`; «не что иное, как» (раздельно, конкретизация) vs «ничто иное» (слитно, отрицание). **Обязательная ё-проверка после каждой записи** (regex: `\bнем\b|\bеще\b|задает|осознается|передает|\bидет\b|найдет|\bчетко\b|твердой|замерзш|\bвсем\b|\bсвое\b`). **«То есть»** — союз: запятая перед, не после (Розенталь §23); «по смыслу» — не вводное; паттерн «Дословно: «X», то есть по смыслу «Y»» (П.01/04). RU-грамматика: `old-rozental.ru` ✅ (§22 уточняющие, §23 пояснительные, §34 союзы); `gramota.ru` ❌ 403.

55. **Следственное `dass` без явного `so`/`derart`** (литер. архаика XIX в.) → RU «**так что**» (без якоря RU «что» читается как комплементное). Контраст: `so lau, dass…` (явный `so`) → «столь… что…»; `Hab' wieder schwarze Haare, dass mir's… graut` (без `so`) → «...чёрные волосы, **так что** мне… страшно».

56. **Сигнализация о подменах предметов/концептов** (доп. к 6). Если RU-эквивалент означает иной материальный предмет, чем DE (`Bahre` = открытые носилки vs «гроб» = закрытый ящик-Sarg) — **сигнализировать** при первоначальном переводе до записи. Особенно при социально-историческом подтексте.

58. **Каждая стихотворная строка — с прописной** (стихотворная конвенция > синтаксической; при сегментации первого слова легко получить строчную). Проверка после записи: первый сегмент каждой строки.

59. **Грамматическая модальность в аннотациях.** Сохранять модальность DE: V1+`?` → «спрашивает»; императив (`lass...!`) → «просит/велит»; декларатив → «говорит». Не растворять разные модальности в «герой обращается».

61. **Сверка DE по типу расхождения с авторитетным источником**:
- Лексика, артикли, падежи → NSA + автограф.
- Экспрессивная пунктуация (`!`/`?`/`;`/`—`) → Müller + Rastl (NSA в нотах упрощает).
- Запятая/точка/двоеточие → Müller + Rastl + NSA.
- Апострофы → Rastl (NSA опускает; П.22 L4 sung «sing», но писать `Sing'`).
- Поэт. капитализация Müller'а (`Einem/Andern` П.1, `Manches`/`Alles` П.17) → Müller + Rastl (NSA в нотах строчные — нотная конв.).
- Шубертовские повторы (27) → NSA + Naxos.
**Naxos — НЕ источник пунктуации** (12+ опечаток). Программная проверка: вывести все lines_de и сверить.

62. **Winterreise — психологический цикл, не натуралистический.** Природные приметы не всегда согласованы реалистически: `bunte` листья (П.16) vs замёрзшие речки/слёзы (П.3,6,7) — противоречие в символическом плане (Müller не следует строгой хронологии). Не делать необоснованных утверждений о «глубокой зиме»/«поздней осени» — учитывать **все** приметы.

63. **Финальная систематическая проверка перед закрытием песни**: (а) каждый сегмент DE↔RU пословно; (б) факт-чек **каждого** утверждения аннотаций, вкл. подразумеваемое/отсылки/реалии (отсылки — grep; DE-цитаты, научные/историч./литературовед. факты, атрибуции — авторитетные источники: **аннотация = источник галлюцинаций**); (в) ё (54); (г) ASCII-кавычки `re.findall(r'"[А-Яа-яёЁ]')`; (д) короткие дефисы между RU-словами; (е) капитализация первых сегментов (58); (ж) JSON; (з) пунктуация DE с источником (61); (и) отображение (порядок сносок, вариант, spanning-подсветка); (к) список спорных мест до объявления финализированной.

## Annotation Guidelines

Два типа: **`lang`** (грамматика — синий) и **`meaning`** (смысл — коричневый); нумерация независимая. **Lang** — Verbklammer, порядок слов, падежи, род, отрицание, идиомы, составные слова. **Meaning** — этимология, мотивы, многозначность, соц. контекст, Schubert vs Müller с эффектом.

**Стиль**: не категорично; без доказательств не утверждать; **не ссылаться на учёных по именам**; **не упоминать словари по именам** («по DWDS»/«по Adelung'у» — нет); регистровые пометы не цитировать буквально; источники — для фактчека, не для текста; **о тексте, не об учебнике** (без лингв. терминологии «Konjunktiv II выражает...»). **Каждый факт проверять** (21, 47); meaning-аннотации с реалиями (научные/историч./литературовед.) — **каждое утверждение и отсылку, вкл. подразумеваемое, в авторитетных источниках до пуша** (галлюцинации); смысл песни сверять с **немецким литературоведением** (Schubert/Müller-Forschung).

**Технические**:
- `line_span: N` — охват N строк (сноска в конце последней; может пересекать строфы).
- `continuation_ranges: [[start,end], ...]` — диапазоны для каждой следующей строки; `[-1,-1]` пропускает строку без подсветки (П.19 L2, П.22 stz1 spricht/sagt); не `null`.
- `segment_range: [start,end]` — включительно.
- Аннотации могут перекрываться (lang+meaning, а также два meaning на одном сегменте). **Но сноска на сегменте — одна** (кроме variant): спанящая ставит сноску на последний сегмент охвата и вытесняет локальную (`InterlinearLine.vue`) — учитывать в `segment_range` (П.24 stillstehen → `[0,0]`, `[3,3]` занят спанящей meaning).
- `\n` в тексте — перенос/разделитель абзацев (П.12, 14, 19, 22, 23).
- **Ударение в RU** — combining acute U+0301 после ударной гласной: `во́рон`, `что́`.
- **Курсив** — `*текст*` → `<em>` (рендер `src/utils/renderText.js` в AnnotationsPanel + tooltip SongView; HTML экранируется; для кириллицы Unicode-курсива нет; П.22 «*против*»).
- **`title_annotations`** — массив `{type,text}` без `segment_range`: композиц./смысловые комментарии к песне целиком; meaning на всю песню — через `line_span`+`continuation_ranges` на все строки (П.22). Примеры: П.12, 13, 14, 22.

**Формат сегментов**:
- Отделяемые приставки: `"de":"pfiff'... aus"` (многоточие при разрыве). Рамки предлогов: `"de":"Auf... Haus"`.
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{"ru":"женщины образ.","de":"Frauenbild"}`.
- Пустой `ru` — артикль без эквивалента. Пустой `de` — RU-добавка для грамматики.
- **Пунктуация только в `ru`, не в `de`**: `{"ru":"ах,","de":"ach"}`.
- **`variant_de`/`variant_ru`** — sung-расхождения Schubert vs Müller (22). Основное — Schubert sung-text; variant — только различающееся слово (общее — в основном de): `{"de":"den Wegen","variant_de":"Straßen"}` (П.20). Общая пунктуация — только в основном RU (не дублировать в variant_ru: П.4 «угасшее,»/«замёрзшее», П.20 «путях,»/«дорогах», П.23 «Ах,»/«Да»). Обычная (не `target:variant`) аннотация на сегменте с `variant_ru` подсвечивает оба слова (П.23 L7 «ах»+«да»).
