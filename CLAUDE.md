# CLAUDE.md — Winterreise

## Project Overview

Интерактивная веб-страница с подстрочным переводом вокального цикла «Зимний путь» (Winterreise) Вильгельма Мюллера / Франца Шуберта. 24 песни, немецкий оригинал + русский перевод с пословной разметкой и аннотациями.

Хостинг: GitHub Pages. Фреймворк: Vue 3 + Vite.

## Architecture

### Страница (публичный режим)

```
┌──────────┬────────────────────────────────────┐
│ Меню     │  Die Wetterfahne · Флюгер     [☀]  │
│          │                                    │
│ 1. Gute  │  ┌─ Оригинал ──┬─ Перевод ───────┐│
│    Nacht │  │              │                 ││
│ 2. Die   │  │ Der Wind     │ Ветер   играет  ││
│  ►Wetter │  │ spielt mit   │ Der W.  spielt  ││
│    fahne │  │ der Wetter-  │                 ││
│ 3. ...   │  │ fahne        │ с    флюгером   ││
│          │  │              │ mit  der Wetter. ││
│          │  │ Auf meines   │                 ││
│          │  │ schönen      │ На доме моей  ¹ ││
│          │  │ Liebchens    │ Auf  meines     ││
│          │  │ Haus.        │ прекрасной милой││
│          │  │              │ schönen Liebch. ││
│          │  │              │ Haus            ││
│          │  └──────────────┴─────────────────┘│
│ [Поиск]  │  ── Пояснения ──────────────────── │
│          │  ¹ «моей прекрасной милой» — ...    │
└──────────┴────────────────────────────────────┘
```

- **Левая колонка**: немецкий оригинал (строфами, как в стихотворении)
- **Правая колонка**: русский перевод, под каждым русским словом/группой — соответствующее немецкое слово/группа мелким шрифтом
- **Сноски**: нумерованные (`¹`, `²`, `³`...) у отмеченных фрагментов. Нумерация сквозная в пределах песни. Наведение — подсветка текста, который был заранее аннотирован как фрагмент. Клик — всплывающее пояснение
- **Режим пояснений**: внизу страницы все аннотации песни с указанием, к какому месту относятся
- **Меню слева**: 24 песни, немецкие названия, кликабельное
- **Заголовок песни**: немецкое + русское название
- **Поиск**: по всем текстам (оригинал + перевод)
- **Тема**: переключатель светлая/тёмная
- **Печать/PDF**: экспорт всех песен, с аннотациями

### Админ-режим

Доступ по секретному URL: `?admin=<SECRET_KEY>`.

Возможности:
- Редактирование русского текста (inline)
- Редактирование немецкого подстрочника под каждым русским словом (inline)
- Выделение группы русских слов → добавление сноски (номер + пояснение)
- Удаление/редактирование существующих аннотаций
- Сохранение: коммит через GitHub Contents API (токен передаётся в URL или хранится в localStorage)

Два пути редактирования:
1. **Браузер**: админка с inline-редактированием
2. **Claude Code**: правка JSON-файлов данных в CLI

### Data Model

Каждая песня — JSON-файл в `src/data/songs/`:

```json
{
  "number": 2,
  "title_de": "Die Wetterfahne",
  "title_ru": "Флюгер",
  "stanzas": [
    {
      "lines_de": [
        "Der Wind spielt mit der Wetterfahne",
        "Auf meines schönen Liebchens Haus."
      ],
      "lines_ru": [
        {
          "segments": [
            { "ru": "Ветер", "de": "Der Wind" },
            { "ru": "играет", "de": "spielt" },
            { "ru": "с", "de": "mit" },
            { "ru": "флюгером", "de": "der Wetterfahne" }
          ],
          "annotations": []
        },
        {
          "segments": [
            { "ru": "На доме", "de": "Auf... Haus" },
            { "ru": "моей", "de": "meines" },
            { "ru": "прекрасной", "de": "schönen" },
            { "ru": "милой.", "de": "Liebchens" }
          ],
          "annotations": [
            {
              "segment_range": [1, 3],
              "text": "Liebchen — ласковое обращение, буквально «милочка». Герой говорит о возлюбленной, из чьего дома он уходит."
            }
          ]
        }
      ]
    }
  ]
}
```

**Правила данных:**
- `segments` — массив пар {ru, de}. Порядок = порядок слов в русском переводе
- `de` может быть пустой строкой (слово добавлено для русского смысла, нет немецкого аналога)
- `segment_range` — индексы [start, end] включительно, указывают какие сегменты подсвечиваются
- Одна строка `lines_ru` соответствует одной или нескольким строкам `lines_de` (не обязательно 1:1)

## Translation Rules (Правила подстрочного перевода)

Подстрочник — это не литературный перевод, а пословное соответствие для изучающих немецкий.

**ВАЖНО: Порядок работы над переводом.**

Перед тем как предложить перевод, для КАЖДОГО знаменательного слова (существительное, глагол, прилагательное, наречие):

1. **Проверить в DWDS** (dwds.de/wb/СЛОВО): все значения, регистр (gehoben, dichterisch, veraltet), этимологию. Не полагаться на первую ассоциацию — она часто ошибочна (irre ≠ безумный, trübe ≠ сумрачный, Flur ≠ луг).
2. **Разобрать на части**: приставка + корень. Каждая часть несёт смысл. Er-sterben ≠ sterben, zer-schmelzen ≠ schmelzen, durch-dringen ≠ dringen.
3. **Проверить, нет ли в песне близких слов**, которые нужно развести (Boden/Erde, Gras/Rasen, Blüte/Blumen).
4. **Проверить точность смысла.** Пословный перевод часто искажает значение. Сначала понять, что означает немецкая строка целиком, потом найти русское выражение с тем же смыслом. Пример: «Kalt starrt ihr Bild darin» — «kalt» описывает состояние образа (он холодный), а не способ действия. «Холодно застыл» — неточно (наречие образа действия), «холодным застыл» — точно (инструментал состояния). Другие типичные ошибки: «на моей руке» вместо «под руку» (идиома), «прошла зелёным лугом» (неестественная конструкция).
5. **Если выбор неочевиден — сигнализировать.** Не делать молча. Написать: «здесь два варианта, вот аргументы за каждый». Пользователь решает. Примеры мест, требующих обсуждения: идиомы, архаизмы, слова с несколькими значениями, расхождение регистра.

**Немецкий текст берём из шубертовской редакции (как поют), а не из оригинала Мюллера.** Шуберт местами менял порядок слов и отдельные строки. Источник: Deutsche Schubert-Lied-Edition (Naxos). Орфографию модернизируем по реформе 1996: ß после кратких гласных → ss (Muß → Muss, Daß → Dass, Laß → Lass), но ß после долгих гласных остаётся (Blumenstrauß, weißen).

1. **Каждое немецкое слово должно быть отражено в подстрочнике.** Не пропускай служебные слова (da, schon, doch и т.д.). Каждое слово оригинала должно появиться в поле `de` какого-либо сегмента.

2. **Сохраняй немецкий порядок слов, если русский при этом звучит приемлемо.** Цель — чтобы читатель видел соответствие между строками. Если порядок слов немецкого оригинала даёт понятный русский текст — сохраняй его.

3. **Меняй порядок только когда русский становится нечитаемым.** Например, «На моей прекрасной любимой доме» — так нельзя, переставляем. Но «Тут я подумал уже в своём безумии» — нормально, сохраняем немецкий порядок.

4. **Одно русское слово = одно немецкое слово (по возможности).** Группируй только когда иначе невозможно. Отделяемые приставки: если приставка имеет ясный русский эквивалент — отдельный сегмент (dahin → «вдаль», mit → «с собой»). Рамка с многоточием (pfiff'...aus, bin...eingezogen) — только когда русский не разделяется на части.

5. **`de` может быть пустой строкой** только для слов, добавленных для русского смысла и не имеющих соответствия в оригинале.

6. **Liebchen = любимая** (не «милая»). Lieb- от Liebe (любовь).

7. **Аннотации: всегда показывай текст пользователю перед добавлением в JSON.** Не коммить аннотации без утверждения. Показать черновик → дождаться «ок» или правок → только тогда записать и запушить. Это правило нарушается чаще всего — следить за ним особо.

8. **В аннотациях всегда переводи немецкие фрагменты.** Если цитируешь немецкий текст в пояснении, рядом давай русский перевод. Например: «(es eher bemerken — это раньше заметить)», а не просто «(es eher bemerken)».

9. **Артикли (ein, eine, der, die, das) не переводятся как «некий/некая/этот».** Варианты: (а) объединить с соседним словом в одном сегменте: `{ "ru": "верной", "de": "ein treues" }`, `{ "ru": "богатая", "de": "eine reiche" }`; (б) пустой сегмент `{ "ru": "", "de": "ein" }` если объединить не получается. Артикли при существительных (der Wetterfahne, den Herzen) группируются с существительным или прилагательным.

10. **Предлоги — отдельные сегменты.** Всегда разделяй: `{ "ru": "с", "de": "mit" }` + `{ "ru": "сердцами", "de": "den Herzen" }`, а не `{ "ru": "с сердцами", "de": "mit den Herzen" }`. То же для «в/in», «на/auf», «о/nach» и т.д.

11. **Связка sein в именном сказуемом** → тире «—» в русском: `{ "ru": "—", "de": "ist" }`. Не переводить как «есть» или «является». Правило распространяется на все формы настоящего времени (ist, seid, sind). Но прошедшее время (war) и сослагательное наклонение (wär') требуют русского глагола: «был», «было бы» — тире не передаёт ни времени, ни наклонения.

12. **Идиомы переводятся по смыслу**, а не буквально. При этом обязательна языковая аннотация с объяснением. Пример: «Was fragen sie nach» → «Что за дело им до» + аннотация про идиому fragen nach.

13. **Время в косвенной речи.** В русском нет согласования времён. После «подумал, что...» действие, одновременное с мыслью, передаётся настоящим временем: «подумал, что он освистывает» (не «освистывал»).

14. **Возвратное притяжательное.** «Meinem» → «своём» (не «моём»), когда подлежащее — «я». В русском «в своём бреду» естественнее, чем «в моём бреду».

15. **Адаптация грамматического рода.** Если род слова различается в немецком и русском, подстрочник следует русскому роду: Sie (она, о Wetterfahne ж.р.) → он (о флюгере, м.р.). Если это создаёт значимое расхождение с оригиналом, добавить языковую аннотацию.

16. **Порядок слов при составных глаголах (Verbklammer).** Когда в немецком вспомогательный глагол и содержательный образуют рамку вокруг подлежащего (bin ich eingezogen, hat sie gemacht), в русском подлежащее ставится перед содержательным глаголом — как в немецком. Вспомогательный глагол объединяется с содержательным через многоточие в поле `de` (как отделяемые приставки). Пример: «Fremd **bin** ich **eingezogen**» → `{ "ru": "я", "de": "ich" }, { "ru": "пришёл", "de": "bin... eingezogen" }` — я перед пришёл, как ich перед eingezogen. Не «Чужим пришёл я».

17. **Предлог, поглощённый падежом.** Если немецкий предлог не имеет эквивалента в русском, потому что его роль выполняет падеж, — сливаем предлог с соседним словом в один сегмент. Пример: «gehüllt **in** Schnee» → «окутан снегом» — `{ "ru": "снегом", "de": "in Schnee" }`, потому что русский творительный падеж поглощает «in».

18. **Никаких пустых сегментов (ru = "").** Если немецкое слово не имеет русского эквивалента, оно сливается с соседним словом в поле `de`. Не создавать сегменты с пустым `ru`. Примеры: `Es` (формальное подлежащее) → `{ "ru": "Следует", "de": "Es zieht... mit" }`; `man` (безличное) → `{ "ru": "меня", "de": "man mich" }`; вспомогательные `bin/hat/hab'` → многоточие с содержательным глаголом: `{ "ru": "пришёл", "de": "bin... eingezogen" }`. Пустой `ru` допустим только в поле `de` сегмента с непустым `ru` (т.е. `{ "ru": "Что", "de": "" }` — ОК, это русское слово без немецкого аналога).

19. **Модальные глаголы — учитывать контекст.** Sollen во втором лице (du sollst) выражает не обязанность, а волю говорящего (обещание, заверение). Не переводить механически как «должен/должна». Пример: «Sollst meinen Tritt nicht hören» → «Моих шагов не услышишь» (заверение), а не «Должна моих шагов не слышать» (обязанность).

20. **Эллиптические конструкции.** Немецкий допускает опущение глагола там, где русский не допускает. Пример: «Sacht, sacht die Türe zu!» — глагола нет, только частица «zu» (закрыто). В русском необходимо добавить глагол («прикрою»). Обязательна языковая аннотация.

21. **Не угадывать значение слов — проверять.** Fein ≠ милый (fein = прекрасный, изящный). Glühend ≠ обжигающе (glühend от glühen = пылать, тлеть, как угли; правильно — «пылающе»). Первая ассоциация может быть ошибочной. Всегда проверять корень слова и словарь (см. порядок работы выше).

22. **Различия Шуберта и Мюллера отмечать в аннотациях.** Если Шуберт изменил текст Мюллера (порядок слов, отдельные слова), это отмечается в смысловой аннотации с указанием оригинала Мюллера и года (цикл «Die Winterreise», 1824). Если различие связано не с правкой Шуберта, а с тем, что Шуберт работал по раннему изданию (Urania, 1823), а Мюллер позже изменил текст в книжном издании (1824), — указывать это точно: «В издании Urania (1823), по которому работал Шуберт: ...», а не «Шуберт изменил».

23. **Элизии (апострофы) в немецком тексте.** Все поэтические сокращения обозначаются апострофом — последовательно в `lines_de`, в `title_de` и в поле `de` сегментов. Примеры: zieh' (ziehe), dacht' (dachte), hätt' (hätte), hab' (habe), Wär' (Wäre), Ruh' (Ruhe), Schreib' (Schreibe), Such' (Suche), Eh' (Ehe), pfiff' (pfiffe, Konjunktiv II), Gefror'ne (Gefrorene), An's (An das). Если элизия стоит перед многоточием (рамка или приставка), апостроф сохраняется: `pfiff'... aus`, `hab'... gedacht`. Не пропускать апостроф нигде — иначе возникает непоследовательность между `lines_de` и полем `de`.

24. **Перевод через отрицание при расхождении коннотаций.** Если немецкое слово несёт другую коннотацию, чем его прямой русский аналог, — переводить через отрицание. Пример: lau (тепловатый, в немецком акцент на «недостаточно горячий») → «негорячи», а не «теплы» (русское «тёплый» звучит положительно). Принцип: so lau — не «очень тёплый» (приближает к горячему), а «очень негорячий» (удаляет от горячего).

25. **«So» в конструкции «so...dass» — всегда степень.** Переводится как «так» или «настолько», никогда как «едва/еле». Направление (много/мало) задаётся контекстом и следствием в dass-придаточном, а не словом «so».

26. **Konjunktiv II передавать в русском.** Немецкий Konjunktiv II (hätte, wäre, wollte, könnte...) нельзя переводить настоящим временем. В независимых предложениях: прошедшее время + «бы» (хотели бы, были бы). После «словно/как будто»: прошедшее время, с «бы» или без (словно хотели бы). Пример ошибки: wolltet → «хотите» (неверно), правильно → «хотели бы».

27. **Варианты при музыкальном повторе.** Если Шуберт при повторе строфы меняет текст, оба варианта указываются через ` / ` в `lines_de` и в соответствующих полях `de` и `ru` сегментов. Пример: `"Und / Ihr dringt doch aus der Quelle"`, `{ "ru": "А / Вы", "de": "Und / Ihr" }`. Первый вариант — основной текст, второй — текст повтора.

28. **Разводить близкие по смыслу немецкие слова.** Если в одной песне (или в соседних) два немецких слова покрывают близкое значение — в русском переводе они должны различаться. Пример: Boden → «почва», Erde → «земля» (оба могут быть «земля», но не должны). Flur ≠ Wiese ≠ Feld — разные слова, разные переводы.

29. **Соответствие регистра.** Если немецкий текст использует возвышенную или устаревшую лексику (durchstreichen вместо durchqueren, die Flur вместо das Feld), русский перевод должен соответствовать по регистру — использовать поэтические/возвышенные эквиваленты, а не нейтральные. Проверять регистр слова в DWDS или Duden (пометки gehoben, dichterisch, veraltet).

30. **Предлог, принадлежащий глаголу, — в рамку с глаголом.** Когда предлог относится к глагольной конструкции (suchen nach, fragen nach), он отражается в поле `de` глагола через многоточие: `{ "ru": "ищу", "de": "such'... nach" }`. Не группировать такой предлог с управляемым существительным (Nach... Spur выглядит как отделяемый глагол nachspüren).

31. **Проверять грамматические утверждения по авторитетным источникам.** Перед тем как писать в аннотации, что две немецкие конструкции различаются по смыслу, — проверить в DWDS, E-VALBU (IDS Mannheim), Duden. Первая ассоциация может быть ложной (пример: «suchen nach подразумевает неуверенность» — не подтверждается словарями; реальное различие — nach подчёркивает интенсивность поиска, E-VALBU).

32. **Немецкие глагольные приставки несут смысл — переводить приставку.** Zer- = полное разрушение (zerbrechen — разбить, zerstören — разрушить). Er- = завершение процесса (ersterben — угаснуть, erstarren — оцепенеть). Ver- = изменение/утрата. Нельзя переводить только корень, игнорируя приставку. Примеры: zerschmelzen ≠ schmelzen (истопить ≠ растопить), erstorben ≠ tot (угасло ≠ мертво).

33. **Sein + Partizip II от глаголов изменения состояния = Перфект, не связка.** «Die Blumen sind erstorben» = цветы угасли (Perfekt), а не «цветы — мертвы» (связка + прилагательное). Правило 11 (sind → тире) не применяется к Перфекту. Признак: если причастие от глагола изменения (ersterben, einziehen, aufstehen), это Перфект.

34. **Проверять весь спектр значений слова, включая конкретные.** Слово может иметь и абстрактное, и предметное значение. Контекст определяет, какое ближе. Пример: Angedenken = и воспоминание (абстрактное), и памятка/сувенир (предметное). Глагол mitnehmen (взять с собой) указывает на предметное. Irre = и безумный (от Irrsinn), и бродячий (от irren — блуждать). Контекст (бездомные псы воют) указывает на «бродячий».

## Annotation Guidelines (Правила аннотаций)

Два типа аннотаций: `lang` (язык, грамматика — синий) и `meaning` (смысл, интерпретация — коричневый). Нумерация независимая для каждого типа.

### Языковые аннотации (lang)

Объясняют грамматические и структурные различия между немецким и русским:
- **Рамочные конструкции** (Verbklammer): hätt'...sollen, hätt'...wollen — вспомогательный и модальный глаголы образуют рамку
- **Порядок слов**: Klammerstruktur (Auf...Haus), V2 vs V-last, отделяемые приставки (pfiff'...aus)
- **Падежи**: Genitiv (Des Hauses), приложения
- **Род**: различия рода между языками (Wetterfahne ж.р. → флюгер м.р.)
- **Отрицание**: двойное отрицание в немецком = утверждение (nimmer nicht), в русском = усиление (никогда не)
- **Опускание союзов**: dass после глаголов мышления
- **Идиомы**: объяснение немецких идиом и их буквального vs реального значения
- **Составные слова**: Frauenbild = Frauen + Bild, словообразование

### Смысловые аннотации (meaning)

Литературная интерпретация, контекст, наблюдения:
- **Этимология**: словообразование, связи между словами (Wetterfahne — wetterwendisch)
- **Перспектива**: смена лица (я → он → они), самоотчуждение
- **Мотивы**: ветер, флюгер, непостоянство — связи между строфами и песнями цикла
- **Многозначность**: несколько пластов смысла (сердца = любовь + расчёт родителей)
- **Социальный контекст**: критика мещанства, неравенство статусов
- **Фольклорные формулы**: устойчивые обороты немецкой народной песни (Feinsliebchen и т.п.) — объяснять традицию, не подавать как индивидуальный авторский выбор
- **Различия Шуберта и Мюллера**: если Шуберт изменил текст, отмечать оригинал и то, как изменение влияет на смысл или музыкальный акцент
- **Перед написанием** — искать, что пишут литературоведы на немецком и английском
- **Не быть категоричным** — если есть несколько интерпретаций, признавать это
- **Не утверждать без доказательств** — не подавать интерпретации как факты (герой — подмастерье? из текста не следует)
- **Не ссылаться на учёных по именам** — не «по Гримму», а формулировать как собственное наблюдение
- **Писать о тексте, а не об учебнике** — аннотация комментирует конкретное место в песне, а не объясняет немецкую грамматику вообще. Никакой лингвистической терминологии (не «делиберативное», «предложное управление», «переходный глагол»). Объяснять так, чтобы понял заинтересованный читатель, а не лингвист
- **Проверять каждое фактическое утверждение** о немецком языке по авторитетным источникам до включения в аннотацию (см. правило 31)

### Технические правила аннотаций

- `line_span: N` — аннотация охватывает N строк (текущую + последующие). Сноска появляется в конце последней охваченной строки
- `continuation_ranges: [[start, end], ...]` — массив диапазонов сегментов для каждой последующей строки (line+1, line+2, ...). Позволяет ограничить подсветку и позицию сноски на последней строке (например, закончить аннотацию на слове «дверь», не захватывая «прикрою»). Если не указан, подсвечиваются все сегменты последующих строк
- Аннотации могут перекрываться: один сегмент может принадлежать и lang, и meaning аннотации
- `segment_range: [start, end]` — включительно, индексы сегментов в строке

### Формат данных (сегменты)

- Отделяемые приставки: `"de": "pfiff'... aus"` (с многоточием)
- Рамки предлогов: `"de": "Auf... Haus"` (с многоточием)
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{ "ru": "женщины образ.", "de": "Frauenbild" }`
- Пустой `ru` (`""`) — немецкое слово без русского эквивалента (артикль)
- Пустой `de` (`""`) — русское слово без немецкого эквивалента (добавленное «что» для dass)
