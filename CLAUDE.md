# CLAUDE.md — Winterreise

## Project Overview

Подстрочный перевод вокального цикла «Зимний путь» (Winterreise) Мюллера / Шуберта. 24 песни, немецкий + русский с пословной разметкой и аннотациями. Vue 3 + Vite, GitHub Pages.

## Architecture

### Страница

- **Левая колонка**: немецкий оригинал (строфами)
- **Правая колонка**: русский перевод; под каждым словом мелким шрифтом — соответствующее немецкое
- **Сноски** `¹²³` у отмеченных фрагментов; сквозная нумерация в пределах песни. Наведение — подсветка фрагмента, клик — пояснение
- **Режим пояснений** внизу страницы — все аннотации песни
- **Меню**: 24 песни (немецкие названия)
- **Заголовок песни**: немецкое + русское название
- Поиск, переключатель темы, печать/PDF

### Админ-режим

URL `?admin=<SECRET_KEY>`. Inline-редактирование русского/немецкого, добавление/правка аннотаций, сохранение через GitHub Contents API (токен в localStorage, права `repo`). Параллельно — правка JSON в Claude Code CLI.

### Data Model

JSON по одной песне в `src/data/songs/`:

```json
{
  "number": 2,
  "title_de": "Die Wetterfahne",
  "title_ru": "Флюгер",
  "stanzas": [{
    "lines_de": ["Der Wind spielt mit der Wetterfahne"],
    "lines_ru": [{
      "segments": [
        { "ru": "Ветер", "de": "Der Wind" },
        { "ru": "играет", "de": "spielt" },
        { "ru": "с", "de": "mit" },
        { "ru": "флюгером", "de": "der Wetterfahne" }
      ],
      "annotations": [
        { "type": "lang", "segment_range": [3, 3], "text": "..." }
      ]
    }]
  }]
}
```

**Правила данных:**
- `segments` — пары {ru, de} в порядке русского
- `de` может быть пустой строкой
- `segment_range` — индексы [start, end] включительно (подсветка)
- `lines_ru.length` == `lines_de.length` (рендер парит по индексу). Verbklammer/рамка через границу строк — одним сегментом-рамкой в одной из lines_ru (см. правило 16)
- `title_annotations`: массив `{ type, text }` для комментариев к названию/песне в целом (см. песни 12-14); без `segment_range`

### Tech Stack & Structure

Vue 3 (Composition API, `<script setup>`) + Vite. Чистый CSS, plain JS (без TS, без UI-фреймворков). Деплой GitHub Actions → GitHub Pages.

- `src/data/songs/*.json` — песни (01–24)
- `src/data/index.json` — метаданные (number, title_de, title_ru, ready)
- `src/components/` — SongList, SongView, InterlinearLine, FootnoteMark, AnnotationsPanel, SearchBar, ThemeToggle; `admin/` (InlineEditor, AnnotationEditor, SaveButton)
- `src/styles/` — main.css, print.css, theme.css
- `src/utils/` — github-api.js, search.js, print.js
- `.github/workflows/deploy.yml`

## Translation Rules

Подстрочник — пословное соответствие для изучающих немецкий, не литературный перевод.

**Точность — приоритет.** Русский должен быть грамотным и реально существующим (проверять сочетаемость: «лежит на душе» — есть, «лежит на уме» — нет), но не обязан звучать красиво. Точное лучше привычного: `блестящих`, не `светлых`; `в споре`, не `наперебой`; `ударялся`, не `натыкался`. **При равной точности** — частотный вариант (`Bergstrom` → «горная река», не «горный поток»). **Устоявшийся русский перевод не аргумент** держаться менее точного — традиция не довод. Не добавлять отсутствующих в оригинале оттенков: уменьшительных суффиксов (`градинки` ≠ `Hagelkörner`), ложных архаизмов, аспектных значений (`взглянуть` мгновенно ≠ `sehn` нейтрально).

**Перевод с немецкого на русский. Английский язык не использовать** — ни в обсуждениях, ни в скобках. Кроме исключительных случаев.

**Порядок работы.** Для КАЖДОГО знаменательного слова (включая частицы и наречия: `immerfort` = непрестанно, не «всегда»; `je` = хоть когда-нибудь):

1. **DWDS** (значения, регистр, этимология). Для XIX в. также **Grimm** (dwds.de/wb/dwb/СЛОВО) и **Adelung 1811** (zeno.org/Adelung-1793; ближе всего к Müller'у — Grimm DWB начат только 1838). Регистр и значения дрейфуют (`starren` 1820-х → «застыть»; сейчас «уставиться»; `wild` у Adelung ядро = `ungestüm` «буйное», не «первобытно-дикое»).
2. **Разобрать на приставку + корень** (Er-sterben ≠ sterben, zer-schmelzen ≠ schmelzen).
3. **Близкие слова в песне развести** (Boden/Erde, Gras/Rasen, Blüte/Blumen).
4. **Контекст важнее этимологии, точность важнее традиции.** Сначала смысл строки целиком, потом русское выражение. Этимология может вести не туда: `irre Hunde vor ihres Herren Haus` = безумные, не бродячие (у бродячих нет хозяина). При **двух одновременных значениях** (`starrt` = и застыл, и уставился; `pfiff'...aus` = и свистит, и освистывает) — отметить оба. Наречие vs инструментал: `Kalt starrt ihr Bild darin` — «холодным застыл» (инструментал состояния), не «холодно застыл». **Частицы** (schon, doch, ja, auch, wohl, je, noch, mal, denn) меняют значение по контексту: `Schon` ≠ всегда «уже» (в `Tret' ich auch schon` — усилитель уступки); `denn` в вопросе — «же» (`Was drängst du denn?`). **Составные формулы**: `Nun ja` (уступительное «ну да») ≠ `Ach ja` (припоминание «ах да»). **Модальные глаголы в значении предположения** (`mag` от `mögen` 3 = «должно быть» / «может быть») — отдельный русский эквивалент, не растворять в инфинитиве (правило 1).
5. **Русская пунктуация.** Повторяющиеся союзы — запятая («И час, и день»). Запятые при обращениях, вводных, деепричастиях. **Не калькировать немецкую пунктуацию автоматически**: `Nun, ...` (DE — запятая после вводного) → «Теперь не далеко уже идти» (RU — обстоятельство времени, без запятой). **При замене слова перепроверить согласование**: «своё отражение» → «свой образ» (м.р.).
6. **Если выбор неочевиден — сигнализировать**, не делать молча. Идиомы, архаизмы, многозначные слова, расхождение регистра — выносить на обсуждение.

**Источник немецкого текста: шубертовская редакция** (как поют), не оригинал Мюллера. Источник: Deutsche Schubert-Lied-Edition (Naxos). Орфография по реформе 1996: ß после кратких гласных → ss (Muß → Muss); после долгих остаётся (Blumenstrauß).

1. **Каждое немецкое слово — в подстрочнике.** Не пропускай служебные (da, schon, doch). Слово с самостоятельным переводом — отдельный сегмент даже в рамке (`hinaus` → «прочь»). Но если русский предлог уже передаёт направление приставки — поглощается: `In die Felsengründe lockte... hin` → «В глуби заманил» («в» = `hin`). **Знаменательные слова (модальные, частицы, наречия) обязаны иметь русский эквивалент**: `mögen` 3 (DWDS) = «должно быть» / «может быть», не растворяться в инфинитиве. Сливать `mag` с `gehn` в рамку `de = "mag gehn"` допустимо только если модальность реально отсутствует или передана другим средством. Включение слова в `de`-поле без русского эквивалента — потеря смысла.

2. **По умолчанию сохраняй немецкий порядок слов** — чтобы читатель видел соответствие. Перестройка — только когда русский **нечитаем**, а не «слегка непривычен». **Поэтическая инверсия V-letzt** (Müller часто ставит финитный глагол в конец строки ради рифмы: `Von der Straße her ein Posthorn klingt` — рифма с `aufspringt`) сохраняется: «С улицы почтовый рожок доносится», не «доносится почтовый рожок».

3. **Меняй порядок только когда русский нечитаем.** «На моей прекрасной любимой доме» — нельзя; «Тут я подумал уже в своём безумии» — ок.

4. **Одно русское = одно немецкое (по возможности).** Группировать только когда иначе невозможно. Отделяемые приставки с ясным русским — отдельный сегмент (`dahin` → «вдаль», `mit` → «с собой»). Рамка с многоточием (`pfiff'...aus`) — когда русский не разделить. **Многоточие — только при разделении словами**; рядом стоящие (Folge nach, Komm her) — без. **Порядок в рамке = текстовый порядок DE**: `bin... eingezogen` (bin раньше); `her... klingt` (her раньше).

5. **`de` пустая** — для слов, добавленных для русского смысла без немецкого эквивалента.

6. **Liebchen = любимая** (не «милая»). Lieb- от Liebe.

7. **Не писать содержательные изменения без приглашения. Не пушить без явного «пушим»/«делаем».** Одобрение перед записью в файл (не только перед commit; файл сразу виден на dev-сервере). Порядок: предложить → «ок» → писать → показать → «пушим» → commit/push. **Аннотации «от себя» по итогам фактчека, записи `variant_de`/`variant_ru`, ретроактивные правки записанных песен — каждый случай отдельно.** Одобрение конкретного шага не распространяется ни на сопутствующие изменения, ни на правила CLAUDE.md, предлагаемые «заодно». Исключение — мелкие правки прямо запрошенные. **Условное одобрение** («если всё чётко, пушим»; «проверь и сделай») — сначала явно сообщить пользователю результат проверки/оценки, только потом действовать. Не пропускать шаг оценки.

8. **В аннотациях переводи немецкие фрагменты.** «(es eher bemerken — это раньше заметить)», не просто «(es eher bemerken)».

9. **Артикли (ein, der, die, das) ≠ «некий/этот».** Объединять с соседним сегментом: `{ "ru": "верной", "de": "ein treues" }`; либо пустой `ru`: `{ "ru": "", "de": "ein" }`. Артикли при существительных (der Wetterfahne) группируются с существительным/прилагательным.

10. **Предлоги — отдельные сегменты.** `{ "ru": "с", "de": "mit" }` + `{ "ru": "сердцами", "de": "den Herzen" }`, не «с сердцами / mit den Herzen».

11. **Связка sein в именном сказуемом → тире «—» в русском**: `{ "ru": "—", "de": "ist" }`. Не «есть»/«является». Распространяется на ist, seid, sind. **Прошедшее (war) и Konjunktiv II (wär')** требуют русского глагола — тире не передаёт ни времени, ни наклонения.

12. **Идиомы**: буквальный перевод предпочтительнее, если по-русски грамматичен (`im Streit` → «в споре», не «наперебой»). По смыслу — только когда буквальный невозможен/непонятен (`Was fragen sie nach` → «Что за дело им до»; `schwer im Sinn liegen` → «тяготить» — «лежит на уме» в русском нет). **Ложные соответствия**: `sich zur Ruhe legen` = просто лечь, а русское «лечь на покой» = умереть/уйти в отставку → переформулировать (`zur Ruh'` → «отдохнуть»; локальное отступление от правила 28). **Сдвиг подлежащего**: `ich hatte X` → «у меня была X» (DE: человек+haben+Akk → RU: `у`+Gen+быть+Nom; «я имел» — грубо/архаично). То же `wie es geht` → «как дела идут» (`es` → «дела»), `was hat es?` → «что с ним?». **Во всех случаях — обязательна языковая аннотация.**

13. **Косвенная речь.** В русском нет согласования времён. После «подумал, что...» одновременное действие → настоящее: «подумал, что он освистывает» (не «освистывал»).

14. **Возвратное притяжательное.** «Meinem» → «своём» (не «моём»), когда подлежащее = «я». «В своём бреду» естественнее.

15. **Род адаптируется.** Sie (Wetterfahne ж.р.) → он (флюгер м.р.). При значимом расхождении — языковая аннотация.

16. **Verbklammer.** Aux + Partizip = рамка-многоточие: `Fremd **bin** ich **eingezogen**` → `{ "ru": "я", "de": "ich" }, { "ru": "пришёл", "de": "bin... eingezogen" }`. Русское подлежащее перед содержательным глаголом, как в немецком (не «Чужим пришёл я»). Содержательный глагол стоит в конце немецкой фразы — в русском тоже в конец: «На каждый камень натыкался», не «Натыкался на каждый камень».

17. **Предлог, поглощённый падежом.** Если немецкий предлог не имеет русского эквивалента (его роль играет падеж), сливаем с соседним: `gehüllt in Schnee` → «окутан снегом» — `{ "ru": "снегом", "de": "in Schnee" }` (творительный поглощает `in`).

18. **Никаких пустых сегментов (`ru = ""`).** Немецкое слово без русского — сливается с соседним в `de`: `Es zieht... mit` → «Следует»; `man mich` → «меня»; `bin... eingezogen` → «пришёл». **Пустой `de`** (при непустом `ru`) — для русских слов, добавленных для грамматики («что» для dass; «Если» для V1-условий; «как» для AcI — правило 40). Перед добавлением проверить, не решается ли пунктуацией.

19. **Модальные глаголы в контексте.** Sollen во 2 л. (du sollst) — воля говорящего (заверение, обещание), не обязанность: `Sollst meinen Tritt nicht hören` → «Моих шагов не услышишь», не «Должна не слышать».

20. **Эллипсис глагола.** Немецкий допускает опущение глагола, русский — нет: `Sacht, sacht die Türe zu!` (только частица «zu»/«закрыто») → «Прикрою тихо-тихо дверь». Обязательна языковая аннотация.

21. **Не угадывать — проверять.** `Fein` ≠ «прекрасная»/«милая» (Adelung 1811: `fein weniger als schön = ganz artig`; Volkslied `Fein Liebchen` → «пригожая»). `Glühend` ≠ обжигающе (от `glühen` = пылать → «пылающе»). `Nun ja` ≠ «ах да» (Grimm: уступительное «ну да»; `Ach ja` = припоминание «ах да»). Если у немецкого корня есть прямой русский (stoßen → ударять, sich stoßen → ударяться), начинать с него; приблизительный синоним (натыкаться) — только если корневой не работает. **Для русского эквивалента — проверять не только толковые немецкие словари** (DWDS / Grimm / Adelung), **но и немецко-русские** (БНРС / Лейн / Langenscheidt / Multitran) — частотный/нормативный русский эквивалент. **«Первое значение слова X — Y»** — сверять с DWDS (нумерованным значением 1), не «по памяти». Первое значение DWDS не всегда подходит контекстно — тогда обосновывать выбор (Adelung 1811 значение 1 шире для XIX в.; переводческая традиция; контекст). Пример: `Tier` DWDS №1 = «животное», но в обращении к спутнику-вороне выбирается «существо» (Adelung 1811 №1 «lebendes, beseeltes Wesen» + 2 из 3 авторитетных английских переводов дают `creature`).

22. **Schubert vs Müller — двойная сверка** (LiederNet + schubertsong.uk + Hampsong Foundation; NSA если доступна; направление правки легко перепутать). Сноска в одном источнике, в другом нет → доверять сноске. **Источники Müller'а и алгоритм атрибуции правок — правило 57.** **При первой итерации перевода песни — обязательно: список всех различий Müller vs Schubert** (включая мелкие: орфография, элизии, артикли, вспомог. глаголы, словопорядок), с цитатами обоих текстов и пояснением (Plusquamperfekt vs Perfekt, изменение аспекта и т. п.). Различия — **до и отдельно** от записи перевода. **Фиксация `variant_de` / `variant_ru`** (см. «Формат сегментов») — только после явного одобрения каждого случая, не «заодно» с переводом.

23. **Элизии (апостроф) в немецком — последовательно** в `lines_de`, `title_de`, поле `de` сегментов. Примеры: zieh', dacht', hätt', hab', Wär', Ruh', pfiff', Gefror'ne, An's. Перед многоточием рамки апостроф сохраняется (`pfiff'... aus`).

24. **Перевод через отрицание при расхождении коннотаций.** `lau` (тепловатый, акцент на «недостаточно горячий») → «негорячи», не «теплы» (рус. «тёплый» позитивно). `So lau` — «очень негорячий», не «очень тёплый».

25. **`So` в `so...dass` — всегда степень.** Переводится «так»/«настолько», никогда «едва/еле». Направление (много/мало) задаётся следствием в dass-придаточном.

26. **Konjunktiv II ≠ настоящее время.** Hätte/wäre/wollte — прошедшее + «бы»: wolltet → «хотели бы». После «словно/как будто» — прошедшее (с «бы» или без).

27. **Музыкальные вариативные повторы — через ` / `** в `lines_de`, поле `de` и `ru` сегментов: `"Und / Ihr dringt doch aus der Quelle"`, `{ "ru": "А / Вы", "de": "Und / Ihr" }`. Первый — основной, второй — повтор.

28. **Разводить близкие немецкие слова, сводить одинаковые.** В одной песне Boden/Erde, Flur/Wiese/Feld — разные русские; одно немецкое слово через весь цикл — один русский (`rauschen` → «шуметь», `Geselle` → «дружище»). Контекстуальное отступление допустимо, но обязательно сигнализировать. **Кластеры синонимов** (Schmerz/Leid/Leiden/Wehen; Fluss/Strom/Bach/Bächlein) продумывать заранее, для всего цикла. **Каузативные пары одного корня** (`dringen/drängen` — Grimm DWB: «drängen — factitiv zu dringen»; + приставочное `durchdringen`; `wild`/`reißend` рядом семантически: необузданность + стремительность) разводить русскими корнями и приставками: `dringen aus → вырываться` (вы-+рв-), `durchdringen → прорвать` (про-+рв-), `drängen → рваться` (Ø+рв-), `wild → буйная` (буй-), `reißend → бурно` (бур-). Корневая родственность DE сохранена при морфологической различимости RU. Применимо к sinken/senken, trinken/tränken и др.

29. **Регистр двунаправленно.** Возвышенное DE (durchstreichen, die Flur) → возвышенное RU; нейтральное DE → нейтральное RU. Не «украшать» поэтизмами обычное у Müller'а. Проверять DWDS (gehoben, dichterisch, veraltet) + Adelung 1811 + Grimm: DWDS `veraltet` отражает современный язык, в 1820-х могло быть стандартным (`ward`). Русские ловушки без основания: «око» (чаще «глаз»), «знать» (чаще «наверное»), «грезить» (чаще «видеть во сне»), «вещать» (чаще «говорить»).

30. **Предлог или направительное наречие глагольной конструкции — в рамку с глаголом.** `suchen nach`, `fragen nach` → `{ "ru": "ищу", "de": "such'... nach" }`. То же для направительных наречий (`her`, `hin`, `herauf`, `hinaus`), грамматически примыкающих к глаголу движения/звучания: `Von der Straße her ein Posthorn klingt` → `her... klingt` идёт с глаголом (порядок в рамке = текстовый порядок: `her` раньше `klingt`), `Von` — отдельный предлог. **Не группировать `her` с `Von`** (выглядит как составной предлог, что грамматически неверно — `her` примыкает к `klingen`, а не к `von`). Не группировать с управляемым существительным (`Nach... Spur` похоже на отделяемый `nachspüren`).

31. **Грамматические утверждения проверять** в DWDS, E-VALBU (IDS Mannheim), Duden. Первая ассоциация может быть ложной (пример: «suchen nach подразумевает неуверенность» — не подтверждается; реальное различие — nach подчёркивает интенсивность).

32. **Глагольные приставки несут смысл.** Zer- = разрушение (zerbrechen, zerstören). Er- = завершение процесса (ersterben — угаснуть, erstarren — оцепенеть). Ver- = изменение/утрата. Не переводить только корень: zerschmelzen ≠ schmelzen; erstorben ≠ tot (угасло ≠ мертво).

33. **Sein + Partizip II от глаголов изменения = Перфект, не связка.** «Die Blumen sind erstorben» = цветы угасли (Perfekt), не «цветы — мертвы». Правило 11 (sind → тире) не применяется к Перфекту. Признак: причастие от глагола изменения (ersterben, einziehen, aufstehen).

34. **Futurisches Präsens.** Wenn + Präsens → русское **будущее**, не настоящее: `Wenn die Gräser sprossen wollen` = когда/если травы захотят прорасти. Отличать от реального настоящего (`Kalt starrt ihr Bild darin` — застыл сейчас).

35. **Спектр значений: абстрактное и предметное.** Контекст определяет. `Angedenken` = и воспоминание, и памятка/сувенир — `mitnehmen` (взять с собой) указывает на предметное. `Irre Hunde` = безумные, не бродячие (у бродячих нет хозяина — а псы у дома хозяина). `Fluss` ≠ автоматически «река»: общее слово для текучей воды; контекст (прозрачная, буйная, замерзает, поэт сам называет Bach) → «речка». **Предлоги тоже многозначны**: `Durch` = «через» (поперёк) или «по» (вдоль) — `Durch des Bergstroms trockne Rinnen / Wind' ich ruhig mich hinab` = вьётся вниз вдоль русел.

36. **Cross-distribution: смыслы распределены по-разному.** Если DE-конструкция и RU-эквивалент делят смысл между словами не параллельно — не писать ложных пар. Пример: `Ob's... wohl` ↔ «Уж не... ли» (`ob` = «не...ли», `wohl` = «уж»). Нельзя `{ "ru": "ли", "de": "wohl" }`. Решение: всю DE-конструкцию в один сегмент с рамкой-многоточием, лишнюю RU-частицу — пустой `de`:
```json
{ "ru": "Уж не", "de": "Ob's... wohl" },
{ "ru": "под",   "de": "unter" },
{ "ru": "его",   "de": "seiner" },
{ "ru": "ли",    "de": "" },
{ "ru": "коркой","de": "Rinde" }
```
Сопровождать языковой аннотацией.

37. **Связность ландшафтной сцены.** Проверять согласованность отдельных слов с общей картиной. «Скальные лощины» (пологие низины) ≠ «горная река» + «спускается вниз» — один из переводов неточен.

38. **Классические/библейские аллюзии — сохранять узнаваемость.** `Jeder Strom wird's Meer gewinnen` = Экклезиаст («все реки текут в море»). «Каждая река обретёт море» — узнаваемо; «каждый поток обретёт море» — теряет.

39. **`erst` в `nun erst`, `jetzt erst`, `in... erst` — временной контраст, не первичность.** Переводить «лишь»/«только» (= не раньше, только сейчас), **не «впервые»**. `Nun merk' ich erst` → «Только теперь замечаю»; `in der Still' erst` → «лишь в тишине». `erst` относится к обстоятельству времени/места, не к объекту/действию.

40. **AcI (Acc + Inf) в русском невозможен.** `Fühlst deinen Wurm... sich regen` — калька «чувствуешь своего червя... встрепенуться» неграмотна. Обязательна **«как»-конструкция**: «Чувствуешь, как твой червь... встрепенётся». «Как» — пустой `de` (грамматическая добавка). Двоеточие — альтернатива, но работает не везде (длинный AcI с инструментальным расширением — не передать).

41. **Возвратные русские глаголы + `Mit` — сохранять «с».** `Mit + Dat + sich-verb` (`sich regen mit heißem Stich`): `Mit` = comitative/manner, не чистый инструменталь. Русские возвратные (встрепенуться, обернуться) **не принимают чистый творительный**: → «С горячим жалом встрепенётся», не «Горячим жалом встрепенётся». В отличие от правила 17 (там Mit инструменталь, поглощается падежом).

42. **Не приписывать оригиналу переклички из перевода.** Если наш русский создал параллель (`unwirtbarem` → «бесприютном» и `Obdach` → «приют»), проверить оригинал: разные ли корни в немецком (`Wirt` vs `Dach`)? Если да — параллели у Müller'а нет, мы её создали. Допустимо, но не подавать как авторский ход.

43. **Перед переводом значимого слова — grep по всем песням.** `grep '"de": "<слово>"' src/data/songs/*.json` — посмотреть прежние переводы. Либо следовать (правило 28), либо сознательно отступить с сигнализацией. Пример: `träumte` в драфте «грезил», тогда как в песне 5 `träumt'` уже было «снилось» с аннотацией «грезил» как отвергнутая альтернатива — grep вскрыл бы.

44. **Избегать русских конструкций с поверхностной омографией.** Глаз сбивается на: «так, как» (сравнит.) vs «так как» (каузальн.); «что бы» vs «чтобы»; «то же» vs «тоже»; «также» vs «так же». Запятая формально разводит, но при беглом чтении наводится первая интерпретация. Для коррелятивного `so wie` — однозначные эквиваленты: «Такими, какими…», «Как…», «Подобно тому, как…». Избегать «Так, как» в начале строки. Принцип: если фраза двусмысленна на беглом восприятии — переформулировать.

45. **Компаунды через дефис для непереводимых корневых семей.** Немецкое слово — часть семьи (`Traum / träumen / Träumer`), в русском — разные корни (`сон / сниться / мечтатель`): для ключевых узлов компаунды (`Traum → сон-мечта`, `Träumer → сновидец-мечтатель`). Применять, когда: (а) многозначность важна для поэтики; (б) ни один русский синоним не покрывает всё (БНРС: разные эквиваленты по корням); (в) контекст требует всей палитры. Дефисный компаунд — обычная грамматика (ср. «поэт-переводчик»), оба компонента склоняются.

46. **«Дословно»/«буквально» vs «прямой перевод» — точная семантическая граница.** «Дословно X — Y» требует **корневого + грамматически параллельного** соответствия (Apfel → яблоко, Tag → день; ich träumte → грезил, та же морфология агенса). Когда корни/морфология не совпадают, но Y — однословный непосредственный эквивалент → **«прямой перевод»** (Träumer → прямой перевод «сновидец»: разные корни, деривация vs композит, но «сновидец» — наиболее непосредственный однословный RU). При частичном покрытии смысла — оговорка («с потерей некоторых смыслов», «в одном из смысловых пластов», «этимологически», «по корню», «в одном из словарных значений»).

47. **В lang-аннотациях цитировать фактическую форму строки, не лемму.** У Müller'а `denke dem Traume nach` — не писать «У Müller'а `dem Traume nachdenken`» (инфинитива в тексте нет). Лемма указывается отдельно: «Глагол `nachdenken` — …», но не атрибутируется Müller'у.

48. **Структурные утверждения о цикле проверять чтением песен.** «X — последнее появление Y», «мотив только в Z», «здесь впервые…» — литературоведы упрощают; конкретный текст может опровергнуть. Проверка: grep + чтение релевантных песен. Пример провала: «Maid в Песне 11 — последнее появление возлюбленной» — неверно (в 13 `Liebchen`, в 19 `liebe Seele`). **При утверждениях о появлении слова в цикле — различать формы**: `lassen` и `verlassen` (с приставкой) — два разных факта. «До этой песни оба появлялись только в Песне 1» неверно, если в Песне 1 был только `lassen` (как `Lass`), а `verlassen` появляется впервые в текущей. **Конкретные перекрёстные ссылки** (другие песни, аннотации, сегменты) проверять чтением/grep'ом перед записью: «в Песне X stz Y seg Z стоит lang+meaning» — программно проверить (Python: пройти по всем annotations и собрать пересечения по segment_range), а не вспоминать. **Утверждения о составе/порядке исторических изданий — по нескольким независимым источникам** (Wikipedia EN/DE, Youens, Reed). Один агент или источник может ошибиться.

49. **Творительный образа действия — слово действия/манеры, не часть тела.** Дополнение к 17. Немецкое `mit + Adj + N` для образа действия (manner of walking/speaking/looking) → русские модели «X шагом/поступью/походкой» (ходьба), «X голосом/взглядом/дыханием». НЕ работают: «X ногой/стопой» для ходьбы, «X рукой» в смысле жеста. Пример: `mit trägem Fuß` → «вялым шагом» (не «вялой ногой/стопой»). Это русское ограничение сочетаемости: немецкое Fuß в идиоме walking-manner → русское «шаг».

50. **Dass-Ausrufesatz: `Ach, dass... so [adj]!` — Klage** (сетование, не удивление). К 1820-м архаично-формально; Müller осознанно. Подтверждено: Adelung 1811 (закрытый список 8 функций — все негативно-эмоциональные), Grimm DWB §15 («meist einen guten oder bösen wunsch enthält»), Heyse 1814 (Bewunderung через `wie + Adj`, не `dass`), DTA 1780–1830 (0 positive marvel). Простое удивление — без dass: `Ach, [Subj] so [Adj]!`. Перевод: **«Ах, что [Subj] так [Adj]!»** + усилительная **«же»** для Klage. Структурно: dass→что, so→так (раздельные сегменты); «же» — пустой `de` (правило 18). «Как же» теряет однозначную Klage (в русском амбивалентно — восхищение/сетование). «Как жаль, что...» — снижает регистр. Пример: `Ach, dass die Luft so ruhig!` → «Ах, что же воздух так спокоен!» (Песня 12).

51. **Шубертовские прямые повторы внутри строки** (дополнение к 27). Schubert часто повторяет слова/фрагменты в вокальной партии без изменения текста. `Lines_de` отражает sung-text, не печатный Müller'а. Проверять партитуру (IMSLP, NSA) до финального lines_de. Пример: Müller `War ich so elend nicht` → Schubert `War ich so elend, so elend nicht`. **Повтор как ресурс перевода**: одно немецкое слово, повторённое дважды, — два слота для разных русских пластов: `so elend, so elend` → «так жалок, так несчастен» (внешний пласт jämmerlich + внутренний unhappy). Применимо когда: (а) у DE многозначность важна, (б) один RU не покрывает всё, (в) повтор Schubert'а даёт два слота.

52. **Регистровый переворот за 200 лет** (дополнение к 29). Регистр некоторых немецких слов изменился между 1820-ми и современностью, иногда **противоположно**. Пример: `hell` — Adelung 1811 `edler` (более возвышенное); `licht` — `«im gemeinen Leben»` (разговорное). Современный DWDS наоборот: `hell` нейтральное, `licht` `gehoben`. Müller'у был доступен 1820-е расклад. **Доверять Adelung 1811 для регистра 1820-х, не современным DWDS-пометам**.

53. **Adelung'овская «мягкая семантика» — точные градации, не сильные эквиваленты** (дополнение к 21, 29). Формулировки «erste Grad», «schwächere Bedeutung», «mit Widerwillen» **значимы**. Примеры:
- `trübe` — Adelung «erste Grad der Unlust»; Grimm «schwächere als betrübt». НЕ «мрачный» (= düster/finster, сильнее), а **«сумрачный»**.
- `träge` — «mit Widerwillen sich bewegend»; «erschlafft, kraftlos» (Grimm). НЕ «медленный», но **«вялый»** (физическое + психологическое).
- `elend` — jämmerlich + bedrückt + (этимол.) `ali-lanti` = «в чужой земле/изгнании». «Несчастен» (только эмоц.) теряет пласты; «жалок/убог» теряет внутреннее. Распределение через шубертовский повтор (правило 51).

**Подход:** для значимого слова до выбора русского — Adelung + Grimm DWB точно с градациями. Не останавливаться на DWDS modern.

54. **Русская типографика в аннотациях, заголовках, `ru`-полях.** Строго:
- **«...»** (ёлочки) для кавычек, **не** `"..."` (ASCII)
- **—** (длинное тире, em-dash) для разделителя/связки/паузы, **не** `-` (дефис; дефис только внутри сложных слов: «бизнес-ланч»)
- **ё** где требуется орфографией: «ещё», «учётом», «всё» (ср.р.), «чёрные», «нём» (косв. падеж от «он»), «осознаётся», «передаёт», «найдёт», «идёт» — **не** «еще», «учетом», «нем», «осознается», «передает», «найдет», «идет». Особо легко пропускается в косвенных падежах личных местоимений (нём/нём/нём/него) и в личных формах глаголов на -ёт. Проверка после записи аннотации: `grep -nE '(\bнем\b|\bеще\b|осознается|передает|идет\b|найдет|не отдает)' file.json`
- **одинарный пробел**, не двойной
- Многоточие — три точки `...` (устоявшаяся конвенция корпуса, 380+ случаев); специальный символ `…` не обязателен

55. **Следственное `dass` без явного `so`/`derart` в главной части** (литературная архаика XIX в.) → русский следственный союз «**так что**» (не «что»). Без якоря в главной части русское «что» читается как комплементное (= «[тот факт,] что»). Контраст: `so lau, dass…` (явный `so`) → «столь… что…» (Песня 3, есть якорь «столь»); `Hab' wieder schwarze Haare, dass mir's… graut` (без `so`) → «У меня снова чёрные волосы, **так что** мне… страшно» (Песня 14, маркер консекутивности на стороне «так что»).

56. **Сигнализация о подменах предметов / концептов** (дополнение к 6). Если RU-эквивалент означает иной материальный предмет, чем DE-оригинал (`Bahre` = открытые погребальные носилки vs «гроб» = закрытый ящик-Sarg; разные предметы при одинаковом погребальном контексте), или иную культурно-специфическую реалию — это **обязательно сигнализировать пользователю при первоначальном переводе песни**, до записи в файл, даже если другого RU нет. Аналогично для идиом с разной материальной основой, родовых/видовых сдвигов. Особенно важно при социально-историческом подтексте (Bahre без Sarg — реальная похоронная практика бедняков 1820-х, для героя-странника актуальная).

57. **Три источника Müller'а, два направления правки** (контекст для правила 22). Müller публиковал серийно: (1) **Urania 1823** (Brockhaus, Лейпциг) — 12 стихов Gute Nacht … Einsamkeit; (2) **Deutsche Blätter für Poesie** 1823 (Schall/Holtei, Бреслау, №№ 41–42, 13–14 марта) — 10 «продолжений» (Krähe, Mut, Wegweiser, Leiermann и др.); (3) **Waldhornist II 1824** (Ackermann, Дессау) — все 24 стиха, с правками относительно (1) и (2). Schubert: Часть I (1–12) по Urania, Часть II (13–24) по Waldhornist; **Deutsche Blätter в Вену 1827 не доходил — ранние версии 13–24 Schubert не видел**. Для правок Müller'а, не дошедших до Schubert'а — «Schubert не знал».

   Расхождение sung-text vs обычный текст — **одна из двух правок**:
   - (а) **Schubert vs Müller-final** — правка композитора при положении на музыку.
   - (б) **Müller-сам-себя 1823→1824** — правка поэта между Deutsche Blätter и Waldhornist; Schubert получил окончательное.

   **Алгоритм** атрибуции — сравнение трёх источников (DB / WH / S):
   - DB = WH = S, отличие у S → правка Schubert'а.
   - DB ≠ WH = S → правка Müller'а 1823→1824. Schubert не редактор.
   - DB = S ≠ WH → невозможно (Schubert не имел DB).

   Если LiederNet / schubertsong.uk помечают вариант как «Müller» — часто **Deutsche Blätter** (ранний Müller), не «Müller-original vs Schubert-правка».

   **Применение к Песне 15 «Die Krähe»**: `ist gezogen` (DB, Perfekt) → `war gezogen` (WH = Schubert, Plusquamperfekt). Правка Müller'а 1823→1824, не Schubert'а; ранний Perfekt Schubert не видел. Аннотация и `variant_de`: «Müller в Deutsche Blätter 1823: `ist`; Müller в Waldhornist 1824 (= Schubert): `war`».

   Алгоритм с тремя источниками применим только к 10 «продолжениям» из Deutsche Blätter. Для песен 1–12 (Urania) и других Waldhornist'а без DB — расхождение = правка Schubert'а напрямую.

58. **Каждая стихотворная строка — с прописной.** При сегментации первого слова строки в `lines_ru` легко получить строчную (если по русской пунктуации не должно быть прописной — например, после запятой при разбивке немецкого предложения по строкам). **Стихотворная конвенция выше синтаксической**: каждая строка `lines_ru` (как и `lines_de`) начинается с прописной буквы. Проверка после записи: первый сегмент каждой строки в каждой строфе. Распространяется на все 24 песни.

59. **Грамматическая модальность в аннотациях.** Точно сохранять модальность немецкого высказывания при описании в meaning-аннотации: V1 + `?` (вопрос) → «спрашивает», не «просит»; императив (`lass...!`) → «просит / велит»; декларатив → «говорит / утверждает»; восклицание (`!`) — отметить эмоциональный регистр. Пример: `Willst mich nicht verlassen?` — это **вопрос** героя к вороне, не просьба; `Lass mich endlich sehn` — это **императив-просьба**. Не растворять разные модальности в общем «герой обращается / говорит».

## Annotation Guidelines

Два типа: `lang` (язык/грамматика — синий) и `meaning` (смысл/интерпретация — коричневый). Нумерация независимая.

**Lang** — грамматика и структура: Verbklammer (`hätt'...sollen`), порядок слов (V2/V-last, отделяемые приставки), падежи (Genitiv, приложения), род (Wetterfahne ж.р. → флюгер м.р.), отрицание (двойное `nimmer nicht` = утверждение в DE, в RU — усиление), опускание союзов (dass после глаголов мышления), идиомы, составные слова (Frauenbild = Frauen + Bild).

**Meaning** — интерпретация и контекст: этимология и связи (Wetterfahne — wetterwendisch); перспектива (смена лица я → он → они, самоотчуждение); мотивы (связи между строфами и песнями цикла: ветер, флюгер, непостоянство); многозначность (сердца = любовь + расчёт); социальный контекст (критика мещанства, неравенство статусов); фольклорные формулы (Volkslied — Feinsliebchen — традиция, не индивидуальный авторский ход); Schubert vs Müller с эффектом.

**Стиль аннотаций:**
- Перед написанием — искать литературоведов на DE/EN.
- Не категорично — несколько интерпретаций признавать.
- Без доказательств не утверждать (герой — подмастерье? из текста не следует).
- Не ссылаться на учёных по именам (не «по Гримму», а собственное наблюдение).
- **Не упоминать словари по именам** в самой аннотации (не «по DWDS», «по Adelung'у», «БНРС»); пометы регистра (gehoben, dichterisch, veraltet, selten) не цитировать буквально. Источники нужны для подготовки и фактчека, не для текста аннотации.
- **Писать о тексте, не об учебнике.** Никакой лингв. терминологии (не «делиберативное», «предложное управление», «переходный»). Не: «Konjunktiv II в немецком выражает нереальное условие». Да: «Locken — манить. Hin в конце добавляет направление: не просто манил, а заманил. Важный оттенок — в конце».
- **Каждый факт проверять** по авторитетным источникам (правило 31). Для исторических — год **публикации**, не наблюдения (Бессель наблюдал Irrlicht 1807, опубликовал 1838 — Müller не мог знать).

### Технические правила аннотаций

- `line_span: N` — охватывает N строк (текущую + последующие). Сноска в конце последней. **Может пересекать границы строф** (lines идут последовательно через всю песню).
- `continuation_ranges: [[start, end], ...]` — диапазоны сегментов для каждой последующей строки (line+1, line+2, ...). Без него подсвечиваются все. **`[-1, -1]` пропускает строку без подсветки** (для аннотаций с разрозненными строками). **Не использовать `null`** — рендер исторически поддерживает оба, но единая конвенция — `[-1, -1]`.
- `segment_range: [start, end]` — индексы включительно.
- Аннотации могут перекрываться (один сегмент — и lang, и meaning).
- В тексте `\n` — перенос строки (CSS `white-space: pre-line`).

### Формат сегментов

- Отделяемые приставки: `"de": "pfiff'... aus"` (с многоточием)
- Рамки предлогов: `"de": "Auf... Haus"`
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{ "ru": "женщины образ.", "de": "Frauenbild" }`
- Пустой `ru` — артикль без эквивалента
- Пустой `de` — русское слово, добавленное для грамматики
- **Пунктуация только в `ru`, не в `de`.** Запятая, точка, восклицательный знак, тире — принадлежат русскому переводу, не немецкому слову. Образец: `{ "ru": "ах,", "de": "ach" }` (не `de: "ach,"`); `{ "ru": "Ну да,", "de": "Nun ja" }`.
- **`variant_de` / `variant_ru` — расхождения Schubert vs Müller** (правила 22, 57). Основное `de`/`ru` — Schubert sung-text; `variant_de` — мюллеровский вариант (если различается); `variant_ru` — русский для мюллеровского варианта (если он отличается от русского Schubert'овского). Образец: `{ "ru": "угасшее,", "de": "erstorben", "variant_de": "erfroren", "variant_ru": "замёрзшее" }` (Песня 4); `{ "ru": "поверил", "de": "glaubt'", "variant_de": "meint'", "variant_ru": "полагал" }` (Песня 14). Для рамок: вариант кладётся на тот же сегмент-рамку: `{ "ru": "рассыпал.", "de": "hat... gestreuet", "variant_de": "hatt'... gestreuet" }` (рендер lines_de может не показать вариант для рамки — известное ограничение, данные всё равно фиксируем). Если русский не различается (только грамматический сдвиг немецкий, например Plusquamperfekt vs Perfekt) — `variant_ru` не нужен.
