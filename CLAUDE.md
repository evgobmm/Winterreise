# CLAUDE.md — Winterreise

## ⚠️ ОБЯЗАТЕЛЬНЫЙ ПРОТОКОЛ ПЕРВОЙ ЗАПИСИ ПЕСНИ

**Перед фиксацией перевода новой песни в JSON** — выполнить и привести вывод:

```bash
python3 scripts/pre-translation-check.py <N>
```

Скрипт автоматически (1) grep каждого значимого DE-слова по всем переведённым песням; (2) для NEW слов — поиск семьи по корню (правило 32); (3) выводит lines_de для ручной сверки пунктуации; (4) печатает чеклист дальнейших шагов.

После запуска скрипта — обязательно:

1. **Привести вывод в чат**, прокомментировать каждое совпадение по словарной семье (применить конвенцию или сознательно отступить с сигнализацией).
2. **Сверка пунктуации** каждой DE-строки по ≥3 источникам (LiederNet + Wikisource Müller WH 1824 + schubertsong.uk). **Hampson Foundation не использовать** (опечатки).
3. **Список спорных мест** с конкретными альтернативами (правило 6).
4. **Müller vs Schubert** различия явно перечислить.
5. **Получить одобрение пользователя** по каждому спорному пункту (правило 7).
6. **Только после этого** — записывать перевод в JSON.

**Без выполнения этого протокола первая запись песни ЗАПРЕЩЕНА.** Все ранее обнаруженные провалы (matt в Песне 18, weist в Песне 19, пунктуация L8 Песни 19) были вызваны игнорированием этого протокола.

## Project Overview

Подстрочный перевод вокального цикла «Зимний путь» (Winterreise) Мюллера / Шуберта. 24 песни, немецкий + русский с пословной разметкой и аннотациями. Vue 3 + Vite, GitHub Pages.

## Architecture

### Страница

- **Левая колонка**: немецкий оригинал (строфами)
- **Правая колонка**: русский перевод; под каждым словом мелким шрифтом — соответствующее немецкое
- **Сноски** `¹²³` у отмеченных фрагментов; сквозная нумерация в пределах песни. Наведение — подсветка фрагмента, клик — пояснение
- **Режим пояснений** внизу страницы — все аннотации песни
- **Меню**: 24 песни (немецкие названия)
- **Заголовок песни**: немецкое + русское название
- Поиск, переключатель темы, печать/PDF

### Админ-режим

URL `?admin=<SECRET_KEY>`. Inline-редактирование, сохранение через GitHub Contents API (токен в localStorage, права `repo`). Параллельно — правка JSON в Claude Code CLI.

### Data Model

JSON по одной песне в `src/data/songs/`:

```json
{
  "number": 2,
  "title_de": "Die Wetterfahne",
  "title_ru": "Флюгер",
  "stanzas": [{
    "lines_de": ["Der Wind spielt mit der Wetterfahne"],
    "lines_ru": [{
      "segments": [
        { "ru": "Ветер", "de": "Der Wind" },
        { "ru": "играет", "de": "spielt" },
        { "ru": "с", "de": "mit" },
        { "ru": "флюгером", "de": "der Wetterfahne" }
      ],
      "annotations": [
        { "type": "lang", "segment_range": [3, 3], "text": "..." }
      ]
    }]
  }]
}
```

**Правила данных:**
- `segments` — пары {ru, de} в порядке русского
- `de` может быть пустой строкой
- `segment_range` — индексы [start, end] включительно (подсветка)
- `lines_ru.length` == `lines_de.length`. Verbklammer/рамка через границу строк — одним сегментом-рамкой в одной из lines_ru (см. правило 16)
- `title_annotations`: массив `{ type, text }` для комментариев к песне в целом; без `segment_range`

### Tech Stack & Structure

Vue 3 (Composition API, `<script setup>`) + Vite. Чистый CSS, plain JS (без TS, UI-фреймворков). Деплой GitHub Actions → GitHub Pages.

- `src/data/songs/*.json` — песни (01–24)
- `src/data/index.json` — метаданные (number, title_de, title_ru, ready)
- `src/components/` — SongList, SongView, InterlinearLine, FootnoteMark, AnnotationsPanel, SearchBar, ThemeToggle; `admin/` (InlineEditor, AnnotationEditor, SaveButton)
- `src/styles/` — main.css, print.css, theme.css
- `src/utils/` — github-api.js, search.js, print.js

## Translation Rules

**Правила ниже — обобщения опыта, не аксиомы.** При противоречии правила и фактической DE/RU грамматики приоритет имеет грамматика. Если правило даёт грамматически некорректный RU — правило неприменимо в этом случае; решать по грамматике.

Подстрочник — пословное соответствие, не литературный перевод.

**Точность — приоритет.** Русский грамотный и реально существующий (проверять сочетаемость: «лежит на душе» — есть, «лежит на уме» — нет), но не обязан звучать красиво. Точное лучше привычного: `блестящих`, не `светлых`; `в споре`, не `наперебой`. **При равной точности** — частотный вариант. **Устоявшийся русский перевод не аргумент** держаться менее точного. Не добавлять отсутствующих оттенков: уменьшительных суффиксов, ложных архаизмов, аспектных значений (`взглянуть` мгновенно ≠ `sehn` нейтрально).

**Перевод DE → RU. Английский не использовать** ни в обсуждениях, ни в скобках, кроме исключительных случаев.

**Порядок работы.** Для КАЖДОГО знаменательного слова (включая частицы и наречия: `immerfort` = непрестанно; `je` = хоть когда-нибудь):

1. **DWDS** (значения, регистр, этимология). Для XIX в. также **Grimm** (dwds.de/wb/dwb/СЛОВО) и **Adelung 1811** (zeno.org/Adelung-1793; ближе всего к Müller'у). Регистр и значения дрейфуют (`starren` 1820-х → «застыть»; сейчас «уставиться»).
2. **Разобрать на приставку + корень** (Er-sterben ≠ sterben, zer-schmelzen ≠ schmelzen).
3. **Близкие слова развести** (Boden/Erde, Gras/Rasen, Blüte/Blumen).
4. **Контекст важнее этимологии, точность важнее традиции.** При двух одновременных значениях (`starrt` = и застыл, и уставился) — отметить оба. **Частицы** (schon, doch, ja, auch, wohl, je, noch, mal, denn) меняют значение по контексту: `Schon` ≠ всегда «уже» (в `Tret' ich auch schon` — усилитель уступки); `denn` в вопросе — «же»; `Doch` в `Doch wieder zu finden` — эмфатическое «всё же», не противительное «однако». **Модальные в значении предположения** (`mag` от `mögen` 3 = «должно быть») — отдельный русский эквивалент (правило 1).
5. **Русская пунктуация.** Запятые при обращениях, вводных, деепричастиях. **Не калькировать DE-пунктуацию автоматически**: `Nun, ...` (DE — запятая) → «Теперь не далеко уже идти» (RU — без запятой). **При замене слова перепроверить согласование**: «своё отражение» → «свой образ» (м.р.).
6. **Если выбор неочевиден — сигнализировать**, не делать молча. Идиомы, архаизмы, многозначные слова, расхождение регистра — выносить на обсуждение.

**Источник DE: шубертовская редакция** (как поют), не оригинал Мюллера. Источник: Deutsche Schubert-Lied-Edition (Naxos). Орфография по реформе 1996: ß после кратких → ss (Muß → Muss); после долгих остаётся (Blumenstrauß).

1. **Каждое DE-слово — в подстрочнике.** Не пропускай служебные (da, schon, doch). Слово с самостоятельным переводом — отдельный сегмент даже в рамке (`hinaus` → «прочь»). Но если русский предлог уже передаёт направление приставки — поглощается: `In die Felsengründe lockte... hin` → «В глуби заманил». **Знаменательные слова (модальные, частицы, наречия) обязаны иметь русский эквивалент**: `mögen` 3 = «должно быть»; не растворяться в инфинитиве.

2. **По умолчанию сохраняй DE порядок слов** — чтобы видеть соответствие. Перестройка — только когда русский **нечитаем**, а не «слегка непривычен». **Поэтическая инверсия V-letzt** (`Von der Straße her ein Posthorn klingt`) сохраняется: «С улицы почтовый рожок доносится». **Поэтическая инверсия Gen впереди** (`meiner Hoffnung Grab`, `des ganzen Winters Eis`) тоже сохраняется в RU: «На моей надежды могиле», «Всей зимы лёд» — поэтически читается. Распрямлять (Gen после) только при реальной нечитаемости. **Постпозитивная аппозиция прилагательных** (`Der Winter kalt und wild`) — поэт. оборот; прилагательные **без склонения** (синтаксически эквивалент предикативному ist). RU сохраняет: «Зима холодная и буйная».

3. **Меняй порядок только когда русский нечитаем.** «На моей прекрасной любимой доме» — нельзя; «Тут я подумал уже в своём безумии» — ок.

4. **Одно русское = одно немецкое (по возможности).** Группировать только когда иначе невозможно. Отделяемые приставки с ясным русским — отдельный сегмент (`dahin` → «вдаль»). Рамка с многоточием (`pfiff'...aus`) — когда русский не разделить. **Многоточие — только при разрыве словами в DE**; рядом стоящие (Folge nach, Komm her, Schaue nach) — без многоточия. **Порядок в рамке = текстовый порядок DE**.

5. **`de` пустая** — для слов, добавленных для русского смысла без DE-эквивалента.

6. **Liebchen = любимая** (не «милая»). Lieb- от Liebe.

7. **Не писать содержательные изменения без приглашения. Не пушить без явного «пушим»/«делаем».** Одобрение перед записью в файл (не только перед commit; файл сразу виден на dev-сервере). Порядок: предложить → «ок» → писать → показать → «пушим» → commit/push. **Аннотации «от себя», `variant_de`/`variant_ru`, ретроактивные правки — каждый случай отдельно.** Одобрение конкретного шага не распространяется на сопутствующие изменения и правила CLAUDE.md «заодно». **Условное одобрение** («если всё чётко, пушим»; «проверь и сделай») — сначала сообщить результат проверки, только потом действовать.

8. **В аннотациях переводи немецкие фрагменты.** «(es eher bemerken — это раньше заметить)», не просто «(es eher bemerken)».

9. **Артикли (ein, der, die, das) ≠ «некий/этот».** Объединять с соседним сегментом: `{ "ru": "верной", "de": "ein treues" }`; либо пустой `ru`: `{ "ru": "", "de": "ein" }`. Артикли при существительных группируются с существительным/прилагательным.

10. **Предлоги — отдельные сегменты.** `{ "ru": "с", "de": "mit" }` + `{ "ru": "сердцами", "de": "den Herzen" }`.

11. **Связка sein → тире «—» отдельным сегментом, всегда.** `{ "ru": "—", "de": "ist" }`. Не «есть»/«является». Распространяется на **все** формы связки: bin, bist, ist, sind, seid. Применяется к **любому** именному сказуемому: с существительным, с предикативным прилагательным (полным или кратким), с местоимением, с **adjektives Partizip** (причастие в адъективной функции, **статив** — описание состояния, не действия; признак: глагол транзитивный — Perfekt с `haben`, или контекстуально длительное состояние с модификаторами типа `manche Stunde`, `wie`). Сегментация: предикатив и связка — **раздельные** сегменты; рамка `Adj. + sein-форма` запрещена (она объединяла бы два разных грамматических элемента под одним RU-словом, что нарушает прозрачность подстрочника). **Порядок в RU**: тире-связка ставится между подлежащим и предикативом по RU-стандарту именного сказуемого («подлежащее — предикатив»), **даже если в DE связка стоит в конце клаузы** (Verbletzt-Stellung Nebensatz'а). Это отступление от DE-порядка слов допустимо по правилу 3 (RU-нечитаемость финального тире). Прецеденты: Песня 10 L1 `wie müd' ich bin` — сегменты `müd'` → «усталый», `bin` → «—», `ich` → «я,»; Песня 19 L5 `wer ... so elend ist` — сегменты `ich` → «я,», `ist` → «—», `so` → «так», `elend` → «жалок-несчастен,»; Песня 5 stz5 L1-L2 `Nun bin ich manche Stunde / Entfernt` — `bin` → «—» в конце L1 (тире между подлежащим/обстоятельством и предикативом из L2 «Отдалён»); Песня 9 L1 `Bin gewohnt das Irregehen` — добавлен сегмент «Я» (правило 18, опущенное в DE подлежащее 1 sg.), `Bin` → «—», `gewohnt` → «привычен» (краткое прил.). **Разграничения**: (а) **Perfekt** (Sein + Partizip II от глаголов изменения, Perfekt с `sein`: ersterben, einziehen, fallen, werden) идёт по правилу 33 (Verbklammer, RU глагол прош. вр.), не по правилу 11. (б) **Funktionsverbgefüge** (устойчивые DE-сочетания типа `zu Ende sein mit`, правило 66) переводятся идиоматически одним RU-глаголом — это разрешённое исключение из правила 11. (в) **Прошедшее (war) и Konjunktiv II (wär')** требуют русского глагола.

12. **Идиомы**: буквальный перевод предпочтительнее, если по-русски грамматичен (`im Streit` → «в споре»). По смыслу — только когда буквальный невозможен (`Was fragen sie nach` → «Что за дело им до»). **Ложные соответствия**: `sich zur Ruhe legen` = просто лечь, а RU «лечь на покой» = умереть → переформулировать. **Сдвиг подлежащего**: `ich hatte X` → «у меня была X»; `wie es geht` → «как дела идут»; `was hat es?` → «что с ним?»; **риторический `Was will ich + Inf`** → «Что/Зачем мне + Inf» (`Was will ich säumen?` → «Что мне мешкать?»). **Во всех случаях — обязательна языковая аннотация.**

13. **Косвенная речь.** В русском нет согласования времён. После «подумал, что...» одновременное действие → настоящее: «подумал, что он освистывает».

14. **Возвратное притяжательное — только в пределах одной RU-клаузы.** Когда DE-посессив (mein/dein/sein/ihr) согласован с субъектом своей клаузы → RU «свой» (для любого лица). **При разбиении на две клаузы** (AcI через «как»/«:», подчинительная, новое предложение) — возвратное «свой» через границу клаузы не работает: антецедент извне передаётся **личным местоимением «его/её/их»**. Пример AcI (правило 40): `Mein Herz sieht ... sein eignes Bild` → «Моё сердце видит: на небе нарисован **его** собственный образ» (две клаузы; «свой» в L10 невозможен — субъект L10 = «образ»).

15. **Род адаптируется.** Sie (Wetterfahne ж.р.) → он (флюгер м.р.). При значимом расхождении — языковая аннотация.

16. **Verbklammer.** Aux + Partizip II = рамка-многоточие: `bin... eingezogen` → «пришёл». Aux + Inf (Modal-Klammer, Futur): `hätt'... sollen`, `wird... gewinnen`. Русское подлежащее перед содержательным глаголом, как в немецком. Содержательный глагол стоит в конце DE — в RU тоже в конец. **НЕ Verbklammer**: стандартный порядок Nebensatz S-Inf-V-fin (`was ich zittern kann` — субъект+инфинитив+финитный модальный — стандартное придаточное, не рамка). Ошибочное оформление S-V-V как рамки ведёт к потере отдельных слов.

17. **Предлог, поглощённый падежом.** `gehüllt in Schnee` → «окутан снегом» — `{ "ru": "снегом", "de": "in Schnee" }` (творительный поглощает `in`).

18. **Пустые сегменты — симметричное правило.** (а) **Пустой `de`** (при непустом `ru`) — для русских слов, добавленных для грамматики («что» для dass; «Если/Когда» для V1-условий; «как» для AcI — правило 40; «Тот» как восстановленный антецедент `der`). Перед добавлением проверить, не решается ли пунктуацией. (б) **Пустой `ru`** (`""`, при непустом `de`) **разрешён** для **служебных DE-элементов**, грамматически нужных в DE, но не имеющих отдельного RU-аналога: (1) формальный/Vorfeld-`es` (включая контракции `'s` = es в позиции формального субъекта; пример: `'s führt` → `führt` → «Ведёт», `'s` → «»); (2) катафорический `es` в позиции объекта, поглощаемый dass-клаузой (`seh's ... an, dass...` → `seh'` → «вижу», `'s` → «», `an` → «по»). **Знаменательные DE-слова** (существительные, глаголы, прилагательные, наречия с самостоятельным значением) пустой `ru` иметь не могут — они должны сливаться с соседним в DE-сегменте (рамка) или иметь явный RU. **Цель**: визуальная честность подстрочника — читатель видит пустую RU-ячейку под служебным DE-элементом и понимает, что это часть DE-грамматики, не несущая отдельного RU. **Артикль** под существительным/прилагательным — отдельный случай: правило 9 (поглощается в соседний сегмент, пустой `ru` не нужен).

19. **Модальные глаголы в контексте.** Sollen во 2 л. (du sollst) — воля говорящего: `Sollst meinen Tritt nicht hören` → «Моих шагов не услышишь».

20. **Эллипсис глагола.** Немецкий допускает опущение, русский — нет: `Sacht, sacht die Türe zu!` → «Прикрою тихо-тихо дверь». Обязательна языковая аннотация.

21. **Не угадывать — проверять.** `Fein` ≠ «прекрасная» (Adelung 1811: `fein weniger als schön`; Volkslied → «пригожая»). `Glühend` ≠ обжигающе (от `glühen` = пылать → «пылающе»). Если у DE-корня есть прямой RU (stoßen → ударять, sich stoßen → ударяться), начинать с него. **Проверять не только DWDS/Grimm/Adelung, но и DE-RU словари** (БНРС / Лейн / Langenscheidt / Multitran). **«Первое значение слова X»** — сверять с DWDS №1, не «по памяти». Первое значение DWDS не всегда контекстно — тогда обосновывать выбор (Adelung 1811 для XIX в.; традиция; контекст). **Обратная сверка RU→DE для регистра**: при сомнении в стилистическом регистре RU-слова — проверить обратным словарём. **Близкие RU-синонимы развести по корням**: «трепетать» (от «трепет» = дрожь; DE `zittern`) ≠ «трепаться» (мотаться на ветру; DE `flattern`). Не подменять одно другим.

22. **Schubert vs Müller — двойная сверка** (LiederNet + schubertsong.uk + Hampsong + NSA). Сноска в одном источнике → доверять ей. **Алгоритм атрибуции — правило 57.** **При первой итерации песни — список всех различий Müller vs Schubert** (включая орфографию, элизии, артикли, словопорядок) с цитатами и пояснением, **до и отдельно** от записи перевода. Проверять модификаторы (`hier und da`, `noch`, `so`) в полном контексте. **`variant_de`/`variant_ru`** — только после явного одобрения каждого случая.

23. **Элизии (апостроф)** — последовательно в `lines_de`, `title_de`, `de` сегментов. Примеры: zieh', dacht', hätt', hab', Wär', Ruh', pfiff', Gefror'ne, An's. Перед многоточием рамки апостроф сохраняется (`pfiff'... aus`).

24. **Перевод через отрицание при расхождении коннотаций.** `lau` (акцент на «недостаточно горячий») → «негорячи», не «теплы» (рус. «тёплый» позитивно).

25. **`So` в `so...dass` — всегда степень.** «так»/«настолько», никогда «едва/еле». Направление задаётся следствием в dass-придаточном.

26. **Konjunktiv II ≠ настоящее.** Hätte/wäre/wollte — прошедшее + «бы»: wolltet → «хотели бы». После «словно/как будто» — прошедшее.

27. **Шубертовские музыкальные повторы — через ` / `** в `lines_de`, `de`/`ru`: `"Und / Ihr dringt..."`, `{ "ru": "А / Вы", "de": "Und / Ihr" }`. Первый — основной, второй — повтор.

28. **Разводить близкие DE-слова, сводить одинаковые.** В одной песне Boden/Erde, Flur/Wiese/Feld — разные RU; одно DE-слово через цикл — один RU (`rauschen` → «шуметь», `Geselle` → «дружище», `wild` → «буйный» во всех контекстах). Контекстуальное отступление допустимо, обязательно сигнализировать.
- **Кластеры синонимов** (Schmerz/Leid/Leiden/Wehen; Fluss/Strom/Bach/Bächlein) — продумывать для всего цикла.
- **Каузативные пары одного корня** разводить русскими корнями и приставками: `dringen aus → вырываться`, `durchdringen → прорвать`, `drängen → рваться`; `wild → буйный`, `reißend → бурно`.
- **Модификаторы блокируют конвенцию**: `So manches X` ≠ `manches X` (`so manch-` — идиома усиления «столько»); `noch ein X` ≠ `ein X` (временной аспект). Проверять so, noch, je, gar.
- **Substantiv vs attributiv**: `manches Blatt` (attr.+Sg N) → «не один лист»; `manches` (subst. без N) → «всякое/разное».
- **Близкие наречия направления**: `hinaus` (через границу, изнутри-наружу) → «вон»; `fort` (общее удаление) → «прочь».

29. **Регистр двунаправленно.** Возвышенное DE → возвышенное RU; нейтральное → нейтральное. Не «украшать» поэтизмами обычное у Müller'а. Проверять DWDS (gehoben, dichterisch, veraltet) + Adelung 1811 + Grimm: DWDS `veraltet` отражает современный язык, в 1820-х могло быть стандартным (`ward`). Русские ловушки без основания: «око», «знать», «грезить», «вещать».

30. **Предлог или направительное наречие глагольной конструкции — в рамку с глаголом.** `suchen nach`, `fragen nach` → `{ "ru": "ищу", "de": "such'... nach" }`. То же для `her`, `hin`, `herauf`, `hinaus`, грамматически примыкающих к глаголу: `Von der Straße her ein Posthorn klingt` → `her... klingt` идёт с глаголом, `Von` — отдельный предлог.

31. **Грамматические утверждения проверять** в DWDS, E-VALBU (IDS Mannheim), Duden. Первая ассоциация может быть ложной.

32. **Глагольные приставки несут смысл.** Zer- = разрушение (zerbrechen). Er- = завершение процесса (ersterben — угаснуть, erstarren — оцепенеть). Ver- = изменение/утрата. Не переводить только корень: zerschmelzen ≠ schmelzen.

33. **Sein + Partizip II от глаголов изменения = Перфект, не связка.** «Die Blumen sind erstorben» = цветы угасли (Perfekt). Правило 11 (sind → тире) не применяется. Признак: причастие от глагола изменения (ersterben, einziehen, aufstehen). **Модификатор `wie` блокирует Perfekt-чтение**: `Mein Herz ist wie erstorben` ≠ `Mein Herz ist erstorben`. С `wie` Partizip II — **адъективное причастие** (статив, сравнение): `ist wie erstorben` → «есть словно угасшее» (связка через тире). Аналогично `wie tot`, `wie betrunken`. Правило 33 применяется к Aux+Partizip II **без модификаторов** `wie/als ob`.

34. **Futurisches Präsens.** Wenn + Präsens → русское **будущее**: `Wenn die Gräser sprossen wollen` = когда захотят прорасти. **V1 + явный Nom-субъект без `wenn/als`** — почти всегда **бессоюзное условие** или **Folgesatz после императива**. В RU обязательно «Если/Когда/И» с пустым `de` (правило 18). Признак: финитный V первым + Nom явный (не Subjektellipse `ich/du`). Пример: `Spielt der Wind mit meinem Blatte` → «**Когда** играет ветер...». Исключение: V1 может быть **ответом на вопрос по рифмовке** (Песня 6 stz3 `Nimmt dich bald das Bächlein auf` — ответ на L2 `wohin geht dein Lauf?`). Проверять **каждое** V1 + явный субъект на условный смысл.

35. **Спектр значений: абстрактное и предметное.** Контекст определяет. `Angedenken` = и воспоминание, и сувенир — `mitnehmen` указывает на предметное. `Fluss` ≠ автоматически «река»: общее слово; контекст (прозрачная, замерзает, поэт сам называет Bach) → «речка». **Предлоги тоже многозначны**: `Durch` = «через» (поперёк) или «по» (вдоль) — `Durch des Bergstroms trockne Rinnen` = вдоль русел.

36. **Cross-distribution: смыслы распределены по-разному.** Если DE-конструкция и RU-эквивалент делят смысл не параллельно — не писать ложных пар. Пример: `Ob's... wohl` ↔ «Уж не... ли» (`ob` = «не...ли», `wohl` = «уж»). Решение: вся DE-конструкция в рамку с многоточием, лишняя RU-частица — пустой `de`. Сопровождать языковой аннотацией.

37. **Связность ландшафтной сцены.** Проверять согласованность отдельных слов с общей картиной. «Скальные лощины» (пологие) ≠ «горная река» + «вниз» — один неточен.

38. **Классические/библейские аллюзии — сохранять узнаваемость.** `Jeder Strom wird's Meer gewinnen` = Экклезиаст. «Каждая река обретёт море» узнаваемо; «каждый поток» теряет.

39. **`erst` в `nun erst`, `jetzt erst`, `in... erst` — временной контраст, не первичность.** «лишь»/«только» (= не раньше, только сейчас), **не «впервые»**. `erst` относится к обстоятельству, не к объекту.

40. **AcI (Acc + Inf/Partizip) в RU невозможен.** Передавать через: (а) **«как»-клаузу** (`Fühlst deinen Wurm... sich regen` → «Чувствуешь, **как** твой червь... встрепенётся»); или (б) **двоеточие** (`Mein Herz sieht ... gemalt sein Bild` → «Моё сердце видит: на небе нарисован **его** ... образ») — стандарт RU после глаголов восприятия (видеть, слышать, чувствовать) для прямой картины. «Как»/добавочное слово — пустой `de` (правило 18). **Обе формы создают новую клаузу** → возвратное «свой» через границу не работает (правило 14).

41. **Возвратные русские глаголы + `Mit` — сохранять «с».** `Mit + Dat + sich-verb`: `Mit` = comitative/manner, не чистый инструменталь. RU возвратные (встрепенуться, обернуться) **не принимают чистый творительный** — «С горячим жалом встрепенётся».

42. **Не приписывать оригиналу переклички из перевода.** Если RU создал параллель (`unwirtbarem` → «бесприютном» и `Obdach` → «приют»), проверить DE: разные корни? Если да — параллели у Müller'а нет.

43. **Перед переводом значимого слова — grep по всем песням.** `grep '"de": "<слово>"' src/data/songs/*.json`. Либо следовать конвенции (правило 28), либо сознательно отступить с сигнализацией.

44. **Избегать русских конструкций с поверхностной омографией.** «так, как» vs «так как»; «что бы» vs «чтобы»; «то же» vs «тоже»; «также» vs «так же». Для `so wie` — «Такими, какими…», «Как…», «Подобно тому, как…». Если фраза двусмысленна на беглом восприятии — переформулировать.

45. **Компаунды через дефис для непереводимых корневых семей.** DE слово — часть семьи (`Traum / träumen / Träumer`), RU — разные корни: для ключевых узлов компаунды (`Traum → сон-мечта`, `Träumer → сновидец-мечтатель`). Применять когда: (а) многозначность важна; (б) ни один RU синоним не покрывает всё; (в) контекст требует всей палитры. Оба компонента склоняются. **Применимо к любой части речи** — существительные (`Traum → сон-мечта`), глаголы (`erlaben → освежать-услаждать` = sich erquicken + geistig genießen).

46. **«Дословно»/«буквально» vs «прямой перевод».** «Дословно X — Y» требует **корневого + грамматически параллельного** соответствия (Apfel → яблоко). Когда корни/морфология не совпадают, но Y — однословный непосредственный эквивалент → **«прямой перевод»** (Träumer → прямой перевод «сновидец»). При частичном покрытии — оговорка («с потерей смыслов», «этимологически», «в одном из значений»).

47. **В lang-аннотациях цитировать фактическую форму строки, не лемму.** У Müller'а `denke dem Traume nach` — не писать «`dem Traume nachdenken`» (инфинитива в тексте нет). Лемма указывается отдельно: «Глагол `nachdenken` — …», но не атрибутируется Müller'у.

48. **Структурные утверждения о цикле проверять чтением песен.** «X — последнее появление Y», «впервые…» — конкретный текст может опровергнуть. Проверка: grep + чтение релевантных песен. **Различать формы**: `lassen` и `verlassen` — два разных факта. **Конкретные перекрёстные ссылки** программно проверять (Python: пройти по annotations и собрать пересечения), а не вспоминать. **Утверждения об исторических изданиях — по нескольким независимым источникам** (Wikipedia EN/DE, Youens, Reed).

49. **Творительный образа действия — слово действия/манеры, не часть тела** (дополнение к 17). DE `mit + Adj + N` для образа действия → русские «X шагом/поступью/походкой» (ходьба), «X голосом/взглядом» (речь/взгляд). НЕ «X ногой/стопой» для ходьбы. Пример: `mit trägem Fuß` → «вялым шагом».

50. **Dass-Ausrufesatz: `Ach, dass... so [adj]!` — Klage** (сетование, не удивление). К 1820-м архаично-формально. Простое удивление — без dass: `Ach, [Subj] so [Adj]!`. Перевод: **«Ах, что [Subj] так [Adj]!»** + усилительная **«же»** для Klage. Структурно: dass→что, so→так (раздельные сегменты); «же» — пустой `de`. «Как же» теряет однозначную Klage. Пример: `Ach, dass die Luft so ruhig!` → «Ах, что же воздух так спокоен!» (Песня 12).

51. **Шубертовские прямые повторы внутри строки** (дополнение к 27). Schubert часто повторяет слова в вокальной партии. `Lines_de` отражает sung-text. Проверять партитуру (IMSLP, NSA). **Повтор как ресурс перевода**: одно DE слово, повторённое дважды, — два слота для разных RU пластов: `so elend, so elend` → «так жалок, так несчастен» (jämmerlich + unhappy).

52. **Регистровый переворот за 200 лет** (дополнение к 29). Регистр некоторых слов изменился между 1820-ми и современностью. Пример: `hell` — Adelung `edler`; `licht` — `«im gemeinen Leben»`. Современный DWDS наоборот. **Доверять Adelung 1811 для регистра 1820-х**.

53. **Adelung'овская «мягкая семантика» — точные градации** (дополнение к 21, 29). Формулировки «erste Grad», «schwächere Bedeutung», «mit Widerwillen» значимы:
- `trübe` — «erste Grad der Unlust». НЕ «мрачный» (= düster), а **«сумрачный»**.
- `träge` — «mit Widerwillen sich bewegend». НЕ «медленный», но **«вялый»**.
- `elend` — jämmerlich + bedrückt + (этимол.) `ali-lanti` = «в чужой земле». Распределение через шубертовский повтор (правило 51).
- `säumen` — «fehlerhaft langsam, ... aus Trägheit». НЕ «медлить» (нейтр.), но **«мешкать»**.
- `matt` — «erste Grad der Schwäche»; weniger als völlig schwach. **«Слабый»** (не «бессильный»).

Для значимого слова — Adelung + Grimm DWB точно с градациями, не только DWDS modern.

54. **Русская типографика в аннотациях, заголовках, `ru`-полях.** Строго:
- **«...»** (ёлочки) для кавычек, **не** `"..."` (ASCII)
- **—** (длинное тире) для разделителя/связки/паузы, **не** `-` (дефис только внутри сложных слов)
- **ё** где требуется орфографией: «ещё», «учётом», «всё», «чёрные», «нём», «осознаётся», «передаёт», «найдёт», «идёт», «замёрзшей», «твёрдой», «своё», «чётко», «задаёт». Особо легко пропускается в косвенных падежах личных местоимений и в личных формах глаголов на -ёт.
- **одинарный пробел**, не двойной
- Многоточие — три точки `...`
- **«Не что иное, как»** (раздельно — конкретизация «именно это»). **«Ничто иное»** (слитно — отрицание «ничто иное не интересует»). Пример: `Es ist nichts als der Winter` → «Это — **не что иное, как** зима».

**Обязательная программная ё-проверка после каждой записи**: `python -c "import json, re; d=json.load(open('FILE')); ... print(re.findall(r'\bнем\b|\bеще\b|задает|осознается|передает|\bидет\b|найдет|\bчетко\b|твердой|замерзш|\bвсем\b|\bсвое\b', text))"`. Каждое подозрительное слово — проверять вручную.

55. **Следственное `dass` без явного `so`/`derart` в главной части** (литературная архаика XIX в.) → русский следственный союз «**так что**». Без якоря русское «что» читается как комплементное. Контраст: `so lau, dass…` (явный `so`) → «столь… что…»; `Hab' wieder schwarze Haare, dass mir's… graut` (без `so`) → «У меня снова чёрные волосы, **так что** мне… страшно».

56. **Сигнализация о подменах предметов / концептов** (дополнение к 6). Если RU-эквивалент означает иной материальный предмет, чем DE (`Bahre` = открытые погребальные носилки vs «гроб» = закрытый ящик-Sarg) — **обязательно сигнализировать пользователю при первоначальном переводе** до записи в файл. Особенно важно при социально-историческом подтексте.

57. **Три источника Müller'а, два направления правки** (контекст для 22). Müller публиковал серийно: (1) **Urania 1823** (Brockhaus, Лейпциг) — 12 стихов Gute Nacht … Einsamkeit; (2) **Deutsche Blätter** 1823 (Бреслау) — 10 «продолжений» (Krähe, Mut, Wegweiser, Leiermann); (3) **Waldhornist II 1824** (Dessau) — все 24 с правками. Schubert: Часть I (1–12) по Urania, Часть II (13–24) по Waldhornist; **Deutsche Blätter в Вену 1827 не доходил**.

Расхождение sung-text vs обычный текст: (а) **Schubert vs Müller-final** — правка композитора; (б) **Müller 1823→1824** — правка поэта между Deutsche Blätter и Waldhornist.

**Алгоритм** (DB / WH / S): DB = WH = S, отличие у S → правка Schubert'а; DB ≠ WH = S → правка Müller'а; DB = S ≠ WH — невозможно. Алгоритм с тремя источниками применим только к 10 «продолжениям». Для Urania (1–12) и Waldhornist'а без DB — расхождение = правка Schubert'а напрямую. LiederNet / schubertsong.uk помечают вариант как «Müller» — часто Deutsche Blätter.

58. **Каждая стихотворная строка — с прописной.** При сегментации первого слова легко получить строчную (если по русской пунктуации не должно быть прописной). **Стихотворная конвенция выше синтаксической**: каждая строка `lines_ru` начинается с прописной. Проверка после записи: первый сегмент каждой строки.

59. **Грамматическая модальность в аннотациях.** Точно сохранять модальность DE: V1 + `?` → «спрашивает», не «просит»; императив (`lass...!`) → «просит / велит»; декларатив → «говорит». Не растворять разные модальности в общем «герой обращается».

60. **`mit` без явного Dat-объекта — наречие сопричастности.** DE `mit` без явного Dat рядом — **наречие** (Verbundensein, Zugehörigkeit, Dabeisein), не предлог. DWDS-примеры: `er fiel mit zu Boden` («он тоже упал»), `dürfen wir mit?`, `ich bin mit von der Partie`. RU перевод — **наречие** «тоже» / «вместе» / «заодно», **НЕ** «с + местоимение» (в DE местоимения нет; добавление в RU нарушает правило 18). Различие с предлогом: `mit ihm` (с ним) — предлог; `mit` без объекта — наречие. Пример: `Fall' ich selber mit zu Boden` → «падаю я сам **тоже** на землю».

61. **Сверка пунктуации DE с авторитетным источником.** При первоначальном переводе и финальной проверке — сверять пунктуацию каждой строки DE с Wikisource (Müller Waldhornist 1824 / Urania 1823), schubertsong.uk (sung-text), LiederNet. Особенно: запятая vs точка с запятой vs точка vs восклицательный знак. Программная проверка: вывести все lines_de и построчно сверить с источником.

62. **Winterreise — психологический цикл, не натуралистический.** Природные приметы **не всегда согласованы реалистически** между песнями: `bunte` листья (Песня 16) vs замёрзшие речки и слёзы (Песни 3, 6, 7) — противоречие, разрешаемое в символическом плане (Youens, Wikipedia). Müller не следует строгой природной хронологии. Не делать необоснованных утверждений о «глубокой зиме» / «поздней осени» на основании одной приметы — учитывать **все** приметы.

63. **Финальная систематическая проверка перед закрытием песни.** Обязательно:
- (а) каждый сегмент DE↔RU проверить пословно;
- (б) каждое фактическое утверждение в аннотациях факт-чекнуть (отсылки на песни — grep; DE-цитаты — корректность);
- (в) программная ё-проверка (правило 54);
- (г) проверка ASCII-кавычек `re.findall(r'"[А-Яа-яёЁ]')`;
- (д) проверка коротких дефисов между RU-словами;
- (е) капитализация первых сегментов строк (правило 58);
- (ж) JSON-валидация;
- (з) сверка пунктуации DE с источником (правило 61);
- (и) **список спорных мест** (вариант X vs Y с обоснованием выбора) явно показать пользователю до объявления песни финализированной.

Без этой проверки песня не считается финализированной.

64. **Futurperfekt через Präsens Perfekt + временное наречие будущего.** Aux+Partizip II + `morgen` / `bis morgen` / `übermorgen` = vollendete Zukunft (разговорный эквивалент Futur II). В RU — **будущее время**, не Präteritum. Признак: семантически прошедшее при наречии будущего нелогично. Пример: `Und morgen früh ist alles zerflossen` → «И завтра утром всё **растечётся**» (не «растеклось»).

65. **Tun-Periphrase (`tun + Inf` в Volkslied).** Народно-разговорная DE-конструкция, особенно в Volkslied-стилизации (Müller — типичный случай). `tun` — вспомогательный, **не «делать»**. Несёт habituell-эмфатику (постоянное, привычное действие). Семантически `Tun sich erlaben` ≡ `sich erlaben`. В RU функциональный аналог — народная частица **«знай»** (Ожегов, БТС: «постоянно, не переставая»; Пушкин, Кольцов, Никитин). Сегментация: tun → «знай» отдельным сегментом (не рамка), смысловой Inf — отдельно, в конце RU (правило 16). Пример: `Tun sich im Guten und Argen erlaben` → «Знай, себя в добром и злом услаждают».

66. **Funktionsverbgefüge — устойчивые DE-сочетания.** Устойчивые единицы типа `offen lassen`, `liegen lassen`, `stehen lassen`, `übrig lassen`, `zu Ende sein mit` — переводить **идиоматически одним RU-глаголом**, не покомпонентно. Сегментация: рамка без многоточия (если DE-слова рядом) или с многоточием (при разрыве). Пример: `noch übrig ließen` — рамка `übrig ließen` → «оставили»; `noch` → «ещё» (контекстуально передаёт `noch übrig` через присоединительное «ещё оставили»).

## Annotation Guidelines

Два типа: `lang` (язык/грамматика — синий) и `meaning` (смысл/интерпретация — коричневый). Нумерация независимая.

**Lang** — грамматика и структура: Verbklammer, порядок слов, падежи, род, отрицание, опускание союзов, идиомы, составные слова.

**Meaning** — интерпретация и контекст: этимология и связи; перспектива (смена лица); мотивы; многозначность; социальный контекст; фольклорные формулы; Schubert vs Müller с эффектом.

**Стиль аннотаций:**
- Перед написанием — искать литературоведов на DE/EN.
- Не категорично — несколько интерпретаций признавать.
- Без доказательств не утверждать.
- Не ссылаться на учёных по именам.
- **Не упоминать словари по именам** (не «по DWDS», «по Adelung'у»); пометы регистра не цитировать буквально. Источники — для подготовки и фактчека, не для текста.
- **Писать о тексте, не об учебнике.** Никакой лингв. терминологии. Не: «Konjunktiv II в немецком выражает нереальное условие». Да: «Locken — манить. Hin в конце добавляет направление: не просто манил, а заманил».
- **Каждый факт проверять** (правило 31). Для исторических — год **публикации**, не наблюдения.

### Технические правила аннотаций

- `line_span: N` — охватывает N строк. Сноска в конце последней. **Может пересекать границы строф.**
- `continuation_ranges: [[start, end], ...]` — диапазоны для каждой последующей строки. **`[-1, -1]` пропускает строку без подсветки**. Не использовать `null`.
- `segment_range: [start, end]` — индексы включительно.
- Аннотации могут перекрываться (один сегмент — и lang, и meaning).
- В тексте `\n` — перенос строки.

### Формат сегментов

- Отделяемые приставки: `"de": "pfiff'... aus"` (с многоточием при разрыве)
- Рамки предлогов: `"de": "Auf... Haus"`
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{ "ru": "женщины образ.", "de": "Frauenbild" }`
- Пустой `ru` — артикль без эквивалента
- Пустой `de` — RU слово, добавленное для грамматики
- **Пунктуация только в `ru`, не в `de`.** `{ "ru": "ах,", "de": "ach" }`.
- **`variant_de` / `variant_ru` — расхождения Schubert vs Müller** (правила 22, 57). Основное `de`/`ru` — Schubert sung-text; `variant_de` — мюллеровский (если различается); `variant_ru` — RU для мюллеровского (если отличается). Образец: `{ "ru": "угасшее,", "de": "erstorben", "variant_de": "erfroren", "variant_ru": "замёрзшее" }` (Песня 4). Для рамок: вариант кладётся на тот же сегмент-рамку. Если RU не различается — `variant_ru` не нужен.
