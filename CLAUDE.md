# CLAUDE.md — Winterreise

## Project Overview

Подстрочный перевод вокального цикла «Зимний путь» (Winterreise) Мюллера / Шуберта. 24 песни, немецкий + русский с пословной разметкой и аннотациями. Vue 3 + Vite, GitHub Pages.

## Architecture

### Страница

- **Левая колонка**: немецкий оригинал (строфами)
- **Правая колонка**: русский перевод; под каждым словом мелким шрифтом — соответствующее немецкое
- **Сноски** `¹²³` у отмеченных фрагментов; сквозная нумерация в пределах песни. Наведение — подсветка фрагмента, клик — пояснение
- **Режим пояснений** внизу страницы — все аннотации песни
- **Меню**: 24 песни (немецкие названия)
- **Заголовок песни**: немецкое + русское название
- Поиск, переключатель темы, печать/PDF

### Админ-режим

URL `?admin=<SECRET_KEY>`. Inline-редактирование русского/немецкого, добавление/правка аннотаций, сохранение через GitHub Contents API (токен в localStorage, права `repo`). Параллельно — правка JSON в Claude Code CLI.

### Data Model

JSON по одной песне в `src/data/songs/`:

```json
{
  "number": 2,
  "title_de": "Die Wetterfahne",
  "title_ru": "Флюгер",
  "stanzas": [{
    "lines_de": ["Der Wind spielt mit der Wetterfahne"],
    "lines_ru": [{
      "segments": [
        { "ru": "Ветер", "de": "Der Wind" },
        { "ru": "играет", "de": "spielt" },
        { "ru": "с", "de": "mit" },
        { "ru": "флюгером", "de": "der Wetterfahne" }
      ],
      "annotations": [
        { "type": "lang", "segment_range": [3, 3], "text": "..." }
      ]
    }]
  }]
}
```

**Правила данных:**
- `segments` — пары {ru, de} в порядке русского
- `de` может быть пустой строкой
- `segment_range` — индексы [start, end] включительно (подсветка)
- Одна `lines_ru` может покрывать несколько `lines_de` (не обязательно 1:1)

### Tech Stack & Structure

Vue 3 (Composition API, `<script setup>`) + Vite. Чистый CSS, plain JS (без TS, без UI-фреймворков). Деплой GitHub Actions → GitHub Pages.

- `src/data/songs/*.json` — песни (01–24)
- `src/data/index.json` — метаданные (number, title_de, title_ru, ready)
- `src/components/` — SongList, SongView, InterlinearLine, FootnoteMark, AnnotationsPanel, SearchBar, ThemeToggle; `admin/` (InlineEditor, AnnotationEditor, SaveButton)
- `src/styles/` — main.css, print.css, theme.css
- `src/utils/` — github-api.js, search.js, print.js
- `.github/workflows/deploy.yml`

## Development Guidelines

## Translation Rules

Подстрочник — пословное соответствие для изучающих немецкий, не литературный перевод.

**Точность — приоритет.** Русский должен быть грамотным и реально существующим в языке (проверять сочетаемость: «лежит на душе» — есть, «лежит на уме» — нет), но не обязан звучать красиво. Точное лучше привычного: `блестящих`, не `светлых`; `в споре`, не `наперебой`; `ударялся`, не `натыкался`. **При равной точности** — частотный вариант (`Bergstrom` → «горная река», не «горный поток»). **Устоявшийся русский перевод не аргумент держаться менее точного — традиция не довод.** Не добавлять отсутствующих в оригинале оттенков: уменьшительных суффиксов (`градинки` ≠ `Hagelkörner`), ложных архаизмов, аспектных значений (`взглянуть` мгновенно ≠ `sehn` нейтрально).

**Перевод с немецкого на русский. Английский язык не использовать** — ни в обсуждениях, ни в скобках. Кроме исключительных случаев.

**Порядок работы.** Перед предложением перевода — для КАЖДОГО знаменательного слова (включая частицы и наречия: `immerfort` = непрестанно, не «всегда»; `je` = хоть когда-нибудь, не «когда-нибудь»):

1. **DWDS** (dwds.de/wb/СЛОВО): значения, регистр, этимология. Для XIX в. также **Grimm** (dwds.de/wb/dwb/СЛОВО) и **Adelung 1811** (zeno.org/Adelung-1793) — ближайший к Мюллеру; Grimm DWB начат только 1838. Регистр и значения дрейфуют (starren в 1820-х первично «застыть», сейчас «уставиться»; `wild` у Adelung ядро = ungestüm «буйное», не «первобытно-дикое»).
2. **Разобрать на приставку + корень.** Er-sterben ≠ sterben, zer-schmelzen ≠ schmelzen.
3. **Близкие слова в песне развести** (Boden/Erde, Gras/Rasen, Blüte/Blumen).
4. **Контекст важнее этимологии, точность важнее традиции.** Сначала понять смысл строки целиком, потом найти русское выражение. Этимология может вести не туда: `irre Hunde vor ihres Herren Haus` = безумные, не бродячие (у бродячих нет хозяина). При **двух одновременных значениях** (`starrt` = и застыл, и уставился; `pfiff'...aus` = и свистит, и освистывает) — отметить оба. Наречие vs инструментал: `Kalt starrt ihr Bild darin` — «холодным застыл» (инструментал состояния), не «холодно застыл» (образ действия). **Частицы (schon, doch, ja, auch, wohl, je, noch, mal) меняют значение по контексту**: `Schon` ≠ всегда «уже» — в `Tret' ich auch schon` это усилитель уступки.
5. **Русская пунктуация.** Повторяющиеся союзы — запятая («И час, и день»). Запятые при обращениях, вводных, деепричастиях. **При замене слова перепроверить согласование**: «своё отражение» → «свой образ» (м.р.).
6. **Если выбор неочевиден — сигнализировать.** Не делать молча. Идиомы, архаизмы, многозначные слова, расхождение регистра — выносить на обсуждение.

**Источник немецкого текста: шубертовская редакция** (как поют), не оригинал Мюллера. Источник: Deutsche Schubert-Lied-Edition (Naxos). Орфография по реформе 1996: ß после кратких гласных → ss (Muß → Muss); после долгих остаётся (Blumenstrauß).

1. **Каждое немецкое слово — в подстрочнике.** Не пропускай служебные (da, schon, doch). Слово с самостоятельным переводом остаётся отдельным сегментом даже в рамке (`hinaus` → «прочь»). Но если русский предлог уже передаёт направление приставки — поглощается: `In die Felsengründe lockte... hin` → «В глуби заманил» («в» = `hin`).

2. **По умолчанию сохраняй немецкий порядок слов.** Цель — чтобы читатель видел соответствие. Перестройка — только когда русский **нечитаем**, а не когда слегка непривычен. В частности, **поэтическая инверсия V-letzt** (Müller часто ставит финитный глагол в конец строки ради рифмы: `Von der Straße her ein Posthorn klingt` — рифма с `aufspringt`) сохраняется в русском: «С улицы почтовый рожок доносится», не «доносится почтовый рожок».

3. **Меняй порядок только когда русский нечитаем.** «На моей прекрасной любимой доме» — нельзя; «Тут я подумал уже в своём безумии» — ок.

4. **Одно русское = одно немецкое (по возможности).** Группируй только когда иначе невозможно. Отделяемые приставки с ясным русским — отдельный сегмент (dahin → «вдаль», mit → «с собой»). Рамка с многоточием (pfiff'...aus) — когда русский не разделить на части. **Многоточие — только при разделении словами**; рядом стоящие (Folge nach, Komm her) — без.

5. **`de` пустая** — для слов, добавленных для русского смысла без немецкого эквивалента.

6. **Liebchen = любимая** (не «милая»). Lieb- от Liebe.

7. **СТОП: не писать содержательные изменения без явного приглашения.** Одобрение перед записью в файл (не только перед commit). Файл сразу виден на dev-сервере. Порядок: предложить → дождаться «ок» → писать → показать → «пушим» → commit/push. **Аннотации «от себя» по итогам фактчека/проверки нельзя добавлять без одобрения.** Исключение — мелкие правки прямо запрошенные.

8. **В аннотациях переводи немецкие фрагменты.** «(es eher bemerken — это раньше заметить)», не просто «(es eher bemerken)».

9. **Артикли (ein, der, die, das) ≠ «некий/этот».** Объединять с соседним сегментом: `{ "ru": "верной", "de": "ein treues" }`; либо пустой `ru`: `{ "ru": "", "de": "ein" }`. Артикли при существительных (der Wetterfahne) группируются с существительным/прилагательным.

10. **Предлоги — отдельные сегменты.** `{ "ru": "с", "de": "mit" }` + `{ "ru": "сердцами", "de": "den Herzen" }`, не «с сердцами / mit den Herzen».

11. **Связка sein в именном сказуемом → тире «—» в русском**: `{ "ru": "—", "de": "ist" }`. Не «есть»/«является». Распространяется на ist, seid, sind. **Прошедшее (war) и Konjunktiv II (wär')** требуют русского глагола — тире не передаёт ни времени, ни наклонения.

12. **Идиомы: буквальный перевод предпочтительнее, если грамматически возможен по-русски** (`im Streit` → «в споре», не «наперебой»). По смыслу — только когда буквальный невозможен/непонятен (`Was fragen sie nach` → «Что за дело им до»; `schwer im Sinn liegen` → «тяготить», т.к. «лежит на уме» не существует в русском). В обоих случаях — обязательна языковая аннотация. **Ложные соответствия идиом**: `sich zur Ruhe legen` = просто лечь; рус. «лечь на покой» = умереть/уйти в отставку — переформулировать (`zur Ruh'` → «отдохнуть»), допуская локальное отступление от правила 28.

13. **Косвенная речь.** В русском нет согласования времён. После «подумал, что...» одновременное действие → настоящее: «подумал, что он освистывает» (не «освистывал»).

14. **Возвратное притяжательное.** «Meinem» → «своём» (не «моём»), когда подлежащее = «я». «В своём бреду» естественнее.

15. **Род адаптируется.** Sie (Wetterfahne ж.р.) → он (флюгер м.р.). При значимом расхождении — языковая аннотация.

16. **Verbklammer.** Когда вспомогательный + содержательный глаголы образуют рамку (bin ich eingezogen), русское подлежащее ставится перед содержательным — как в немецком. Вспомогательный + Partizip — рамка-многоточие: `Fremd **bin** ich **eingezogen**` → `{ "ru": "я", "de": "ich" }, { "ru": "пришёл", "de": "bin... eingezogen" }`. Не «Чужим пришёл я». Содержательный глагол в рамке стоит в конце немецкой фразы — в русском тоже в конец: «На каждый камень натыкался», не «Натыкался на каждый камень».

17. **Предлог, поглощённый падежом.** Если немецкий предлог не имеет русского эквивалента (его роль играет падеж), сливаем с соседним: `gehüllt in Schnee` → «окутан снегом» — `{ "ru": "снегом", "de": "in Schnee" }` (творительный поглощает `in`).

18. **Никаких пустых сегментов (`ru = ""`).** Немецкое слово без русского эквивалента сливается с соседним в `de`: `Es zieht... mit` → «Следует»; `man mich` → «меня»; `bin... eingezogen` → «пришёл». **Пустой `de`** (при непустом `ru`) — для русских слов, добавленных для грамматики (например, «что» для dass). Перед добавлением проверить, не решается ли пунктуацией. V1-условия → «Если» с пустым `de`. Для AcI — «как»-clause (см. правило 40).

19. **Модальные глаголы в контексте.** Sollen во 2 л. (du sollst) — воля говорящего (заверение, обещание), не обязанность: `Sollst meinen Tritt nicht hören` → «Моих шагов не услышишь», не «Должна не слышать».

20. **Эллипсис глагола.** Немецкий допускает опущение глагола, русский — нет: `Sacht, sacht die Türe zu!` (только частица «zu»/«закрыто») → «Прикрою тихо-тихо дверь». Обязательна языковая аннотация.

21. **Не угадывать значения — проверять.** Fein ≠ милый (= прекрасный, изящный). Glühend ≠ обжигающе (от glühen = пылать; «пылающе»). Если у немецкого корня есть прямой русский эквивалент (stoßen → ударять, sich stoßen → ударяться), начинать с него. Приблизительный синоним (натыкаться) — только если корневой не работает.

22. **Шуберт vs Мюллер: сверять минимум по двум источникам** (LiederNet + schubertsong.uk + Hampsong Foundation; NSA если доступна) — направление правки легко перепутать. Если в одном источнике сноска о варианте, а в другом нет — доверять тому, у кого сноска. **Контекст**: 12 стихов в Urania 1823, весь цикл в Waldhornist II (1824) с правками; Шуберт песни 1–12 по Urania (правок 1824 не знал), 13–24 по Waldhornist. Для правок Мюллера, не дошедших до Шуберта, — «Шуберт не знал».

23. **Элизии (апостроф) в немецком — последовательно** в `lines_de`, `title_de`, поле `de` сегментов. Примеры: zieh', dacht', hätt', hab', Wär', Ruh', pfiff', Gefror'ne, An's. Перед многоточием рамки апостроф сохраняется (`pfiff'... aus`).

24. **Перевод через отрицание при расхождении коннотаций.** `lau` (тепловатый, акцент на «недостаточно горячий») → «негорячи», не «теплы» (рус. «тёплый» позитивно). `So lau` — «очень негорячий», не «очень тёплый».

25. **`So` в `so...dass` — всегда степень.** Переводится «так»/«настолько», никогда «едва/еле». Направление (много/мало) задаётся следствием в dass-придаточном.

26. **Konjunktiv II ≠ настоящее время.** Hätte/wäre/wollte — прошедшее + «бы»: wolltet → «хотели бы». После «словно/как будто» — прошедшее (с «бы» или без).

27. **Музыкальные вариативные повторы — через ` / `** в `lines_de`, поле `de` и `ru` сегментов: `"Und / Ihr dringt doch aus der Quelle"`, `{ "ru": "А / Вы", "de": "Und / Ihr" }`. Первый — основной, второй — повтор.

28. **Разводить близкие немецкие слова и сводить одинаковые.** Если в одной песне (или соседних) Boden/Erde, Flur/Wiese/Feld — разные русские. Обратное: одно немецкое слово — один русский эквивалент через все песни (`rauschen` → «шуметь», `Geselle` → «дружище»). Контекстуальное отступление допустимо, но обязательно сигнализировать. **Кластеры синонимов** (Schmerz/Leid/Leiden/Wehen; Fluss/Strom/Bach/Bächlein) продумывать заранее, для всего цикла, во избежание ретроактивных правок.

29. **Регистр двунаправленно.** Возвышенное немецкое (durchstreichen вместо durchqueren, die Flur вместо das Feld) → возвышенное русское. **Обратно тоже верно**: нейтральное немецкое → нейтральное русское. Не «украшать» поэтизмами обычное у Мюллера. Проверять DWDS (gehoben, dichterisch, veraltet) + Adelung 1811 и Grimm: DWDS `veraltet` отражает современный язык, слово могло быть стандартным в 1820-х (`ward`). Типичные русские ловушки без основания: «око» (чаще «глаз»), «знать» (чаще «наверное»), «грезить» (чаще «видеть во сне»), «вещать» (чаще «говорить»).

30. **Предлог или направительное наречие глагольной конструкции — в рамку с глаголом.** `suchen nach`, `fragen nach` → `{ "ru": "ищу", "de": "such'... nach" }`. То же для направительных наречий (`her`, `hin`, `herauf`, `hinaus`), грамматически примыкающих к глаголу движения/звучания: `Von der Straße her ein Posthorn klingt` → `her... klingt` идёт с глаголом (порядок в рамке отражает текстовый порядок слов: `her` раньше `klingt`), `Von` — отдельный предлог. **Не группировать `her` с `Von`** (выглядит как составной предлог, что грамматически неверно — `her` примыкает к `klingen` (herklingen), а не к `von`). Не группировать с управляемым существительным (`Nach... Spur` похоже на отделяемый `nachspüren`).

31. **Грамматические утверждения проверять** в DWDS, E-VALBU (IDS Mannheim), Duden. Первая ассоциация может быть ложной (пример: «suchen nach подразумевает неуверенность» — не подтверждается; реальное различие — nach подчёркивает интенсивность).

32. **Глагольные приставки несут смысл.** Zer- = разрушение (zerbrechen, zerstören). Er- = завершение процесса (ersterben — угаснуть, erstarren — оцепенеть). Ver- = изменение/утрата. Не переводить только корень: zerschmelzen ≠ schmelzen; erstorben ≠ tot (угасло ≠ мертво).

33. **Sein + Partizip II от глаголов изменения = Перфект, не связка.** «Die Blumen sind erstorben» = цветы угасли (Perfekt), не «цветы — мертвы». Правило 11 (sind → тире) не применяется к Перфекту. Признак: причастие от глагола изменения (ersterben, einziehen, aufstehen).

34. **Futurisches Präsens.** Wenn + Präsens → русское **будущее**, не настоящее: `Wenn die Gräser sprossen wollen` = когда/если травы захотят прорасти. Отличать от реального настоящего (`Kalt starrt ihr Bild darin` — застыл сейчас).

35. **Спектр значений: абстрактное и предметное.** Контекст определяет. `Angedenken` = и воспоминание, и памятка/сувенир — `mitnehmen` (взять с собой) указывает на предметное. `Irre` = и безумный, и бродячий — псы у дома хозяина (= не бродячие). `Fluss` ≠ автоматически «река»: общее слово для текучей воды; контекст (прозрачная, буйная, замерзает, поэт сам называет Bach) — «речка». **Предлоги тоже многозначны**: `Durch` = «через» (поперёк) или «по» (вдоль), зависит от направления. `Durch des Bergstroms trockne Rinnen / Wind' ich ruhig mich hinab` — герой вьётся вниз вдоль русел.

36. **Cross-distribution: смыслы распределены по-разному.** Если немецкая конструкция и русский эквивалент распределяют смыслы между словами по-разному — не писать ложных пар. Пример: `Ob's... wohl` ↔ «Уж не... ли» (`ob` = «не...ли», `wohl` = «уж»). Нельзя `{ "ru": "ли", "de": "wohl" }`. Решение: вся немецкая конструкция в один сегмент с рамкой-многоточием, лишняя русская частица — пустой `de`:
```json
{ "ru": "Уж не", "de": "Ob's... wohl" },
{ "ru": "под",   "de": "unter" },
{ "ru": "его",   "de": "seiner" },
{ "ru": "ли",    "de": "" },
{ "ru": "коркой","de": "Rinde" }
```
Сопровождать языковой аннотацией.

37. **Связность ландшафтной сцены.** Проверять согласованность отдельных слов с общей картиной. «Скальные лощины» (пологие низины) ≠ «горная река» + «спускается вниз» — один из переводов неточен.

38. **Классические/библейские аллюзии — сохранять узнаваемость.** `Jeder Strom wird's Meer gewinnen` = Экклезиаст («все реки текут в море»). «Каждая река обретёт море» — узнаваемо; «каждый поток обретёт море» — теряет.

39. **`erst` в `nun erst`, `jetzt erst`, `in... erst` — временной контраст, не первичность.** Переводить «лишь»/«только» (= не раньше, только сейчас), **не «впервые»**. `Nun merk' ich erst` → «Только теперь замечаю»; `in der Still' erst` → «лишь в тишине». `erst` относится к обстоятельству времени/места, не к объекту/действию.

40. **AcI (Acc + Inf) в русском невозможен.** `Fühlst deinen Wurm... sich regen` — калька «чувствуешь своего червя... встрепенуться» неграмотна. Обязательна **«как»-конструкция**: «Чувствуешь, как твой червь... встрепенётся». «Как» — пустой `de` (грамматическая добавка). Двоеточие — альтернатива, но работает не везде (длинный AcI с инструментальным расширением — не передать).

41. **Возвратные русские глаголы + `Mit` — сохранять «с».** `Mit + Dat + sich-verb` (`sich regen mit heißem Stich`) — `Mit` = comitative/manner, не чистый инструменталь. Русские возвратные (встрепенуться, обернуться) **не принимают чистый творительный**: `Mit heißem Stich sich regen` → «С горячим жалом встрепенётся», не «Горячим жалом встрепенётся». Отличается от правила 17: там Mit чисто инструменталь и поглощается падежом; здесь предлог обязателен.

42. **Не приписывать оригиналу переклички из перевода.** Если наш русский создал параллель (`unwirtbarem` → «бесприютном» и `Obdach` → «приют»), проверить оригинал: разные ли корни в немецком (`Wirt` vs `Dach`)? Если да — параллели у Мюллера нет, мы её создали. Допустимо, но не подавать как авторский ход.

43. **Перед переводом значимого слова — grep по всем песням.** `grep '"de": "<слово>"' src/data/songs/*.json` — посмотреть прежние переводы. Либо следовать (правило 28), либо сознательно отступить с сигнализацией. Пример: `träumte` в драфте «грезил», тогда как в песне 5 `träumt'` уже было «снилось» с аннотацией «грезил» как отвергнутая альтернатива — grep вскрыл бы.

44. **Избегать русских конструкций с поверхностной омографией.** Глаз сбивается на: «так, как» (сравнит.) vs «так как» (каузальн.); «что бы» vs «чтобы»; «то же» vs «тоже»; «также» vs «так же». Запятая формально разводит, но при беглом чтении наводится первая интерпретация. Для коррелятивного `so wie` использовать однозначные эквиваленты: «Такими, какими…», «Как…», «Подобно тому, как…». Избегать «Так, как» в начале строки. Принцип: если фраза двусмысленна на беглом восприятии — переформулировать.

45. **Компаунды через дефис для непереводимых корневых семей.** Немецкое слово — часть семьи (`Traum / träumen / Träumer`), в русском — разные корни (`сон / сниться / мечтатель`): для ключевых узлов компаунды (`Traum → сон-мечта`, `Träumer → сновидец-мечтатель`). Применять, когда: (а) многозначность важна для поэтики; (б) ни один русский синоним не покрывает всё (БНРС: разные эквиваленты по корням); (в) контекст требует всей палитры. Дефисный компаунд — обычная грамматика (ср. «поэт-переводчик»), оба компонента склоняются.

46. **«Дословно»/«буквально» — точная семантическая граница.** «Дословно X — Y» требует, чтобы Y был пословно-буквальным переводом X (корневым, грамматически параллельным). Если покрывается часть — формулировать осторожнее: «в одном из смысловых пластов», «этимологически», «по корню», «в одном из словарных значений». Пример ошибки: «Träumer — дословно «сновидец»» — «сновидец» только одна ячейка из träum-семьи.

47. **В lang-аннотациях цитировать фактическую форму строки, не лемму.** У Мюллера `denke dem Traume nach` — не писать «У Мюллера `dem Traume nachdenken`» (инфинитива в тексте нет). Лемма указывается отдельно: «Глагол `nachdenken` — …», но не атрибутируется Мюллеру.

48. **Структурные утверждения о цикле проверять чтением песен.** «X — последнее появление Y», «мотив только в Z», «здесь впервые…» — литературоведы упрощают; конкретный текст может опровергнуть. Сквозная проверка: grep + чтение релевантных песен. Пример провала: «Maid в песне 11 — последнее появление возлюбленной» — неверно (в 13 `Liebchen`, в 19 `liebe Seele`). **Утверждения о составе/порядке исторических изданий (Urania 1823, Waldhornist 1824) — по нескольким независимым источникам** (Wikipedia EN/DE, Youens, Reed) — один агент или источник может ошибиться.

49. **Творительный образа действия — слово действия/манеры, не часть тела.** Дополнение к правилу 17. Немецкое `mit + Adj + N` для образа действия (manner of walking/speaking/looking) → русские модели «X шагом/поступью/походкой» (ходьба), «X голосом/взглядом/дыханием». НЕ работают: «X ногой/стопой» для ходьбы, «X рукой» в смысле жеста. Пример: `mit trägem Fuß` → «вялым шагом» (не «вялой ногой/стопой»). Это русское ограничение сочетаемости: немецкое Fuß в идиоме walking-manner → русское «шаг».

50. **Dass-Ausrufesatz: `Ach, dass... so [adj]!` — конструкция Klage.** Самостоятельное dass-предложение как восклицание = сетование/жалоба, не нейтральное удивление. К 1820-м архаично-формально; Müller использует осознанно. Источники: Adelung 1811 (закрытый список 8 функций — все негативно-эмоциональные); Grimm DWB §15 «meist einen guten oder bösen wunsch enthält»; Heyse 1814 — Bewunderung через `wie + Adj`, не через `dass`; DTA 1780-1830 — 0 positive marvel. Простое удивление — без dass: `Ach, [Subj] so [Adj]!`.

Перевод: **«Ах, что [Subj] так [Adj]!»** + частица **«же»** для Klage-интенсификации. Структурно: dass→что, so→так (раздельные сегменты), «же» — пустой `de` (правило 18). «Как же» теряет однозначную Klage (в русском амбивалентно — восхищение/сетование). «Как жаль, что...» — снижает регистр.

51. **Шубертовские прямые повторы внутри строки** (дополнение к правилу 27). Schubert часто повторяет слова/фрагменты в вокальной партии без изменения текста. `Lines_de` отражает sung-text, не печатный Müller'а. Проверять партитуру (IMSLP, NSA) до финального lines_de. Пример: Müller `War ich so elend nicht` → Schubert `War ich so elend, so elend nicht`. **Повтор как ресурс перевода**: одно немецкое слово, повторённое дважды, — два слота для разных русских пластов: `so elend, so elend` → «так жалок, так несчастен» (внешний пласт jämmerlich + внутренний unhappy). Применимо когда (а) у немецкого многозначность важна, (б) один русский не покрывает всё, (в) повтор Шуберта даёт два слота.

52. **Регистровый переворот за 200 лет** (дополнение к правилу 29). Регистр некоторых немецких слов изменился между 1820-ми и современностью, иногда **противоположно**. Пример: `hell` — Adelung 1811 `edler` (более возвышенное); `licht` — `«im gemeinen Leben»` (разговорное). Современный DWDS наоборот: `hell` нейтральное, `licht` `gehoben`. Müller'у был доступен 1820-е расклад. **Доверять Adelung 1811 для регистра 1820-х, не современным DWDS-пометам**.

53. **Adelung'овская «мягкая семантика» — точные градации, не сильные эквиваленты** (дополнение к правилам 21, 29). У ряда немецких слов Adelung фиксирует более точечную/мягкую семантику, чем «прямой» русский эквивалент. Формулировки «erste Grad», «schwächere Bedeutung», «mit Widerwillen» **значимы**. Примеры:
- `trübe` — «erste Grad der Unlust» (Adelung); «schwächere als betrübt» (Grimm). НЕ «мрачный» (= düster/finster, сильнее), а **«сумрачный»**
- `träge` — «mit Widerwillen sich bewegend»; «erschlafft, kraftlos» (Grimm). НЕ «медленный», но **«вялый»** (физическое + психологическое)
- `elend` — jämmerlich + bedrückt + (этимол.) `ali-lanti` = «в чужой земле/изгнании». «Несчастен» (только эмоц.) теряет пласты; «жалок/убог» теряет внутреннее. Через шубертовский повтор (правило 51) — распределение

**Подход:** для значимого слова до выбора русского — прочесть Adelung + Grimm DWB точно (с градациями). Не останавливаться на DWDS modern.

## Annotation Guidelines

Два типа: `lang` (язык/грамматика — синий) и `meaning` (смысл/интерпретация — коричневый). Нумерация независимая.

### Языковые аннотации (lang)

Грамматика и структура: Verbklammer (hätt'...sollen), порядок слов (V2/V-last, отделяемые приставки), падежи (Genitiv, приложения), род (Wetterfahne ж.р. → флюгер м.р.), отрицание (двойное `nimmer nicht` = утверждение в немецком, в русском — усиление), опускание союзов (dass после глаголов мышления), идиомы (буквальное vs реальное значение), составные слова (Frauenbild = Frauen + Bild).

### Смысловые аннотации (meaning)

Интерпретация и контекст:
- **Этимология и связи** (Wetterfahne — wetterwendisch)
- **Перспектива** — смена лица (я → он → они), самоотчуждение
- **Мотивы** — связи между строфами и песнями цикла (ветер, флюгер, непостоянство)
- **Многозначность** — несколько пластов смысла (сердца = любовь + расчёт)
- **Социальный контекст** — критика мещанства, неравенство статусов
- **Фольклорные формулы** — устойчивые обороты Volkslied (Feinsliebchen) объяснять как традицию, не индивидуальный авторский ход
- **Шуберт vs Мюллер** — отмечать оригинал и музыкальный/смысловой эффект

**Стиль аннотаций:**
- Перед написанием — искать литературоведов на немецком/английском
- Не быть категоричным — несколько интерпретаций признавать
- Не утверждать без доказательств (герой — подмастерье? из текста не следует)
- Не ссылаться на учёных по именам (не «по Гримму», а собственное наблюдение)
- **Писать о тексте, не об учебнике.** Никакой лингвистической терминологии (не «делиберативное», «предложное управление», «переходный глагол»). **Не:** «Konjunktiv II в немецком языке выражает нереальное условие». **Да:** «Locken — манить. Hin в конце добавляет направление: не просто манил, а заманил. Как часто в немецком, важный оттенок — в конце.»
- **Каждое фактическое утверждение проверять** по авторитетным источникам (правило 31). Для исторических утверждений — год **публикации**, не год наблюдения (Бессель наблюдал Irrlicht в 1807, опубликовал в 1838 — Мюллер не мог знать).

### Технические правила аннотаций

- `line_span: N` — охватывает N строк (текущую + последующие). Сноска в конце последней. **Может пересекать границы строф** (lines идут последовательно через всю песню).
- `continuation_ranges: [[start, end], ...]` — диапазоны сегментов для каждой последующей строки (line+1, line+2, ...). Без него подсвечиваются все. **`[-1, -1]` пропускает строку без подсветки** (для аннотаций с разрозненными строками).
- `segment_range: [start, end]` — индексы включительно.
- Аннотации могут перекрываться (один сегмент — и lang, и meaning).
- В тексте `\n` — перенос строки (CSS `white-space: pre-line`).

### Формат сегментов

- Отделяемые приставки: `"de": "pfiff'... aus"` (с многоточием)
- Рамки предлогов: `"de": "Auf... Haus"`
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{ "ru": "женщины образ.", "de": "Frauenbild" }`
- Пустой `ru` — артикль без эквивалента
- Пустой `de` — русское слово, добавленное для грамматики
