## ⚠️ ПРОТОКОЛ ПЕРВОЙ ЗАПИСИ ПЕСНИ

Перед первой записью JSON — выполнить `python3 scripts/pre-translation-check.py <N>` (grep значимых DE-слов; для NEW слов — поиск семьи по корню; lines_de для сверки пунктуации; чеклист).

После скрипта:
1. **Вывод в чат**, прокомментировать каждое совпадение по словарной семье (применить конвенцию или отступить с сигнализацией).
2. **Сверка DE-текста** по иерархии (см. «Источник DE»): лексика — NSA + автограф; экспрессивная пунктуация и апострофы — Müller WH1824 + Rastl; Naxos PDF справочный (12+ опечаток); LiederNet / Hampson — НЕ использовать.
3. **Список спорных мест** с альтернативами (правило 6).
4. **Müller vs Schubert** различия явно.
5. **Получить одобрение** по каждому пункту (правило 7).
6. **Только после этого** — JSON.

## ⚠️ ПРОТОКОЛ «ПРОВЕРКА ТОЧНОСТИ ПЕРЕВОДА»

Триггеры: «проверь точность», «исследуй слово», «полное исследование».

**Главный принцип**: Müller 1820-х — переводим **его смыслы**, не современные.

**Приоритет источников**:
1. **Смыслы 1820-х**: Adelung Wien 1811 + Grimm DWB — **ГЛАВНОЕ**. Если Adelung ≠ DWDS modern — у Müller'а Adelung.
2. **Контекст** оригинала + цикла — главное наравне со словарями 1820-х (поэт. компаунды имеют расширенное контекст. значение).
3. **DWDS + Pfeifer** — современное/этимология (вспомог.).
4. **Duden + Duden Synonyme** — синонимы (вспомог.).
5. **БНРС, Multitran, PONS, Langenscheidt DE→RU** — RU-кандидаты (вспомог.).
6. **БРНС / Multitran RU→DE** — только sanity check, **не аргумент**.
7. **RU поэт. традиция XIX в.** — последняя очередь, только при явном цитировании (галлюцинации).

**Процедура**:
- **Шаг 0 (форсинг)**: `python3 scripts/word-check.py <DE-слово> [RU-кандидат...]` — все 14 источников. **Полный вывод в чат**. Для ⚠ источников (DWDS, Pfeifer, Duden, Duden Syn, PONS, Langenscheidt, Leipzig — SPA/JS) — WebFetch URL вручную. Не повторять Шаг 0 для слова, исследованного в диалоге.
- **Шаг 1**: анализ — Adelung+Grimm→1820-е; DWDS/Duden/Wiktionary→совр.; БНРС/Multitran/PONS/Langenscheidt→RU.
- **Шаг 2**: контекст (приоритет над словарём).
- **Шаг 3** (контрольный): обратная RU→DE сверка.

Пример: `törichtes` — Adelung: «ehedem unsinnig, toll»; контекст цикла (безумие) → «безумное», не «безрассудное».

**Без Шагов 0-3 заявлять «исследование проведено» ЗАПРЕЩЕНО.**

## Project Overview

Подстрочный перевод цикла «Зимний путь» (Winterreise) Мюллера/Шуберта. 24 песни, DE+RU с пословной разметкой и аннотациями. Vue 3 + Vite, GitHub Pages.

## Architecture

**Страница**: левая колонка — DE оригинал (строфами); правая — RU перевод, под каждым словом DE мелким шрифтом. Сноски `¹²³` с подсветкой/пояснением. Меню — 24 песни. Заголовок: DE+RU. Поиск, тема, печать/PDF.

**Админ-режим**: URL `?admin=<SECRET_KEY>`. Inline-редактирование через GitHub Contents API (токен `repo` в localStorage). Параллельно — JSON в Claude Code CLI.

**Data Model** — JSON по песне в `src/data/songs/`:

```json
{
  "number": 2,
  "title_de": "Die Wetterfahne",
  "title_ru": "Флюгер",
  "stanzas": [{
    "lines_de": ["Der Wind spielt mit der Wetterfahne"],
    "lines_ru": [{
      "segments": [
        { "ru": "Ветер", "de": "Der Wind" },
        { "ru": "играет", "de": "spielt" },
        { "ru": "с", "de": "mit" },
        { "ru": "флюгером", "de": "der Wetterfahne" }
      ],
      "annotations": [{ "type": "lang", "segment_range": [3, 3], "text": "..." }]
    }]
  }]
}
```

**Правила данных**: `segments` в порядке RU; `de` может быть пустой; `segment_range` — индексы [start, end] включит.; `lines_ru.length == lines_de.length` (Verbklammer через строки — рамка-многоточие в одной из `lines_ru`); строфы — по Müller WH1824 (каждая авторская — отдельный объект); `title_annotations` — массив `{type, text}` без `segment_range`.

**Tech Stack**: Vue 3 (Composition API, `<script setup>`) + Vite + CSS + plain JS. GitHub Actions → GitHub Pages. `src/data/songs/*.json` (01-24), `src/data/index.json` (метаданные), `src/components/`, `src/styles/`, `src/utils/`.

## Translation Rules

**Правила ниже — обобщения опыта, не аксиомы.** При противоречии правила и грамматики — приоритет грамматике.

Подстрочник = пословное соответствие, не литер. перевод.

**Точность > поэтика.** RU грамотный, существующий (проверять сочетаемость), но **не обязан звучать красиво**. Точное > привычного (`блестящих`, не `светлых`; `в споре`, не `наперебой`). При равной точности — частотный вариант. **Устоявшийся RU — не аргумент** держаться менее точного. Не добавлять отсутствующих оттенков (уменьшит. суффиксы, ложные архаизмы, аспект). **Поэтически переводим только если DE поэт. в 1820-х** (Adelung gehoben/dichterisch, Grimm poetisch).

**Многословный/дефисный перевод обязателен, если точнее** (правило 45). Примеры: `Verlangen → «страстное стремление»`, `Täuschung → «обман-иллюзия»`.

**Перевод DE→RU. Английский не использовать** (кроме исключений).

**Порядок работы** для каждого знаменательного слова (вкл. частицы/наречия: `immerfort` = непрестанно; `je` = хоть когда-нибудь):

1. **DWDS** (значения, регистр, этимология). Для XIX в. также **Grimm** (`dwds.de/wb/dwb/<W>`) и **Adelung Wien 1811** через BSB. Регистр дрейфует.
2. **Разбор на приставку + корень** (Er-sterben ≠ sterben).
3. **Близкие слова развести** (Boden/Erde, Blüte/Blumen).
4. **Контекст > этимологии и словаря**. При двух значениях (`starrt` = застыл+уставился) — отметить оба. Модальные предположения (`mag` = должно быть) — отдельный RU. Поэт. компаунды имеют расширенное контекст. значение (Felsengrund по DWDS = Felstal, но в П.9 → «скальные ущелья»; Wüstenei — Grimm включает felsige Berglandschaften).

**Частицы doch/denn/nun по контексту** (конвенции цикла):
- **denn**: Ja/Nein- или soll/sind-V1-вопрос → «**Неужели**» (П.03, 04, 21); w-вопрос → «**же**» (П.13, 20); discourse «then» после Nun → «**тогда**» (П.21).
- **doch**: эмфат. подтверждение → «**ведь**» (П.03); с `ja` → «**ведь же**» (П.20); в w-вопросе → «**же**» (П.06); в Ja/Nein-вопросе контраст → «**И всё же**» (П.21).
- **Nun**: temporal → «**Теперь**» (П.01, 05, 10, 11, 15); discourse → «**Ну да**» / «**Что ж**» (П.13, 21).
- **wohl** (модальная): всегда «**, наверное,**» вводным (П.11, 13, 15, 21).
- **so** в `so... dass` — степень (правило 25).
- **schon** в `Tret' ich auch schon` — усилитель уступки.

5. **RU пунктуация.** Запятые при обращениях/вводных/деепричастиях. **Не калькировать DE автоматически** (`Nun, ...` DE → «Теперь не далеко» RU без запятой). При замене слова — перепроверить согласование («своё отражение» → «свой образ» м.р.).
6. **Если выбор неочевиден — сигнализировать**. Идиомы, архаизмы, многозначные слова, расхождение регистра — выносить на обсуждение.

**Источник DE** — шубертовская редакция (как поют). Иерархия по убыванию авторитета:

1. **Автограф Schubert'а 1827** — `schubert-scores/autograph-schubert.pdf` (74 стр., Часть I+II; рукопись с зачёркиваниями). Первоисточник. Описки есть (П.8 L5 `jeden` vs грам. `jedem`).
2. **NSA Bärenreiter, ред. W. Dürr** — `schubert-scores/nsa-baerenreiter.pdf` (82 стр., NSA-номера 110-191). Urtext, исправляет описки автографа. **В нотах** упрощает экспрессивную пунктуацию и опускает апострофы (нотная конвенция).
3. **Rastl Original Spelling** (schubertsong.uk, «Original Spelling»). Реконструкция Schubert sung-text в Müller-орфографии. **Лучший для апострофов/пунктуации.**
4. **Müller WH1824** (Wikisource: `Die_Winterreise_(Müller)`). Оригинал. Понимание «что Schubert изменил» + проверка пунктуации Schubert'а.
5. **Naxos PDF** (Deutsche Schubert-Lied-Edition, 1999). **Опечатки 12+ в 20 песнях** (`mit zum` П.16, `Habe ich ja` П.20, `im mattem` П.18). Не основной.

**Когда сверяться**:
- Слово/артикль/падеж → NSA + автограф.
- Экспрессивная пунктуация (`!`,`?`,`;`,`—`) → Müller + Rastl.
- Апострофы (`hab'`, `in's`, `'s`) → Rastl.
- Поэт. капитализация (`Einem/Andern`, `Manches`, `Alles`) → Müller + Rastl (NSA в нотах строчные — нотная конв., не текстуальная).
- Шубертовские повторы (правило 27) → NSA + Naxos.

Орфография по реформе 1996: ß после кратких → ss (Muß→Muss); после долгих остаётся (Blumenstrauß). Th→T (Thore→Tore, Thränen→Tränen) — наша конвенция модернизации.

1. **Каждое DE-слово — в подстрочнике.** Не пропускать служебные (da, schon, doch); знаменательные (модал./част./нареч.) обязаны иметь RU (`mögen` 3 = «должно быть»). Слово с самостоятельным RU — отдельный сегмент даже в рамке (`hinaus → прочь`, `nach → вслед`). RU-предлог поглощает приставку, если передаёт направление (`In die Felsengründe lockte... hin → «В глуби заманил»`). **Прозрачность**: буквальный разбор > словарный (`Gibt... sich hin → отдаёт... себя`, не «предаётся»).

2. **По умолчанию сохранять DE порядок слов.** Перестройка — только когда RU **нечитаем**. **Поэт. инверсия V-letzt** (`Von der Straße her ein Posthorn klingt` → «С улицы почтовый рожок доносится») сохраняется. **Поэт. инверсия Gen впереди** (`meiner Hoffnung Grab` → «На моей надежды могиле») тоже. **Постпозитивная аппозиция** (`Der Winter kalt und wild`) — поэт. оборот, прил. без склонения (предикат-эквивалент): «Зима холодная и буйная».

3. **Менять порядок только когда RU нечитаем.** «На моей прекрасной любимой доме» — нельзя; «Тут я подумал уже в своём ослеплении» — ок.

4. **Одно RU = одно DE по возможности.** Группировать только когда иначе невозможно. Отделяемые приставки с ясным RU — отдельный сегмент (`dahin → «вдаль»`). Рамка с многоточием (`pfiff'...aus`) — когда RU не разделить. **Многоточие — при разрыве словами в DE**; рядом стоящие (Folge nach, Komm her) — без многоточия. **Порядок в рамке = текстовый порядок DE**.

5. **`de` пустая** — для DE без RU-эквивалента (внутри рамки).

6. **Liebchen = любимая** (не «милая»). Lieb- от Liebe.

7. **Не писать содержательные изменения без приглашения. Не пушить без явного «пушим»/«делаем».** Одобрение перед записью в файл (не только перед commit; файл сразу виден на dev-сервере). Порядок: предложить → «ок» → писать → показать → «пушим» → commit/push. **Аннотации, `variant_de`/`variant_ru`, ретроактивные правки — каждый случай отдельно.** Одобрение конкретного шага не распространяется на сопутствующие. **Условное одобрение** («если всё чётко, пушим») — сначала результат, потом действие.

8. **В аннотациях переводить немецкие фрагменты**: «(es eher bemerken — это раньше заметить)», не просто «(es eher bemerken)».

9. **Артикли (ein, der, die, das) ≠ «некий/этот»** по умолчанию. Артикль **слева** от существительного — объединять со СЛЕДУЮЩИМ сегментом: `{"ru":"верной","de":"ein treues"}`. **НЕ группировать с предыдущим** (`Welch ein törichtes` → `Welch` + `ein törichtes` + `Verlangen`). **Исключения** — артикль несёт смысл: (а) числительное «один» при контрасте (`Einen Weiser` П.20 = «Один указатель»); (б) демонстрат. `das` без сущ. (`Das nenn ich einen Morgen` П.18 → «Это называю...»); (в) анафор. `der` = конкретный известный (П.08 «der Tag» = «тот день»).

10. **Предлоги — отдельные сегменты.** `{"ru":"с","de":"mit"}` + `{"ru":"сердцами","de":"den Herzen"}`.

11. **Связка sein → «—» отдельным сегментом, и её исключения.** (а) **Основное**: bin/bist/ist/sind/seid — тире, не «есть»/«является». Применяется к любому именному сказуемому (сущ., прил., местоим., adjektives Partizip — статив; модификатор `wie`/`manche Stunde`). **Рамка `Adj. + sein-форма` запрещена**. Тире — RU-стандарт даже при DE Verbletzt (`wie müd' ich bin` → «усталый / — / я,»). Прошедшее war / Конъюнктив II wär' — RU-глагол. (б) **Modal + sein-Inf** — пара RU-глаголов (`Könnt... sein → «Можете... быть»`, не тире). (в) **Perfekt от глаголов изменения** (бывш. правило 33): «Die Blumen sind erstorben» = цветы угасли (Perfekt) — основная связка не применяется. Признак: причастие от глагола изменения (ersterben, einziehen, aufstehen). **Модификатор `wie` блокирует Perfekt-чтение**: `Mein Herz ist wie erstorben` — адъективное причастие (статив), связка через тире («есть словно угасшее»); аналогично `wie tot`, `wie betrunken`. (г) **Funktionsverbgefüge** (бывш. правило 66): `offen lassen`, `liegen lassen`, `zu Ende sein mit` — переводить **идиоматически одним RU-глаголом**, не покомпонентно. Сегментация: рамка без многоточия (DE рядом) или с многоточием (при разрыве).

12. **Идиомы**: буквальный перевод предпочтительнее, если грамматичен (`im Streit → в споре`). По смыслу — когда буквально невозможен (`Was fragen sie nach → Что за дело им до`). **Ложные соответствия**: `sich zur Ruhe legen` = просто лечь, RU «лечь на покой» = умереть. **Сдвиг подлежащего**: `ich hatte X → у меня была X`; `was hat es? → что с ним?`. **Сдвиг управления**: DE транзитив (Akk. без предлога) → RU предложное (`klagen + Akk`: `was es mir klagt → «на что жалуется»`, не «что»). **Однозначные идиомы**: `bei sich denken → подумать про себя` (молча). **Все случаи — с lang-аннотацией.**

13. **Косвенная речь.** В RU нет согласования времён. После «подумал, что...» одновременное действие → настоящее: «подумал, что он освистывает».

14. **Возвратное «свой» — только в пределах одной RU-клаузы.** DE-посессив, согласованный с субъектом своей клаузы → «свой». При разбиении на две клаузы (AcI через «как»/«:», подчинит., новое предл.) — «свой» через границу не работает: антецедент извне через «его/её/их». Пример: `Mein Herz sieht ... sein eignes Bild` → «Моё сердце видит: на небе нарисован **его** собственный образ» (правило 40).

15. **Род адаптируется.** Sie (Wetterfahne ж.р.) → он (флюгер м.р.). При значимом расхождении — lang-аннотация.

16. **Verbklammer и tun-Periphrase.** Aux + Partizip II = рамка-многоточие (`bin... eingezogen → «пришёл»`). Aux + Inf (Modal-Klammer, Futur): `hätt'... sollen`, `wird... gewinnen`. RU подлежащее перед смысловым глаголом, как в DE. Содержательный verb в конце DE → в конце RU. **НЕ Verbklammer**: стандартный Nebensatz S-Inf-V-fin (`was ich zittern kann` — субъект+инф.+финит. модал — стандартное придаточное, не рамка). **Tun-Periphrase** (бывш. правило 65) в Volkslied: `tun` — вспомог. (не «делать»), несёт habituell-эмфатику. RU аналог — народная частица **«знай»** (Ожегов: «постоянно, не переставая»). Сегментация: tun → «знай» отдельным сегментом, смысловой Inf — в конце RU. Пример: `Tun sich... erlaben` → «Знай, себя... услаждают». **Субстантивированный инфинитив** (das Klagen) → RU инфинитив, не отглаг. сущ. (глагольная природа + связь с финитной формой): `Klagen ist für Toren → «Жаловаться — для безумцев»` (не «Жалобы»; связь с `klagt`).

17. **Предлог, поглощённый падежом, + mit-конструкции.** `gehüllt in Schnee → «окутан снегом»` — `{"ru":"снегом","de":"in Schnee"}` (творит. поглощает `in`). **Творительный образа действия** (бывш. правило 49) — слово действия/манеры, не часть тела: DE `mit + Adj + N` → RU «X шагом/поступью/походкой» (ходьба), «X голосом/взглядом» (речь/взгляд). НЕ «X ногой/стопой» для ходьбы (`mit trägem Fuß` → «вялым шагом»). **`mit` без явного Dat — наречие сопричастности** (бывш. правило 60), не предлог: «тоже»/«вместе»/«заодно», НЕ «с + местоимение» (`Fall' ich selber mit zu Boden` → «падаю я сам тоже на землю»). Различие: `mit ihm` — предлог; `mit` без объекта — наречие.

18. **Никаких пустых сегментов (`ru = ""`).** DE без RU — сливается с соседним в `de` (рамка). **Пустой `de`** (при непустом `ru`) — для RU-добавок для грамматики: «что» для dass, «Если/Когда» для V1-условий, «как» для AcI (правило 40), «Тот» как восстановленный антецедент `der`, RU-предлог при DE Dat./Akk. без предлога (`mir → ко мне/от меня/у меня`). Перед добавлением — не решается ли пунктуацией. **`'s` = es** (формальный субъект) сливается с глаголом (`'s führt → Ведёт`). **Различение `'s`**: перед глаголом = `es` (поглощается); перед сущ. ср.р. или после предлога = `das` (артикль, правило 9: `An's Haus`).

19. **Модальные глаголы в контексте.** Sollen во 2 л. (du sollst) — воля говорящего: `Sollst meinen Tritt nicht hören → «Моих шагов не услышишь»`.

20. **Эллипсис глагола.** DE допускает опущение, RU — нет: `Sacht, sacht die Türe zu! → «Прикрою тихо-тихо дверь»`. Обязательна lang-аннотация. Если опущенный verb **неопределим** (намеренный эллипсис-возглас) — RU **тире-маркер**, не домысливание глагола: `Lustig in die Welt hinein → «Весело — в мир»` (не «Иду весело»).

21. **Не угадывать — проверять; Adelung-градации значимы.** (а) Приоритет — смыслы 1820-х (Adelung + Grimm). Если Adelung ≠ DWDS modern — у Müller'а Adelung. **Близкие RU-синонимы развести по корням** (трепетать DE zittern ≠ трепаться DE flattern). **RU→DE обратная сверка — только sanity check, не аргумент**. При «проверить точность» — Шаг 0 `scripts/word-check.py` + вывод в чат (см. протокол). (б) **Грамматические утверждения** (бывш. правило 31): DWDS, Duden, E-VALBU (grammis.ids-mannheim.de/verbvalenz). Первая ассоциация может быть ложной. (в) **Adelung-градации значимы** (бывш. правило 53): «erste Grad», «mit Widerwillen», «weniger als…» меняют RU: `trübe` (erste Grad der Unlust) → «сумрачный», не «мрачный»; `träge` (mit Widerwillen) → «вялый»; `säumen` → «мешкать»; `matt` → «слабый»; `elend` → компаунд (правило 45). Для значимого слова — Adelung + Grimm DWB, не только DWDS modern.

22. **Schubert vs Müller — по иерархии «Источник DE»; три источника Müller'а.** Приоритет: автограф + NSA > Rastl > Müller WH1824. LiederNet/Hampson — НЕТ. При первой итерации — список различий с цитатами **до** записи. **`variant_de`/`variant_ru` — только для sung-вариантов** (Песня 4 erstorben/erfroren, Песня 20 Wegen/Straßen, Песня 23 L7 Ach/Ja). Текстуальные правки M vs S, которые не поют, — НЕ повод для variant (П.21 L12 «Und» из WH 1824 — не поют, variant убран; разница — в meaning-аннотации). **Три источника Müller'а** (бывш. правило 57): (1) **Urania 1823** — 12 стихов; (2) **Deutsche Blätter 1823** — 10 «продолжений» (Krähe, Mut, Wegweiser, Leiermann); (3) **Waldhornist II 1824** — все 24 с правками. Schubert: Часть I (1-12) по Urania, Часть II (13-24) по Waldhornist; Deutsche Blätter в Вену 1827 не доходил. **Алгоритм DB/WH/S**: DB=WH=S, отличие у S → правка Schubert'а; DB≠WH=S → правка Müller'а; DB=S≠WH невозможно. Применим только к 10 «продолжениям». Для Urania/Waldhornist'а без DB — расхождение = правка Schubert'а. LiederNet/schubertsong.uk «Müller» — часто Deutsche Blätter.

23. **Элизии (апостроф)** — последовательно в `lines_de`, `title_de`, `de` сегментов: zieh', dacht', hätt', hab', Wär', Ruh', pfiff', Gefror'ne, An's. Перед многоточием рамки апостроф сохраняется (`pfiff'... aus`). **Apokope конца -e/-en** → апостр.; **синкопа -e- середины слова** (sehn/stehn/angesehn) — БЕЗ апостр. (Müller WH1824 convention).

24. **Перевод через отрицание при расхождении коннотаций.** `lau` (акцент «недостаточно горячий») → «негорячи», не «теплы» (RU «тёплый» позитивно).

25. **`So` в `so...dass` — всегда степень.** «так»/«настолько», никогда «едва/еле». Направление — следствием в dass-придаточном.

26. **Konjunktiv II ≠ настоящее.** Hätte/wäre/wollte — прошедшее + «бы»: wolltet → «хотели бы». После «словно/как будто» — прошедшее без «бы».

27. **Шубертовские музыкальные повторы.** (а) **Повтор через ` / `** в `lines_de`, `de`/`ru`: `"Und / Ihr dringt..."`, `{"ru":"А / Вы","de":"Und / Ihr"}`. Первый — основной, второй — повтор. (б) **Прямые повторы внутри строки** (бывш. правило 51): Schubert повторяет слова в вокальной партии. `Lines_de` отражает sung-text. Проверять по NSA Bärenreiter (основной). Naxos подтверждает. Примеры: `Wein', wein'` П.16 L12, `so elend, so elend` П.12 L12. **Повтор как ресурс**: одно DE дважды — два слота для разных RU пластов: `so elend, so elend` → «так жалок, так несчастен».

28. **Одно DE через цикл — один RU; близкие в одной песне — разные; конвенция `erst`.** Конвенции: `rauschen → шуметь`, `Geselle → дружище`. Близкие (Boden/Erde, Flur/Wiese/Feld; Schmerz/Leid/Leiden) — разные RU. Каузативные пары: `dringen aus → вырываться`, `durchdringen → прорвать`, `drängen → рваться`. **Модификаторы блокируют** (so/noch/je/gar). **Substantiv vs attributiv**: `manches Blatt` (attr.) → «не один лист» vs `manches` (subst.) → «всякое». **Морф. пары в одной песне** (Wegweiser/Weiser → Путевой указатель/указатель) — различать. **Авторские лексич. цепочки** (Tot-X П.21; weisen-: Wegweiser/Weiser П.20 → weisest ab П.21; treu+Wanderstab: П.02 + П.15 + П.21; безум-: törichtes П.20 → Toren П.22) — сохранять корень в RU; если связь теряется — отметить в lang-аннотации. Контекстуальное отступление — с сигнализацией (правило 6). **RU шире обоих DE** — дифференциация искусственна: оставить одно RU + lang-аннотация о различии (П.22 sprechen/sagen → оба «говорит»; RU «говорить» покрывает и процесс, и содержание). **Капитализация по контексту**: `Gott` → «Бог» (монотеист. Создатель, П.1) vs «бог» (политеистич./общефилос., П.22 с «Götter» — строчная). **Конвенция `erst`** (бывш. правило 39): `erst` в `nun erst`, `jetzt erst`, `in... erst` — временной контраст, не первичность: «лишь»/«только» (= не раньше, только сейчас), **не «впервые»**. Относится к обстоятельству, не объекту.

29. **Регистр двунаправленно, переворот за 200 лет.** (а) Возвышенное DE → возвышенное RU; нейтральное → нейтральное. Не «украшать» поэтизмами обычное у Müller'а. Проверять DWDS (gehoben/dichterisch/veraltet) + Adelung + Grimm: DWDS `veraltet` отражает совр. язык, в 1820-х могло быть стандартным (`ward`). Ловушки без основания: «око», «знать», «грезить», «вещать». (б) **Регистровый переворот за 200 лет** (бывш. правило 52): регистр некоторых слов изменился. Пример: `hell` — Adelung `edler`; `licht` — «im gemeinen Leben». Совр. DWDS наоборот. **Доверять Adelung 1811 для регистра 1820-х**.

30. **Предлог/направит. наречие глагольной конструкции — в рамку с глаголом.** `suchen nach`, `fragen nach` → `{"ru":"ищу","de":"such'... nach"}`. То же для `her`, `hin`, `herauf`, `hinaus` (`Von der Straße her ein Posthorn klingt` → `her... klingt` с глаголом, `Von` — отдельный предлог).

32. **Глагольные приставки несут смысл.** Zer- = разрушение (zerbrechen). Er- = завершение (ersterben — угаснуть). Ver- = изменение/утрата. Не переводить только корень: zerschmelzen ≠ schmelzen.

34. **Futurisches Präsens; Futurperfekt через Präsens Perfekt.** (а) Wenn + Präsens → RU **будущее**: `Wenn die Gräser sprossen wollen` = когда захотят прорасти. **V1 + явный Nom без wenn/als** — почти всегда бессоюзное условие или Folgesatz после императива; в RU «Если/Когда/И» с пустым `de` (правило 18). Пример: `Spielt der Wind mit meinem Blatte` → «Когда играет ветер...». Исключение: V1 как ответ на вопрос по рифмовке. (б) **Futurperfekt через Präsens Perfekt + наречие будущего** (бывш. правило 64): Aux+Partizip II + `morgen`/`bis morgen`/`übermorgen` = vollendete Zukunft (разг. эквивалент Futur II). RU — **будущее**, не Präteritum. Признак: семантически прошедшее при наречии будущего нелогично. Пример: `Und morgen früh ist alles zerflossen` → «И завтра утром всё **растечётся**».

35. **Спектр значений: абстрактное и предметное; классические/библейские аллюзии.** (а) Контекст определяет. `Angedenken` = воспоминание+сувенир — `mitnehmen` → предметное. `Fluss` — общее; контекст (прозрачная, замерзает, Bach) → «речка». **Предлоги тоже многозначны**: `Durch` = «через» (поперёк) или «по» (вдоль) — `Durch des Bergstroms trockne Rinnen` = вдоль русел. (б) **Классические/библейские аллюзии — сохранять узнаваемость** (бывш. правило 38): `Jeder Strom wird's Meer gewinnen` = Экклезиаст. «Каждая река обретёт море» узнаваемо.

36. **Cross-distribution: смыслы распределены по-разному.** Если DE-конструкция и RU делят смысл не параллельно — не писать ложных пар. Пример: `Ob's... wohl` ↔ «Уж не... ли» (`ob` = «не...ли», `wohl` = «уж»). Решение: вся DE-конструкция в рамку с многоточием, лишняя RU-частица — пустой `de`. С lang-аннотацией.

37. **Связность ландшафтной сцены.** Проверять согласованность отдельных слов с общей картиной. «Скальные лощины» (пологие) ≠ «горная река» + «вниз» — один неточен.

40. **AcI (Acc + Inf/Partizip) в RU невозможен.** Передавать через: (а) **«как»-клаузу** (`Fühlst deinen Wurm... sich regen → «Чувствуешь, как твой червь встрепенётся»`); или (б) **двоеточие** (`Einen Weiser seh' ich stehen → «Один указатель вижу я: стоит»`) — стандарт RU после глаголов восприятия. **Предпочесть (б)** при меньшем числе пустых de. Обе формы создают новую клаузу → «свой» через границу не работает (правило 14).

41. **Возвратные RU + `Mit` — сохранять «с».** `Mit + Dat + sich-verb`: `Mit` = comitative/manner, не чистый инструменталь. RU возвратные (встрепенуться, обернуться) не принимают чистый творительный — «С горячим жалом встрепенётся».

42. **Не приписывать оригиналу переклички из перевода.** Если RU создал параллель (`unwirtbarem → «бесприютном»` и `Obdach → «приют»`), проверить DE: разные корни? Если да — параллели у Müller'а нет.

43. **Перед переводом значимого слова — grep по всем песням.** `grep '"de": "<слово>"' src/data/songs/*.json`. Либо следовать конвенции (правило 28), либо отступить с сигнализацией. Также — другие морф. формы корня в этой же песне (Wegweiser↔Weiser): различать.

44. **Избегать RU конструкций с поверхностной омографией.** «так, как» vs «так как»; «что бы» vs «чтобы»; «то же» vs «тоже»; «также» vs «так же». Для `so wie` — «Такими, какими…», «Как…», «Подобно тому, как…». Двусмысленные на беглом восприятии — переформулировать.

45. **Компаунды через дефис для непереводимых корневых семей/многослойных значений.** RU без однословного эквивалента — 2 или 3+ компонентов через дефис. Применять когда: (а) многозначность важна; (б) ни один RU-синоним не покрывает всё; (в) контекст требует всей палитры. Все компоненты склоняются согласованно. Любая часть речи: сущ. (`Traum → сон-мечта`, `Täuschung → обман-иллюзия`), глаг. (`erlaben → освежать-услаждать`), прил. (`elend → жалок-несчастен-обездолен`).

46. **«Дословно»/«буквально» vs «прямой перевод».** «Дословно X — Y» требует корневого + грамматически параллельного соответствия (Apfel → яблоко). Когда корни/морфология не совпадают, но Y — однословный эквивалент → «прямой перевод» (Träumer → прямой перевод «сновидец»). При частичном покрытии — оговорка.

47. **В lang-аннотациях цитировать фактическую форму строки, не лемму.** Лемма отдельно (`Глагол nachdenken — ...`), не атрибутируется Müller'у. **Перед оформлением стандартной lang-аннотации** (отрицание, sollen, AcI, эллипсис) — посмотреть как в других песнях того же типа и единообразить `segment_range` (обычно узкий, на DE-слово) и формулировку. Внутри одной аннотации все RU-формы одного DE-слова — единообразны.

48. **Структурные утверждения о цикле проверять чтением песен.** «X — последнее появление Y», «впервые…» — конкретный текст может опровергнуть. Проверка: grep + чтение. Различать формы (`lassen` и `verlassen`). Перекрёстные ссылки программно проверять (Python). Утверждения об исторических изданиях — по нескольким независимым источникам.

50. **Dass-Ausrufesatz: `Ach, dass... so [adj]!` — Klage** (сетование, не удивление; к 1820-м архаично). Перевод: «Ах, что [Subj] так [Adj]!» + усилительная «же». Структура: dass→что, so→так; «же» — пустой `de`. Пример: `Ach, dass die Luft so ruhig!` → «Ах, что же воздух так спокоен!» (П.12).

54. **Русская типографика в аннотациях, заголовках, `ru`-полях:** «...» ёлочки (не `"..."`); — длинное тире (не `-`, дефис только в компаундах); **ё** обязательна (ещё/всё/её/идёт/найдёт/чётко/своё/нём); одинарный пробел; многоточие `...`; «не что иное, как» (раздельно, конкретизация) vs «ничто иное» (слитно, отрицание). **Обязательная ё-проверка после каждой записи** (regex: `\bнем\b|\bеще\b|задает|осознается|передает|\bидет\b|найдет|\bчетко\b|твердой|замерзш|\bвсем\b|\bсвое\b`).

55. **Следственное `dass` без явного `so`/`derart`** (литер. архаика XIX в.) → RU «**так что**». Без якоря RU «что» читается как комплементное. Контраст: `so lau, dass…` (явный `so`) → «столь… что…»; `Hab' wieder schwarze Haare, dass mir's… graut` (без `so`) → «У меня снова чёрные волосы, **так что** мне… страшно».

56. **Сигнализация о подменах предметов/концептов** (доп. к 6). Если RU-эквивалент означает иной материальный предмет, чем DE (`Bahre` = открытые носилки vs «гроб» = закрытый ящик-Sarg) — **сигнализировать** при первоначальном переводе до записи. Особенно при социально-историческом подтексте.

58. **Каждая стихотворная строка — с прописной.** При сегментации первого слова легко получить строчную. Стихотворная конвенция > синтаксической: каждая строка `lines_ru` с прописной. Проверка после записи: первый сегмент каждой строки.

59. **Грамматическая модальность в аннотациях.** Сохранять модальность DE: V1 + `?` → «спрашивает»; императив (`lass...!`) → «просит/велит»; декларатив → «говорит». Не растворять разные модальности в «герой обращается».

61. **Сверка пунктуации DE с авторитетным источником** (доп. к разделу «Источник DE»):
- Экспрессивная (`!`/`?`/`;`/`—`): Müller WH1824 + Rastl (NSA в нотах упрощает).
- Запятая/точка/двоеточие: Müller + Rastl + NSA.
- Апострофы: Rastl (NSA опускает; пример П.22 L4 sung «sing» но писать `Sing'`).
- Поэт. капитализация Müller'а (`Einem/Andern` П.1 L23, `Manches` П.17 L3, `Alles` П.17 L5): сохранять (поэт. субстантивация).
- Лексика, артикли: NSA + автограф.

**Naxos PDF — НЕ источник пунктуации** (12+ опечаток). Программная проверка: вывести все lines_de и сверить.

62. **Winterreise — психологический цикл, не натуралистический.** Природные приметы не всегда согласованы реалистически: `bunte` листья (П.16) vs замёрзшие речки/слёзы (П.3, 6, 7) — противоречие в символическом плане (Youens). Müller не следует строгой природной хронологии. Не делать необоснованных утверждений о «глубокой зиме» / «поздней осени» — учитывать **все** приметы.

63. **Финальная систематическая проверка перед закрытием песни.** Обязательно: (а) каждый сегмент DE↔RU пословно; (б) факт-чек утверждений в аннотациях (отсылки — grep; DE-цитаты — корректность); (в) ё-проверка (правило 54); (г) ASCII-кавычки `re.findall(r'"[А-Яа-яёЁ]')`; (д) короткие дефисы между RU-словами; (е) капитализация первых сегментов (правило 58); (ж) JSON-валидация; (з) пунктуация DE с источником (правило 61); (и) **список спорных мест** до объявления финализированной.

## Annotation Guidelines

Два типа: **`lang`** (грамматика — синий) и **`meaning`** (смысл — коричневый). Нумерация независимая.

**Lang** — Verbklammer, порядок слов, падежи, род, отрицание, идиомы, составные слова.
**Meaning** — этимология, мотивы, многозначность, соц. контекст, Schubert vs Müller с эффектом.

**Стиль**: не категорично; без доказательств не утверждать; **не ссылаться на учёных по именам**; **не упоминать словари по именам** («по DWDS», «по Adelung'у» — нет); регистровые пометы не цитировать буквально. Источники — для фактчека, не для текста. **О тексте, не об учебнике** — без лингв. терминологии («Konjunktiv II в немецком выражает...»). Каждый факт проверять (правила 21, 47).

### Технические

- `line_span: N` — охват N строк. Сноска в конце последней. Может пересекать строфы.
- `continuation_ranges: [[start,end], ...]` — диапазоны для каждой следующей строки. `[-1,-1]` пропускает строку без подсветки (П.19 L2, П.22 stz1 spricht/sagt). Не `null`.
- `segment_range: [start, end]` — индексы включит.
- Аннотации могут перекрываться (lang + meaning на одном сегменте).
- `\n` в тексте — перенос/разделитель абзацев (П.12, 14, 19).
- **Ударение в RU** — combining acute U+0301 после ударной гласной: `во́рон`, `что́`.
- **Курсив** — `*текст*` → `<em>` (рендер `src/utils/renderText.js` в AnnotationsPanel + tooltip SongView; HTML экранируется). Для кириллицы Unicode-курсива нет. Пример: П.22 «*против*».
- **`title_annotations`** — композиционные/смысловые комментарии к песне целиком (без `segment_range`); meaning-аннотация на всю песню — через `line_span` + `continuation_ranges` на все строки (П.22). Примеры title_annotations: П.12, 13, 14, 22.

### Формат сегментов

- Отделяемые приставки: `"de":"pfiff'... aus"` (многоточие при разрыве).
- Рамки предлогов: `"de":"Auf... Haus"`.
- Составные слова: одним сегментом в `de`, можно разбить в `ru`: `{"ru":"женщины образ.","de":"Frauenbild"}`.
- Пустой `ru` — артикль без эквивалента.
- Пустой `de` — RU-добавка для грамматики.
- **Пунктуация только в `ru`, не в `de`.** `{"ru":"ах,","de":"ach"}`.
- **`variant_de`/`variant_ru`** — sung-расхождения Schubert vs Müller (правило 22). Основное — Schubert sung-text; variant — только различающееся слово (общее — в основном de): `{"de":"den Wegen","variant_de":"Straßen"}` (П.20). Пунктуация общая — только в основном RU (не дублировать в variant_ru: П.4 «угасшее,»/«замёрзшее», П.20 «путях,»/«дорогах», П.23 «Ах,»/«Да»).
