# Winterreise — Зимний путь

Подстрочный перевод вокального цикла Франца Шуберта «Зимний путь» (Winterreise, D 911, 1827) на стихи Вильгельма Мюллера — все 24 песни.

**Сайт: https://evgobmm.github.io/Winterreise/**

Что внутри:

- Немецкий текст в шубертовской редакции — сверен по автографу 1827 года, Новому шубертовскому изданию (Bärenreiter) и прижизненным изданиям Мюллера.
- Пословный русский подстрочник: под каждым русским словом — соответствующее немецкое.
- Комментарии двух типов: языковые (грамматика, идиомы, порядок слов) и смысловые (мотивы, контекст, отличия редакции Шуберта от Мюллера).
- Значения слов выверены по словарям эпохи (Adelung 1811, Grimm) и современным (DWDS, Duden).
- Записи исполнений для каждой песни.

## Данные

Тексты и разметка лежат в `src/data/songs/*.json` — по файлу на песню: строки оригинала, сегменты пословного перевода, аннотации. Это основная ценность репозитория; приложение (Vue 3 + Vite) — лишь способ их показать.

## Что выверено человеком

Я внимательно проверял только то, что отображается на сайте: тексты песен, пословный перевод, комментарии, оформление страниц. Остальные части репозитория — код приложения (компоненты, стили, скрипты интерфейса), служебные скрипты для работы над переводом (`scripts/`), правила и инструкции для ИИ-ассистента (`CLAUDE.md`), конфигурация сборки и публикации (Vite, GitHub Actions), тексты коммитов — написаны в основном ИИ-ассистентом и содержат фрагменты, которые я ни разу не проверял; поэтому они могут не отражать мою волю.
